## 一维差分

**本质上是对多次区间修改，单次查询的优化。主要是对结果数组进行往前差分，把中间值全部弄成0，只保留端点值，最终再进行对应次数的逆向前缀和操作，得到多次区间修改后的结果**

### 1.航班预定统计 https://leetcode.cn/problems/corporate-flight-bookings/description/

**思路：区间覆盖问题，并且是多次修改，最终为单次查询，就应该马上想到差分。然后差分数组的长度得看两方面：1.索引是否从1开始；2.r最大是多少，因为我们要用到r + 1。之后，直接写就行**

**代码：**
```python
class Solution:
    def corpFlightBookings(self, bookings: List[List[int]], n: int) -> List[int]:
        diff = [0] * (n + 2)
        for l, r, w in bookings:
            diff[l] += w 
            diff[r + 1] -= w 
        
        ans = list(accumulate(diff))
        return ans[1:-1]
```

### 2.三步必杀 https://www.luogu.com.cn/problem/P4231

**思路：一维等差数列的差分题，主要是用最终得到的结果，去倒推出仅需要修改端点值的差分，发现需要往前做两次差分得到，只需要修改l, l + 1, r + 1, r + 2这四个端点值，然后再做两次差分，就可以得到最终结果。**

**代码：**
```python

    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n, m = MII()
    diff = [0] * (n + 3)
    for _ in range(m):
        l, r, s, e = LII()
        d = (e - s) // (r - l)

        diff[l] += s
        diff[l + 1] += d - s
        diff[r + 1] -= d + e
        diff[r + 2] += e

    for i in range(n + 2):
        diff[i + 1] += diff[i]
    for i in range(n + 2):
        diff[i + 1] += diff[i]

    print(reduce(xor, diff), max(diff))

    solve()
```

### 2.用邮票贴满网格图 https://leetcode.cn/problems/stamping-the-grid/description/

**思路：先求二维前缀和，然后遍历并且判断能否贴邮票，如果可以贴就加入二维差分数组中，最后算整体的二维前缀和，并且判断对应位置是否两个都是0，如果是的话，说明不能贴满，直接返回False，最终检查完，说明能贴完，返回True**

**代码：**
```python
class Solution:
    def possibleToStamp(self, grid: List[List[int]], h: int, w: int) -> bool:
        m, n = len(grid), len(grid[0])

        # print("grid:")
        # for row in grid:
        #     print(*row)
        # print()

        pre = [[0] * (n + 1) for _ in range(m + 1)]
        for i, row in enumerate(grid):
            for j, x in enumerate(row):
                pre[i + 1][j + 1] += pre[i + 1][j] + pre[i][j + 1] - pre[i][j] + x

        # print("pre:")
        # for row in pre:
        #     print(*row)
        # print()

        def check(a, b, c, d):
            a += 1; b += 1; c += 1; d += 1
            return pre[c][d] - pre[a - 1][d] - pre[c][b - 1] + pre[a - 1][b - 1]

        diff = [[0] * (n + 2) for _ in range(m + 2)]
        for a in range(m):
            for b in range(n):
                c = a + h - 1; d = b + w - 1
                if c >= m or d >= n:
                    continue
                if check(a, b, c, d) == 0:
                    diff[a][b] += 1
                    diff[a][d + 1] -= 1
                    diff[c + 1][b] -= 1
                    diff[c + 1][d + 1] += 1

        # print("diff:")
        # for row in diff:
        #     print(*row)
        # print()

        pre_diff = [[0] * (n + 2) for _ in range(m + 2)]
        for i in range(m + 1):
            for j in range(n + 1):
                x = diff[i][j]
                pre_diff[i + 1][j + 1] += pre_diff[i + 1][j] + pre_diff[i][j + 1] - pre_diff[i][j] + x

        # print("pre_diff:")
        # for row in pre_diff:
        #     print(*row)
        # print()

        for i in range(m):
            for j in range(n):
                if grid[i][j] == pre_diff[i + 1][j + 1] == 0:
                    return False

        return True
```

### 3.最强祝福力场 https://leetcode.cn/problems/xepqZ5/

**思路：数学变化，先把0.5小数通过$2 * x + r$、$2 * x - r$、$2 * y + r$、$2 * y - r$给除掉，然后再进行离散化处理，最后用二维差分以及二维前缀和，找最大值即可。需要注意的是，本题中$(a, b)$为左上角，应该两个都是-号，为什么？因为他是矩阵坐标系的左上角，而不是直角坐标系的左上角；同理$(c, d)$为右下角，应该两个都是+号，这个点需要额外注意**

**代码：**
```python
class Solution:
    def fieldOfGreatestBlessing(self, forceField: List[List[int]]) -> int:
        xs = []; ys = []
        for x, y, r in forceField:
            xs.append((x << 1) - r)
            xs.append((x << 1) + r)
            ys.append((y << 1) - r)
            ys.append((y << 1) + r)

        xs = sorted(set(xs))
        ys = sorted(set(ys))

        # print("xs", xs)
        # print("ys", ys)
        # print()

        m, n = len(xs), len(ys)
        diff = [[0] * (n + 2) for _ in range(m + 2)]
        for x, y, r in forceField:
            # 这里(a, b)是矩阵左上角，不是坐标系左上角，(c, d)同理
            a = bisect_right(xs, (x << 1) - r)
            b = bisect_right(ys, (y << 1) - r)
            c = bisect_right(xs, (x << 1) + r)
            d = bisect_right(ys, (y << 1) + r)
            
            diff[a][b] += 1 
            diff[a][d + 1] -= 1
            diff[c + 1][b] -= 1
            diff[c + 1][d + 1] += 1

        # print("diff: ")
        # for row in diff:
        #     print(*row)
        # print()
        
        ans = 0
        pre = [[0] * (n + 3) for _ in range(m + 3)]
        for i in range(m + 2):
            for j in range(n + 2):
                x = diff[i][j]
                pre[i + 1][j + 1] += pre[i][j + 1] + pre[i + 1][j] - pre[i][j] + x
                ans = max(ans, pre[i + 1][j + 1])
        return ans 
```

## 区间覆盖问题

### 1.小红的最佳区间 https://ac.nowcoder.com/acm/contest/134529/D

**思路：思维转化 + 离散化差分。首先考虑一个问题，我们要让$[l,\ r]$区间被$[L,\ L + k]$区间覆盖，哪些L点可取？因为题目说了端点重合就行，因此$L$满足条件的范围就是$[l - k, r]$，我们就在这上面做离散化差分即可**

**代码：**
```python
import sys
from collections import defaultdict

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n, k = MII()
    mx = 0
    a = []
    for _ in range(n):
        l, r = MII()
        mx = fmax(mx, r)
        a.append([l, r])

    diff = defaultdict(int)
    for x, y in a:
        diff[x - k] += 1
        diff[y + 1] -= 1

    ans = 0
    pre = 0
    for k, v in sorted(diff.items()):
        pre += v
        ans = fmax(ans, pre)

    print(ans)

t = 1
# t = II()
for _ in range(t):
    solve()
```

## 区间交并问题

### 1.使数组互补的最少操作次数 https://leetcode.cn/problems/minimum-moves-to-make-array-complementary/description/?envType=daily-question&envId=2026-05-13

**思路：思维 + 差分。这题难的原因不在于通过区间+-想到差分，而是思维上怎么处理题目给出的条件。本题引入了一个$a[k]$数组，表示$nums[i] + nums[n - 1 - i]$变成k需要的操作次数，接着我们就想到对于每一对$nums[i] + nums[n - 1 - i]$都算出$a[k]$，那么答案就是他们的总和了。那操作数怎么算？就根据区间的交并来得到操作数为0，1，2的具体区间，接着想到对于区间的+-来说，差分是高效的处理方式。最终，前缀和算整体变成k的最小值就是答案**

**代码：**
```python
class Solution:
    def minMoves(self, nums: List[int], limit: int) -> int:
        n = len(nums)
        diff = [0] * (limit * 2 + 2)

        for i in range(n // 2):
            x = nums[i]
            y = nums[-1 - i]
            l = min(x, y) + 1
            r = max(x, y) + limit

            diff[2] += 2
            diff[l] -= 2

            diff[l] += 1
            diff[r + 1] -= 1

            diff[x + y] -= 1
            diff[x + y + 1] += 1
            
            diff[r + 1] += 2

        ans = inf
        sum_d = 0
        for d in diff[2 : limit * 2 + 1]:
            sum_d += d
            ans = min(ans, sum_d)
        return ans
```
