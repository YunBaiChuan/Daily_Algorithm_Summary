## 树形dp：

### 1.没有上司的舞会 https://www.nowcoder.com/practice/f703237089ee42d9b37e01d70e14e2fc?channelPut=tracker2

**思路：看到题目中的孩子和父亲只能选一个，并且是求解最大值问题，首先要想到树形dp。那他其实很像是状态机，不过是在树上用dfs跑的，具体来说他的状态转移如下：**

| 状态 | 含义 | 转移方程 | 解释 |
|---|---|---|---|
| dp[u][0] | 不选节点 u | dp[u][0] += max(dp[v][0], dp[v][1]) | 不选当前结点u的时候，那么我们可以选或不选他的孩子结点，取max即可
| dp[u][1] | 选节点 u | dp[u][1] += dp[v][0] | 选当前节点u的时候，他的孩子节点我们就不能选

**然后就是re问题，一般是栈爆了，所以说可以扩一下栈深度**

**代码：**
```python
import sys
sys.setrecursionlimit(10000)
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
有向树，dp
'''

def solve():
    n = II()
    w = LII()
    g = [[] for _ in range(n + 1)]
    is_sub = [False] * (n + 1)  # 标记是否做过“子节点”

    for _ in range(n - 1):
        k, l = MII()
        g[k].append(l)
        is_sub[l] = True

    # 找根：唯一没当过子节点的就是总 boss
    # 如果索引从1开始，这里千万别用enumerate，包错的
    root = next(i for i in range(1, n + 1) if not is_sub[i]) # 由于这里只取一个元素，所以说用next包裹一层，只取满足条件的第一个元素，和拓扑排序里面的[i for i, x in enumerate(a) if x != 0]不同，他是有多个没限制的元素

    # dp[u][0] 不选 u 的最大值
    # dp[u][1] 选 u 的最大值
    dp = [[-inf] * 2 for _ in range(n + 1)]

    # 这里是有向树，如果是无向树的话可以多传一个fa参数
    # 之后for里面多一条判断语句即可if v == fa: continue
    def dfs(u):
        dp[u][1] = w[u - 1] # 初始化，选。u - 1是因为索引从1开始
        dp[u][0] = 0 # 初始化，不选
        for v in g[u]:
            dfs(v) # 先算子树
            dp[u][0] += max(dp[v][0], dp[v][1]) # 不选u，那就在下属中选或不选找最大
            dp[u][1] += dp[v][0]    # 选 u 就不能选直接下属

    dfs(root)
    print(max(dp[root][0], dp[root][1]))

t = 1
# t = II()
for _ in range(t):
    solve()
```

### 2.Rinne Loves Edges https://www.nowcoder.com/practice/080d6bd6edb043bd855f633f70edde1d?channelPut=tracker2

**思路：想法还是比较简单的，一个dfs从s遍历到叶子节点，每次删除路径上的最小权重边，最终累加到答案中**

```python
    n, m, s = LII()
    g = [[] for _ in range(n + 1)]
    for _ in range(m):
        u, v, w = LII()
        g[u].append((v, w))
        g[v].append((u, w))
```

### 3.二叉搜索子树的最大键值和 https://leetcode.cn/problems/maximum-sum-bst-in-binary-tree/description/

**思路：树形dp入门题，主要是分析父节点与子节点的依赖关系，需要哪些信息进行传递。然后就可以进行全集整合，返回信息给上层，经过信息的再处理和整合之后，得到最终答案**

**代码：**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxSumBST(self, root: Optional[TreeNode]) -> int:
        ans = 0 
        def dfs(node):
            if not node:
                return True, inf, -inf, 0 
            is_lbst, min_l, max_l, sum_l = dfs(node.left)
            is_rbst, min_r, max_r, sum_r = dfs(node.right)
            is_bst = is_lbst and is_rbst and max_l < node.val < min_r
            mn = min(min_l, node.val)
            mx = max(max_r, node.val)
            s = sum_l + sum_r + node.val
            if is_bst:
                nonlocal ans 
                ans = max(ans, s)
            return is_bst, mn, mx, s 
        dfs(root)
        return ans
```

### 4.二叉树的直径 https://leetcode.cn/problems/diameter-of-binary-tree/

**思路：和第3题差不多，就是需要的信息更少了，大流程一致：1.当前点x选或不选；2.答案求解需要哪些信息全集；3.整合答案，向上传递；4.更新答案，输出最终结果**

**代码：**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        ans = 0 
        def dfs(node):
            if not node:
                return 0, 0
            dl, hl = dfs(node.left)
            dr, hr = dfs(node.right)
            d = max(dl, dr)
            h = max(hl, hr) + 1
            nonlocal ans 
            ans = max(ans, d, hl + hr)
            return d, h
        dfs(root)
        return ans 
```

### 5.在二叉树中分配硬币 https://leetcode.cn/problems/distribute-coins-in-binary-tree/description/

**思路：套路也是和上面两题差不多，主要区别在于需要传递哪些信息，问题的分析**

**代码：**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def distributeCoins(self, root: Optional[TreeNode]) -> int:
        ans = 0
        def dfs(node):
            if not node:
                return 0, 0, 0
            cnt_l, sum_l, move_l = dfs(node.left)
            cnt_r, sum_r, move_r = dfs(node.right)
            cnts = cnt_l + cnt_r + 1 
            sums = sum_l + sum_r + node.val
            moves = move_l + move_r + abs(cnt_l - sum_l) + abs(cnt_r - sum_r)
            nonlocal ans 
            ans = max(ans, moves)
            return cnts, sums, moves 
        dfs(root)
        return ans 
```

### 6.监控二叉树 https://leetcode.cn/problems/binary-tree-cameras/

**思路：没有dp数组，仅仅是几个变量0、1、2表示三种状态，通过返回状态来更新答案**

**代码：**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def minCameraCover(self, root: Optional[TreeNode]) -> int:
        ans = 0 
        def dfs(node):
            if not node:
                return 1
            left = dfs(node.left)
            right = dfs(node.right)
            if left == 0 or right == 0:
                nonlocal ans 
                ans += 1
                return 2 
            if left == 1 and right == 1:
                return 0 
            return 1
        if dfs(root) == 0:
            ans += 1
        return ans
```

### 7.路径总和III https://leetcode.cn/problems/path-sum-iii/description/

**思路：本题需要用hash表保存前缀信息，有点像前缀和题目中那个hash表做法，不过是在树上进行操作的**

**代码：**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        d = defaultdict(int)
        d[0] = 1 

        def dfs(node, s):
            if not node:
                return 0
            s += node.val 
            ans = d[s - targetSum]
            d[s] += 1
            ans += dfs(node.left, s)
            ans += dfs(node.right, s)
            d[s] -= 1
            return ans 

        return dfs(root, 0)
```

### 8.到达首都的最少油耗 https://leetcode.cn/problems/minimum-fuel-cost-to-report-to-the-capital/description/

**思路：每个根节点的代价由两部分组成：1.子节点的代价之和；2.子节点到根节点的代价之和。因此，构建两个数组size、cost，然后就可以跑一个树型dp，其最终结果为cost[0]，即根节点为0的整体代价**

**代码：**
```python
class Solution:
    def minimumFuelCost(self, roads: List[List[int]], seats: int) -> int:
        n = len(roads) + 1
        g = [[] for _ in range(n)]
        for u, v in roads:
            g[u].append(v)
            g[v].append(u)
        
        size = [0] * n 
        cost = [0] * n 

        def dfs(u, fa):
            size[u] = 1
            for v in g[u]:
                if v != fa:
                    dfs(v, u)
                    size[u] += size[v]
                    cost[u] += cost[v]
                    cost[u] += (size[v] + seats - 1) // seats
        dfs(0, -1)
        return cost[0]
```

### 9.相邻字符不同的最长路径 https://leetcode.cn/problems/longest-path-with-different-adjacent-characters/

**思路：也是收集信息，求全集，之后进行状态转移**

**代码：**
```python
class Solution:
    def longestPath(self, parent: List[int], s: str) -> int:
        n = len(parent)
        g = [[] for _ in range(n)]
        for i in range(1, n):
            g[parent[i]].append(i)
        
        def dfs(u):
            if not g[u]:
                return 1, 1
            mx1 = mx2 = 0 # 最长链与次长链
            mx_path = 1
            for v in g[u]:
                mx_from_head, mx_the_path = dfs(v)
                mx_path = max(mx_path, mx_the_path)
                if s[u] != s[v]:
                    if mx_from_head > mx1:
                        mx2 = mx1 
                        mx1 = mx_from_head
                    elif mx_from_head > mx2:
                        mx2 = mx_from_head
            mx_from_head = mx1 + 1 
            mx_path = max(mx_path, mx1 + mx2 + 1)
            return mx_from_head, mx_path
        
        mx_from_head, mx_path = dfs(0)
        return mx_path
```

### 10.移除子树后的二叉树高度 https://leetcode.cn/problems/height-of-binary-tree-after-subtree-removal-queries/description/

**思路：dfn序：用dfs遍历一遍之后，得到的序号，并且根据dfn序号得到节点对应的子树大小、深度。他的作用在于，可以根据dfn序快速判断两个节点是否存在包含关系。然后，本题还需要用到前后缀分解最大值，去掉中间一段子树之后，剩下的最大值其实就是前缀区间中的最大值，与后缀区间中的最大值，其中更大的一方**

**代码：**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def treeQueries(self, root: Optional[TreeNode], queries: List[int]) -> List[int]:
        MAXN = 10 ** 5 + 5
        dfn = [0] * MAXN
        deep = [0] * MAXN 
        size = [0] * MAXN
        maxl = [0] * MAXN 
        maxr = [0] * MAXN
        
        dfn_cnt = 0
        def dfs(node, k):
            nonlocal dfn_cnt
            dfn_cnt += 1
            i = dfn_cnt
            dfn[node.val] = i
            deep[i] = k 
            size[i] = 1
            if node.left:
                dfs(node.left, k + 1)
                size[i] += size[dfn[node.left.val]]
            if node.right:
                dfs(node.right, k + 1)
                size[i] += size[dfn[node.right.val]]
        dfs(root, 0)

        for i in range(1, dfn_cnt + 1):
            maxl[i] = max(maxl[i - 1], deep[i])
        
        for i in range(dfn_cnt, 0, -1):
            maxr[i] = max(maxr[i + 1], deep[i])
        
        m = len(queries)
        ans = [0] * m 
        for i, x in enumerate(queries):
            left_max = maxl[dfn[x] - 1]
            right_max = maxr[dfn[x] + size[dfn[x]]]
            ans[i] = max(left_max, right_max)
        
        return ans
```

### 12.小苯的路径计数 https://ac.nowcoder.com/acm/contest/131111/D

**思路：这题思路还是比较简单的，有点树形dp的意思。主要问题在于统计路径的时候，dfs里面的cnt表示每个节点为开头时的路径数，因此每个节点跑一个dfs，然后累加就行，但是这样复杂度非常高，需要考虑优化。具体优化就是跑一次dfs，然后全局统计ans，这个优化技巧需要掌握**

### 13.绿豆蛙的归宿 https://www.nowcoder.com/practice/764c0b1b34d24122bd9ae75220e370a3?channelPut=tracker2

**思路：期望 + 树形dp。主要是理解题意，以及样例是怎么推导的。然后就是直接写记忆化搜索会爆栈，要改成栈模拟**

**思路：树上dp。方法容易看出来，不过状态转移需要好好分析一下。定义$dp[i][j]$表示前$i$个节点中，染色为$j = 0, 1$，即白色和红色的方案数。最开始方案数都是0，然后有初始化为$dp[u][0] = dp[u][1] = 1$，接着最重要的状态转移方程就是：1.u为白色，那么v必然要染成红色，即$dp[u][0] = dp[u][0] * dp[v][1] % mod$；2.u为红色，那么v可以是白色、红色，即$dp[u][1] = dp[u][1] * (dp[v][0] + dp[v][1]) % mod$**

### 1.买卖股票的最佳时机II https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/description/

**思路：非常经典，典型的状态机dp入门题目。对于每天来说，我们两种状态，也就是持有/不持有(卖出)，也是从后往前写记忆化，边界条件为i < 0，如果此时持有股票那是不可能的情况，否则为0。然后就写持有与不持有股票的状态转移即可。一般先写持有股票，那么状态就是上一天已经持有股票与上一天不持有股票但是购买了股票中取max；否则不持有股票，那就是上一天不持有股票与上一天持有股票但是卖掉了中取max。最终返回dfs(n - 1, False)即可，因为我们最后一天不需要持有股票，会额外花钱。剩下的就是记忆化搜索翻译，一般来说，简单的状态机dp就写个二维列表就行，空间不会超，空间压缩可以不用太关心**

**记忆化搜索_板子提取：**
```python
n = len(prices)
@cache 
def dfs(i, hold):
    if i < 0: # 边界条件
        return -inf if hold else 0 
    # 第i天持有股票的状态转移
    if hold:
        return max(dfs(i - 1, True), dfs(i - 1, False) - prices[i]) 
    # 第i天不持有股票的状态转移
    return max(dfs(i - 1, False), dfs(i - 1, True) + prices[i])
return dfs(n - 1, False) 
# 最终答案是max(dfs(n - 1, True), dfs(n - 1, False))
# 如果最后一天买了股票的话，后面是卖不出去了，相当于浪费钱，因此答案就是dfs(n - 1, False)
```

**动态规划_板子提取：最常见的写法，推荐**
```python

dp[0][0] = 0

    # 这里确实是1：1翻译的，动态规划的hold被显式拆出来了

    dp[i + 1][0] = max(dp[i][0], dp[i][1] + p)
```

```python
f0, f1 = 0, -inf 
for p in prices:

    # 因为新f1的计算需要原先的f0，如果直接覆盖f0就会导致新f1的计算错误
    # new_f0 = max(f0, f1 + p) 
    # f1 = max(f1, f0 - p)
```

### 1.最长递增子序列 https://leetcode.cn/problems/longest-increasing-subsequence/description/

**思路：这是一个序列dp问题，也可以用二分贪心来做。从序列dp的角度出发，我们定义$f[i]$表示以$i$索引结尾的最长递增子序列，那么就可以枚举$j$，为$range(i)$，也就是$i$前面的元素，然后判断是否有元素$nums[i] > nums[j]$，这里$>$是严格递增子序列，如果是非递减的话，加个等号即可。满足这个条件，我们每轮就可以更新$dp[i]$的最大值，结束后再$dp[i] += 1$即可**

### 3.俄罗斯套娃信封问题 https://leetcode.cn/problems/russian-doll-envelopes/description/

**思路：首先对信封的第一维，即宽度进行从小到大排序，这样可以满足宽度小的信封能够放到宽度大的信封里面；其次对信封的第二维，即高度进行从大到小排序。最终，对高度那一维做LIS就是最终答案了。为什么高度那一维是从大到小排序？因为对于宽度相同来说，我们无论如何是不能进行套娃的，题目明确说了两个维度都要严格小于才行，所以说高度从大到小进行排序，就可以彻底排除宽度维度的影响了**

**代码：**
```python
class Solution:
    def maxEnvelopes(self, envelopes: List[List[int]]) -> int:
        envelopes.sort(key=lambda x:(x[0], -x[1]))
        a = [y for x, y in envelopes]
        
        def LIS(a):
            g = []
            for x in a:
                p = bisect_left(g, x)
                if p == len(g):
                    g.append(x)
                else:
                    g[p] = x 
            return len(g)
        
        return LIS(a)
```

```python
            g = []
            for j in range(i, n, k):

                    g.append(x)
                else:
                    g[p] = x
```

### 6.最长不下降子序列 https://www.luogu.com.cn/problem/P8776

**思路：非常好的一道题，属于是LIS的集大成者。首先，我们普遍意义上算的LSI其实是从左往右算，然后以$a[i]$做结尾的最长递增子序列。反转一下思路，如果我们从右往左算LIS，能得到什么？那就是从左往右看，以$a[i]$为开头的最长递减子序列了。那如果我们要算以$a[i]$做结尾的最长递减子序列怎么办？这牵扯到一个技巧，所有数变成负数来算。简单点就是每次进入g的都是-x，这样用LIS就能得到最长递减子序列长度了，如此下来我们从右往左算最长递减子序列，那么从左往右看就是以$a[i]$开头的最长递增子序列了**

**接着，再用滑动窗口，并且再做一个正向的LIS，窗口右端是用于判断的，窗口左端是用于进入g的，这个和上一题那个ends不同的是一样的思路。接下来就能得到当前答案为左 + 中 + 右，这三个部分之和，然后不断取max，就可以得到最终答案了**

## 不相交区间的最值问题

### 1.最多可以参加的会议数目II https://leetcode.cn/problems/maximum-number-of-events-that-can-be-attended-ii/description/

**思路：用dp，以及排序后的二分优化。定义$dp[i + 1][j]$表示从$0$到$i$中选择$j$个不重叠的会议进行参加，所得到的最大价值。那么其状态转移方程就是$dp[i + 1][j] = fmax(dp[i][j], dp[p + 1][j - 1] + value)$，也就是选或不选的逻辑。这里的p就需要用前面的排序二分进行查找优化，最终就可以解决本题**

**代码：**
```python
# 手写 max 更快
fmax = lambda a, b: b if b > a else a

class Solution:
    def maxValue(self, events: List[List[int]], k: int) -> int:
        # 特判 k=1 的情况可以更快
        if k == 1:
            return max(e[2] for e in events)

        events.sort(key=lambda e: e[1])  # 按照结束时间排序
        n = len(events)
        f = [[0] * (k + 1) for _ in range(n + 1)]
        for i, (start_day, _, value) in enumerate(events):
            p = bisect_left(events, start_day, hi=i, key=lambda e: e[1])  # hi=i 表示二分上界为 i（默认为 n）
            for j in range(1, k + 1):
                # 为什么是 p 不是 p+1：上面算的是 >= start_day，-1 后得到 < start_day，但由于还要 +1，抵消了
                f[i + 1][j] = fmax(f[i][j], f[p][j - 1] + value)
        return f[n][k]
```

### 1.dd爱冒险 https://www.nowcoder.com/practice/e97ab78badf64bbf947fe2097d5de355?channelPut=tracker2

**思路：要恰好走n - 1条路，把所有的点都给经过，这其实是一个旅行商问题，或者说是一个哈密顿路径问题。这种问题的n一般都很小，因为他的时间复杂度起手就是$2^n$级别，我们要用状压dp来做，比较难，先放着**

### 8.最小的必要团队 https://leetcode.cn/problems/smallest-sufficient-team/description/

**思路：首先进行状态压缩，之后就写状压dp，并且从前往后收集答案。对于python来说，最好是不要使用手写dp缓存表，慢到爆。就可以直接写个lru_cache的dfs，之后再用非固定的列表收集答案即可。当然，也可以改成动态规划，之后用标准的dp表来收集答案**

**代码：**
```python
class Solution:
    def smallestSufficientTeam(self, req_skills: List[str], people: List[List[str]]) -> List[int]:
        inf = 10 ** 18
        d = defaultdict(int)
        for i, skill in enumerate(req_skills):
            d[skill] = i 
        
        n, m = len(req_skills), len(people)
        arr = [0] * m 
        for i in range(m):
            status = 0
            for skill in people[i]:
                status |= 1 << d[skill]
            arr[i] = status

        @lru_cache(maxsize=None)
        def dfs(status, i):
            if status == (1 << n) - 1:
                return 0 
            if i == m:
                return inf
            p1 = dfs(status, i + 1)
            p2 = dfs(status | arr[i], i + 1) + 1
            return min(p1, p2)   
        
        ans = []
        s, i = 0, 0
        while s != (1 << n) - 1 and i < m:
            if dfs(s | arr[i], i + 1) + 1 <= dfs(s, i + 1):
                ans.append(i)
                s |= arr[i]
            i += 1
        
        return ans
```

### 4.环绕字符串中唯一的子字符串 https://leetcode.cn/problems/unique-substrings-in-wraparound-string/description/

**思路：依旧直接定义状态，推状态转移方程。本题定义的$dp[x]$表示以字符x为结尾，往前延伸的最大长度。算出来之后，我们累加一下所有最大长度就是子字符串的数量，这个很精妙，他这样做能保证去重。其实就相当于找从结尾往前逐个延伸的字符串，比如"abc"找的就是"c"、"bc"、"abc"，刚好就是他的长度，我们再仔细观察可以发现"b"字符长度为2，那就是"b"、"ab"，最后"a"字符长度为1，那就是"a"，累加起来就是正确的答案，为6。**

**扩展：子串的计算公式为n * (n + 1) // 2，如果有多个相同的字串这样去算，肯定会造成重复。因此，使用本题的每个字符往前求其最大延伸长度，然后累加起来，就可以得到去重后的答案**

**代码：**
```python
class Solution:
    def findSubstringInWraproundString(self, s: str) -> int:
        dp = defaultdict(int)
        l = 0
        for i, x in enumerate(s):
            if i > 0 and (ord(x) - ord(s[i - 1])) % 26 == 1:
                l += 1
            else:
                l = 1
            dp[x] = max(dp[x], l)
        return sum(dp.values())
```

**思路：斐波那契数列变种，矩阵快速幂优化。观察题目的数据量可知，这题我们需要用到矩阵快速幂进行优化。首先写两个函数：1.矩阵乘法函数mat_mul、2.矩阵快速幂函数mat_pow。由于本题的斐波那契数列递推比一般的斐波那契数列递推用到的数要更前一位，所以说就需要用到3维的递推矩阵**

**接着对3维递推矩阵进行矩阵快速幂操作，其幂次为$n - 2$。为什么？因为从索引0开始算起，那就有$f[0] = 0、f[1] = 1、f[2] = 1$，有这三个基本数据，就可以推出后面的$f[3]、f[4]....f[n]$。因此，最大的基本数据索引为$2$，那我们要推到$n$，就只需要走$n - 2$步即可。此时的答案就是$res$数组第一行向量$[res[0][0],\ res[0][1],\ res[0][2]]\bullet\ [f[2],\ f[1],\ f[0]]$，也就是$[res[0][0],\ res[0][1],\ res[0][2]]\bullet\ [1,\ 1,\ 0]$，然后把答案添加到out数组中，最终整体输出即可**

```python
    def isScramble(self, s: str, t: str) -> bool:
        n = len(s)
        @lru_cache(maxsize=None)
        def dfs(l1, r1, l2, r2):
            if l1 == r1:
                return s[l1] == t[l2]

            # 不交错
            i, j = l1, l2 
            while i < r1:
                if dfs(l1, i, l2, j) and dfs(i + 1, r1, j + 1, r2):
                    return True 
                i += 1
                j += 1
            
            # 交错
            i, j = l1, r2 
            while i < r1:
                if dfs(l1, i, j, r2) and dfs(i + 1, r1, l2, j - 1):
                    return True 
                i += 1
                j -= 1
            
            return False 
        return dfs(0, n - 1, 0, n - 1)
```

### 3.环形子数组的最大和 https://leetcode.cn/problems/maximum-sum-circular-subarray/description/

**思路：环形子数组我们要分两种情况进行讨论：1.无隔断，那就直接用第1的方法即可；2.有隔断，只取左右两端算最大和。此时中间是连续的，并且数组整体和s不变，那就可以算出中间连续部分的最小数组和，然后用s去减它，就能得到隔断的最大子数组和。最终，两个情况取最大值即可。**

**不过要注意：当数组全是负数的时候，会得到空数组s = 0的结果，但这是错的，我们的子数组要非空，那就可以判断一下这个情况，如果是的话直接输出max_ans即可**

**代码：**
```python
class Solution:
    def maxSubarraySumCircular(self, nums: List[int]) -> int:
        inf = 10 ** 18
        n = len(nums)

        min_pre, max_pre = inf, -inf
        min_ans, max_ans = inf, -inf
        for i in range(n):
            max_pre = max(max_pre + nums[i], nums[i])
            max_ans = max(max_ans, max_pre)
            min_pre = min(min_pre + nums[i], nums[i])
            min_ans = min(min_ans, min_pre)

        s = sum(nums)
        return max_ans if min_ans == s else max(max_ans, s - min_ans)
```

### 8.被7整除的子序列最大整数和（自制）

**思路：定义$dp[i][j]$表示前i个数中，余数为j的最大整数和。因此状态转移其实就两种情况：1.不选当前数，那就是$dp[i][j] = dp[i - 1][j]$；2.选当前数，那就是$dp[i][j] = dp[i][need] + a[i]$，这两种情况取max即可**

**代码：**
```python
import random
from itertools import combinations

def solve(n, a):
    inf = 10 ** 18
    dp = [[-inf] * 7 for _ in range(n + 1)]
    dp[0][0] = 0
    for i in range(n):
        cur = a[i] % 7
        for j in range(7):
            need = (j - cur) % 7
            dp[i + 1][j] = max(dp[i][j], dp[i][need] + a[i])
    return dp[-1][0]

def brute(n, a):
    res = 0
    for k in range(n + 1):
        for c in combinations(range(n), k):
            s = sum(a[i] for i in c)
            if s % 7 == 0:
                res = max(res, s)
    return res
```

**思路：这题算是数组最大乘积dp + 网格图dp了。整体写下来有点像树型dp的信息传递那块，也就是每次递归需要两个参数才能确定状态的转移，对于此题来说就是最大乘积和最小乘积。这里就只给出记忆化搜索的版本**

```python
class Solution:
    def maxProductPath(self, grid: List[List[int]]) -> int:
        mod = 10 ** 9 + 7
        @lru_cache(maxsize=None)
        def dfs(i: int, j: int) -> Tuple[int, int]:
            x = grid[i][j]
            if i == j == 0:
                return x, x

            res_min, res_max = inf, -inf
            if i > 0:
                mn, mx = dfs(i - 1, j)
                res_min = min(mn * x, mx * x)
                res_max = max(mn * x, mx * x)
            if j > 0:
                mn, mx = dfs(i, j - 1)
                res_min = min(res_min, mn * x, mx * x)
                res_max = max(res_max, mn * x, mx * x)

            return res_min, res_max

        ans = dfs(len(grid) - 1, len(grid[0]) - 1)[1]
        return -1 if ans < 0 else ans % mod
```

### 3.最后一块石头的重量II https://leetcode.cn/problems/last-stone-weight-ii/description/

**思路：也是0-1背包的转换，题目要求最终碰撞后石头的体积越小，其实可以转换成两个集合A、B，算$|sum(A) - sum(B)|$尽可能小。因此，我们就可以在stones数组中做$need = (sum(A) + sum(B))\ //\ 2$的0-1背包，也就是找下限。这里就可以用bool数组来写，最终为True的最大不超过need就是能凑出的答案，然后用$s - res$再$-res$就是绝对值差值最小的答案了**

**代码：**
```python
class Solution:
    def lastStoneWeightII(self, stones: List[int]) -> int:
        n = len(stones)
        s = sum(stones)

        need = s // 2 
        dp = [False] * (need + 1)
        dp[0] = True
        for i in range(n):
            for j in range(need, -1, -1):
                if j >= stones[i]:
                    dp[j] |= dp[j - stones[i]]
        res = max([i for i, x in enumerate(dp) if x])
        return s - res - res
```

### 8.正则表达式 https://leetcode.cn/problems/regular-expression-matching/

**思路：分类讨论，最后一种情况是用的完全背包解决，用记忆化搜索就能解决了，改dp不是很好写**

**代码：**

```python
class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        m, n = len(s), len(p)
        @lru_cache(maxsize=None)
        def dfs(i, j):
            if i == m:
                if j == n:
                    return True 
                else:
                    return j + 1 < n and p[j + 1] == '*' and dfs(i, j + 2)
            elif j == n:
                return False 
            else:
                if j + 1 == n or p[j + 1] != '*':
                    return (s[i] == p[j] or p[j] == '.') and dfs(i + 1, j + 1)
                else:
                    p1 = dfs(i, j + 2)
                    p2 = (s[i] == p[j] or p[j] == '.') and dfs(i + 1, j)

                    return p1 | p2
                    
        return dfs(0, 0)
```

### 9.移除盒子 https://leetcode.cn/problems/remove-boxes/description/

**思路：多一个参数k表示当前位置有前k个相同的前缀，然后就写区间dp即可，这里写的记忆化搜索，dp的三维翻译有点复杂**

**代码：**
```python
class Solution:
    def removeBoxes(self, boxes: List[int]) -> int:
        n = len(boxes)
        @lru_cache(maxsize=None)
        def dfs(i, j, k):
            if i > j:
                return 0 

            s = i
            while s + 1 <= j and boxes[i] == boxes[s + 1]:
                s += 1
            
            cnt = k + s - i + 1
            ans = cnt * cnt + dfs(s + 1, j, 0)
            for m in range(s + 2, j + 1):
                if boxes[i] == boxes[m]:
                    ans = max(ans, dfs(s + 1, m - 1, 0) + dfs(m, j, cnt))
            
            return ans 
        return dfs(0, n - 1, 0)
```

### 10.合并石头的最小代价 https://leetcode.cn/problems/minimum-cost-to-merge-stones/description/

```python
class Solution:

        n = len(stones)
        if (n - 1) % (k - 1) != 0:
            return -1 
        
        pre = [0] + list(accumulate(stones))

        dp = [[0] * n for _ in range(n)]
        for i in range(n - 2, -1, -1):
            for j in range(i + 1, n):
                ans = inf 
                for m in range(i, j, k - 1):
                    ans = min(ans, dp[i][m] + dp[m + 1][j])
                if (j - i) % (k - 1) == 0:
```

### 1.统计特殊整数 https://leetcode.cn/problems/count-special-integers/description/

**思路：数位dp板子题，整个一大段都是模板。主要是用记忆化搜索来写的，比较好写一点。他有四个参数，其含义如下：1.i表示填到第几位了；2.mask用于记录是否重复填写过，他是用一个int整数来压缩了集合；3.is_limit表示前面填的数是否都和n对应，如果有对应就是True，否则为False；4.is_num表示前面是否填了数字，如果填了就是True，否则为False。**

```python
class Solution:
    def countSpecialNumbers(self, n: int) -> int:
        s = str(n)
        @lru_cache(maxsize=None)
        def dfs(i, mask, is_limit, is_num):
            if i == len(s):
                return 1 if is_num else 0 
            res = 0 
            if not is_num:
                res = dfs(i + 1, mask, False, False)
            low = 0 if is_num else 1 
            up = int(s[i]) if is_limit else 9
            for d in range(low, up + 1):
                if mask >> d & 1 == 0:
                    res += dfs(i + 1, mask | (1 << d), is_limit and d == up, True)
```

### 4.至少有一位重复数字 https://leetcode.cn/problems/numbers-with-repeated-digits/

**思路：第一题其实是1到n中没有重复数字的统计，那么本题的至少有一位重复数字，就是总的个数减去没有重复数字的个数，即答案**

**代码：**
```python

        s = str(n)
        @lru_cache(maxsize=None)
        def dfs(i, mask, is_limit, is_num):
```

### 2.网格图中得分最大的路径 https://leetcode.cn/problems/maximum-path-score-in-a-grid/?envType=daily-question&envId=2026-04-30

**思路：网格图dp。定义dfs(i, j, t)表示从(0, 0)到达(i, j)位置，花费不超过t时，能够获得的最大分数，答案就是dfs(m - 1, n - 1, k)**

**代码：**
```python
class Solution:
    def maxPathScore(self, grid: List[List[int]], k: int) -> int:
        cost = {0: 0, 1: 1, 2: 1}
        val = {0: 0, 1: 1, 2: 2}
        m, n = len(grid), len(grid[0])
        @lru_cache(maxsize=None)
        def dfs(i, j, t):
            if i < 0 or j < 0 or t < 0:
                return -inf
            if i == 0 and j == 0:
                return val[grid[i][j]] if t >= 0 else -inf
            return max(dfs(i - 1, j, t - cost[grid[i][j]]), dfs(i, j - 1, t - cost[grid[i][j]])) + val[grid[i][j]]
        ans = dfs(m - 1, n - 1, k)
        dfs.cache_clear()
        return ans if ans > -inf else -1
```

## dp的枚举优化

### 1.规划兼职工作 https://leetcode.cn/problems/maximum-profit-in-job-scheduling/

**思路：排序之后，用二分来优化枚举，从而实现dp的优化**

**代码：**
```python
class Solution:
    def jobScheduling(self, startTime: List[int], endTime: List[int], profit: List[int]) -> int:
        n = len(startTime)
        jobs = sorted(zip(endTime, startTime, profit))
        dp = [0] * (n + 1)
        for i, (_, st, p) in enumerate(jobs):
            j = bisect_right(jobs, (st, inf, inf), 0, i)
            dp[i + 1] = max(dp[i], dp[j] + p)
        return dp[-1]
```

### 2.K个逆序对数组 https://leetcode.cn/problems/k-inverse-pairs-array/

**思路：从画图分析中，我们可以看到每个位置的状态依赖是上一行其左侧格子的不断累加。因此为了优化掉for的枚举行为，我们在状态转移之前可以预处理出前缀和数组，这样状态转移的枚举行为就优化到了O(1)**

**代码：**
```python
class Solution:
    def kInversePairs(self, n: int, k: int) -> int:
        mod = 10 ** 9 + 7 
        dp = [[0] * (k + 1) for _ in range(n + 1)]
        dp[0][0] = 1
        for i in range(n):
            pre = [0] + list(accumulate(dp[i]))
            for j in range(k + 1):
                dp[i + 1][j] = (pre[j + 1] - pre[max(j - i, 0)]) % mod
        return dp[-1][-1]
```

### 3.前缀平方和序列 https://www.nowcoder.com/practice/5a4a7feaa70e45f1ad69ac86f0e4b246?channelPut=tracker2

**思路：二维dp + 状态数优化 + 枚举优化。首先要根据数据量进行dp的状态定义：$dp[i][j]$表示当前第i个前缀和为j * j，然后就可以得到状态转移方程为$dp[i][j] = sum(dp[i - 1][k])\ (k < j)$，对于前i - 1个前缀和来说，只要其小于k，我们都可以通过$j * j - k * k$找到一个数来满足状态转移，然后这个转移方程我们可以用前缀和来进行枚举优化，最终累加$dp[n][j]$到ans中得到结果**

### 2.两个排列的最长公共子序列 https://www.luogu.com.cn/problem/P1439

**思路：从数据量可以知道，用经典的$O(n^2)$序列dp是无法通过本题的。考虑到是两个排列的最长公共子序列，我们可以把a数组的元素的索引，映射到b数组，之后再在b数组上做一个LIS就可以得到答案了。为什么？因为序列映射保证两个数组都是递增状态，并且值对应，所以说算一个LIS是正确的**

### 1.Longest Subsequence https://ac.nowcoder.com/acm/contest/127703/D

**思路：他和位置无关，之和具体的值有关。因此，我们可以用hash表来写一个值域dp，其转移方程就是$d[x] = fmax(d.get(x - 1,\ 0),\ d.get(x + 1,\ 0)) + 1$，也就是从题目说的两种值域情况中进行转移，用get方法写比较方便，对于这种值域dp来说**

**代码：**
```python
import sys
from collections import defaultdict

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    a = LII()

    d = defaultdict(int)
    for x in a:
        d[x] = fmax(d.get(x - 1, 0), d.get(x + 1, 0)) + 1

    print(max(d.values()))

# t = 1
t = II()
for _ in range(t):
    solve()
```

### 3.Not Adjacent 2 https://atcoder.jp/contests/abc456/tasks/abc456_d

**思路：序列dp/值域dp。定义$dp[i]$表示以索引i字符结尾的合法序列方案数，状态转移就是每次遍历到位置时，额外添加除自己以外的其他字符，并且+1，因为单独也算一个方案，最终输出总的结果即可**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

mod = 998244353

def solve():
    s = I()

    dp = [0, 0, 0]
    mp = {'a': 0, 'b': 1, 'c': 2}
    total = 0

    for x in s:
        i = mp[x]
        add = (1 + total - dp[i]) % mod
        dp[i] = (dp[i] + add) % mod
        total = (total + add) % mod

    print(total % mod)

t = 1
# t = II()
for _ in range(t):
    solve()
```

### 1.香槟塔（类似杨辉三角） https://leetcode.cn/problems/champagne-tower/description/?envType=daily-question&envId=2026-02-14

**思路：严格来说，应该是一道dp。这里用了滚动数组nxt来实现，和一般固定的m * n动态数组不同，这里每层的n是不一样的。接着，他的状态转移方程其实就是，当上一层如果有多余的酒流下来即$x > 1$，那么就可以转移到j和j + 1上面，之后滚动更新即可**

**这里注意下，由于题目只要求算到query_row，我们又是初始化拿poured算的，那就可以只算到query_row**

**代码：**
```python
class Solution:
    def champagneTower(self, poured: int, query_row: int, query_glass: int) -> float:
        cur = [float(poured)]
        for i in range(1, query_row + 1):
            nxt = [0.0] * (i + 1)
            for j, x in enumerate(cur):
                if x > 1:
                    nxt[j] += (x - 1) / 2
                    nxt[j + 1] += (x - 1) / 2 
            cur = nxt 
        return min(1.0, cur[query_glass])
```
