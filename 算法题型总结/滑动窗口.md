## 不定长滑动窗口问题

**注意：while一般用>或者>=来写，这样就只需要更新左端点，而不用去动右端点。具体答案怎么取，需要通过题意判断是越短越合法，还是越长越合法**

### 求子数组长度，越短越合法/越长越合法

#### 1.无重复字符的最长字串 https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/

**思路：越短越不容易重复，也就越合法**

**代码：**
```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        ans = left = 0
        cnt = defaultdict(int)  # 维护从下标 left 到下标 right 的字符及其出现次数
        for right, c in enumerate(s):
            cnt[c] += 1
            while cnt[c] > 1:  # 窗口内有重复字母
                cnt[s[left]] -= 1  # 移除窗口左端点字母
                left += 1  # 缩小窗口
            ans = max(ans, right - left + 1)  # 更新窗口长度最大值
        return ans
```

#### 2.长度最小的子数组 https://leetcode.cn/problems/minimum-size-subarray-sum/description/

**思路：越长和越大，也就越合法**

**代码：**
```python
class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        n = len(nums)
        ans = n + 1  # 也可以写 inf
        s = left = 0
        for right, x in enumerate(nums):  # 枚举子数组右端点
            s += x
            while s >= target:  # 满足要求
                ans = min(ans, right - left + 1)
                s -= nums[left]
                left += 1  # 左端点右移
        return ans if ans <= n else 0
```

#### 3.最小覆盖字串 https://leetcode.cn/problems/minimum-window-substring/description/

**思路：主要是Counter的妙用冷知识，Counter之间可以进行比较的，主要是比较相同键的值**

**代码：**
```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        inf = 10 ** 9 + 7
        ansl = ansr = -1
        ans = inf 

        target = Counter(t)
        cnt = Counter()
        l = 0 
        for r, x in enumerate(s):
            cnt[x] += 1
            while cnt >= target:
                if r - l + 1 < ans:
                    ans = r - l + 1
                    ansl, ansr = l, r
                cnt[s[l]] -= 1
                if cnt[s[l]] == 0:
                    del cnt[s[l]]
                l += 1
                
        return s[ansl:ansr + 1] if ans < inf else ""
```

#### 4.替换字串得到平衡字符串 https://leetcode.cn/problems/replace-the-substring-for-balanced-string/

**思路：依旧邪修做法，先处理出原字符串s多出来的不要的字符，这是需要修改的字符，然后跑一遍上面那题的最小覆盖长度，就是我们需要的答案**

**代码：**
```python
class Solution:
    def balancedString(self, s: str) -> int:
        n = len(s)
        s_cnt = Counter(s)

        target = 'QWER' * (n // 4)
        target_cnt = Counter(target)

        need_cnt = s_cnt - target_cnt
        if not need_cnt:
            return 0

        cnt = Counter()
        l = 0
        ans = 10 ** 18 
        for r, x in enumerate(s):
            cnt[x] += 1
            while cnt >= need_cnt:
                ans = min(ans, r - l + 1)
                cnt[s[l]] -= 1
                if cnt[s[l]] == 0:
                    del cnt[s[l]]
                l += 1
        return ans 
```

### 求子数组个数，越短越合法/越长越合法/恰好

#### 1.乘积小于k的子数组 https://leetcode.cn/problems/subarray-product-less-than-k/description/

**思路：越短乘积越小，越满足条件**

**代码：**
```python
class Solution:
    def numSubarrayProductLessThanK(self, nums: List[int], k: int) -> int:
        if k <= 1:
            return 0
        ans = left = 0
        prod = 1
        for right, x in enumerate(nums):
            prod *= x
            while prod >= k:  # 不满足要求
                prod //= nums[left]
                left += 1  # 缩小窗口
            # 对于固定的 right，有 right-left+1 个合法的左端点
            ans += right - left + 1
        return ans
```

#### 2.统计最大元素至少出现k次的子数组 https://leetcode.cn/problems/count-subarrays-where-max-element-appears-at-least-k-times/description/

**思路：越长出现次数越多，越满足条件**

**代码：**
```python
class Solution:
    def countSubarrays(self, nums: List[int], k: int) -> int:
        mx = max(nums)
        ans = cnt_mx = left = 0
        for x in nums:
            if x == mx:
                cnt_mx += 1
            while cnt_mx >= k:
                if nums[left] == mx:
                    cnt_mx -= 1
                left += 1
            ans += left
        return ans
```

#### 3.和相同的二元子数组 https://leetcode.cn/problems/binary-subarrays-with-sum/

**思路：用两次滑动窗口解决，用两个至少/两个至多来表示恰好**

**代码：**
```python
class Solution:
    def numSubarraysWithSum(self, nums: List[int], goal: int) -> int:
        def solve(nums, goal):
            ans = cnt = 0
            left = 0
            s = 0
            for right, v in enumerate(nums):
                s += v
                while right >= left and s > goal:
                    s -= nums[left]
                    left += 1
                ans += left
            return ans 
        return solve(nums, goal - 1) - solve(nums, goal)
        '''
        这里记住：大 - 小就行，不容易出错
        > goal - 1 - > goal
        举个例子，goal为4，那么> goal - 1就是4 5 6...
        然后> goal就是5 6 7...，两者相减就恰好是goal为4

        >= goal - >= goal + 1
        举个例子，goal为4，那么>= goal就是4 5 6...
        然后>= goal + 1就是5 6 7...，两者相减就恰好是goal为4
        '''
```

#### 4.K个不同整数的子数组 https://leetcode.cn/problems/subarrays-with-k-different-integers/description/

**思路：恰好滑动窗口类型，直接复用上面那题的做法即可**

**代码：**
```python
class Solution:
    def subarraysWithKDistinct(self, nums: List[int], k: int) -> int:
        def solve(goal):
            cnt = Counter()
            ans = l = 0
            for r, x in enumerate(nums):
                cnt[x] += 1
                while len(cnt) >= goal:
                    cnt[nums[l]] -= 1
                    if cnt[nums[l]] == 0:
                        del cnt[nums[l]]
                    l += 1
                ans += l 
            return ans 
        return solve(k) - solve(k + 1)
```

#### 6.Kevin喜欢零 https://www.nowcoder.com/practice/5827292a5d464d6391a886e96254b008?channelPut=tracker2

**思路：大方向是恰好滑动窗口类型，但是结尾0不具有单调性，因此需要进行转化，那该如何转化？考虑后缀0，那就应该能被10整除，因此就可以进行因子的分解，也就是$10^k = (2 * 5)^k = 2^k * 5^k$，那就提前预处理出每个数对应的因子个数，这样窗口就具备了单调性，然后就可以跑一个恰好滑动窗口了**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n, k = MII()
    a = LII()

    cnt2 = []
    cnt5 = []
    for x in a:
        c2 = c5 = 0
        while x % 2 == 0:
            c2 += 1
            x //= 2
        while x % 5 == 0:
            c5 += 1
            x //= 5
        cnt2.append(c2)
        cnt5.append(c5)

    def func(k):
        if k == 0:
            return n * (n + 1) // 2

        ans = 0
        sum2 = sum5 = 0
        l = 0

        for r in range(n):
            sum2 += cnt2[r]
            sum5 += cnt5[r]
            while sum2 >= k and sum5 >= k:
                sum2 -= cnt2[l]
                sum5 -= cnt5[l]
                l += 1
            ans += l

        return ans

    ans = func(k) - func(k + 1)
    print(ans)


t = II()
for _ in range(t):
    solve()
```

#### 1·至少有k个重复字符的最长字串 https://leetcode.cn/problems/longest-substring-with-at-least-k-repeating-characters/

**思路：由于事先不确定target到底包含哪些字符，所以说这题不能用邪修Counter做法，那就要用枚举包含的字符个数unique以及滑动窗口计算有几个字符有效的valid值，最终枚举26种可能性，然后取其中的max长度就是答案**

**代码：**
```python
class Solution:
    def longestSubstring(self, s: str, k: int) -> int:
        n = len(s)
        ans = 0
        
        for unique in range(1, 27):
            cnt = Counter()
            l = valid = 0
            
            for r, c in enumerate(s):
                cnt[c] += 1
                if cnt[c] == k:
                    valid += 1
                
                while len(cnt) > unique:
                    left = s[l]
                    if cnt[left] == k:
                        valid -= 1
                    cnt[left] -= 1
                    if cnt[left] == 0:
                        del cnt[left]
                    l += 1

                if valid == unique:
                    ans = max(ans, r - l + 1)
        
        return ans
```

#### 2.小红关鸡 https://www.nowcoder.com/practice/5982b0bfe2fe4ed2a872fd5383271f5f?channelPut=tracker2

**思路：不定长滑动窗口找最大覆盖个数。严格来说，其实是一个双指针。我们就根据不超过区间长度k这个条件，先排序，再用不定长滑动窗口找到合法区间范围中，能够覆盖的最大点数，最终再除总个数，就是答案**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n, k = MII()
    a = LII()

    if n == 1:
        print(1.0)
        return

    a.sort()
    cnt = ans = 0
    l = 0
    pre = -inf
    for r, x in enumerate(a):
        cnt += 1
        if pre == -inf:
            pre = x
            continue
        while x - pre > k:
            cnt -= 1
            pre = a[l + 1]
            l += 1
        ans = fmax(ans, cnt)
    print(ans / n)

# t = II()

    solve()
```
