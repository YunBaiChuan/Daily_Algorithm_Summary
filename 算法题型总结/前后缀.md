### 3.表现良好的最长时间段 https://leetcode.cn/problems/longest-well-performing-interval/

**思路：也是要先转化成1或-1，此时问题变成子数组和$s > 0$的最长子数组，而不是固定的等于0。由于1或-1这个数组的特殊性，假设我们碰到-3这个值，那么就需要在hash表中查找-4即可，此时$-3 - (-4) = 1$满足条件，那怎么不去找-5，-6，-7等这些？**

**因为你-5，肯定是-4的基础上-1，所以说-5前面必然会有个-4，同理-6前面必然有个-5，-7前面必然有-6等等，这个逐步增加的特性，让我们只需要找$pre[i] - 1$这个值就行，他就是可能满足条件的最长子数组了**

**代码：**
```python
class Solution:
    def longestWPI(self, hours: List[int]) -> int:
        n = len(hours)

        pre = [0] * (n + 1)
        for i, x in enumerate(hours):
            pre[i + 1] = pre[i] + (1 if x > 8 else -1)
        
        ans = 0
        d = defaultdict(int)
        for i in range(n + 1):
            if pre[i] > 0:
                ans = max(ans, i)
                
            if pre[i] - 1 in d:
                ans = max(ans, i - d[pre[i] - 1])
            elif pre[i] not in d:
                d[pre[i]] = i 
        
        return ans 
```

### 4.使数组和能够被p整除 https://leetcode.cn/problems/make-sum-divisible-by-p/description/

**思路：前缀和，hash表，模运算。由于改成了删除最小，所以说不管有没有更新答案，都需要进行$d[pre[i]]$的更新，以保证取得更小的长度**

**代码：**
```python
class Solution:
    def minSubarray(self, nums: List[int], p: int) -> int:
        n = len(nums)
        
        pre = [0] * (n + 1)
        for i, x in enumerate(nums):
            pre[i + 1] = (pre[i] + x) % p 
        
        mod = pre[-1]
        if mod == 0:
            return 0
        
        ans = n
        d = defaultdict(int)
        for i in range(n + 1):
            if (pre[i] + p - mod) % p in d:
                ans = min(ans, i - d[(pre[i] + p - mod) % p])
            d[pre[i]] = i 
    
        return ans if ans < n else -1
```

### 5.每个元音包含偶数次的最长字符串 https://leetcode.cn/problems/find-the-longest-substring-containing-vowels-in-even-counts/description/

**思路：前缀和，hash表，奇偶性，位运算。我们要算每个元音包含偶数次的最长字符串，从奇偶性可知：1.奇奇为偶；2.偶偶为偶。那么，我们可以用位运算来存放这几个元音的奇偶状态，并且利用异或和奇偶性的适配性，直接前缀和+hash表找和自己一样的状态就行了**

**代码：**
```python
class Solution:
    def findTheLongestSubstring(self, s: str) -> int:
        n = len(s)

        d = defaultdict(int)
        d[0] = -1

        def check(c):
            if c == 'a': return 0
            elif c == 'e': return 1 
            elif c == 'i': return 2 
            elif c == 'o': return 3 
            elif c == 'u': return 4 
            else: return -1

        ans = status = 0
        for i in range(n):
            m = check(s[i])
            if m != -1:
                status ^= 1 << m 
            
            if status in d:
                ans = max(ans, i - d[status])
            else: # 由于是自己和自己的关系，因此这里写else就行
                d[status] = i 
        
        return ans 
```

### 6.最大的以1为边界的正方形 https://leetcode.cn/problems/largest-1-bordered-square/description/

**思路：枚举左上角及边长，然后算出右下角，最后再用前缀和快速求边框值（两个前缀和矩形相减）并且判断是否为周长，然后不断更新答案**

**代码：**
```python
class Solution:
    def largest1BorderedSquare(self, grid: List[List[int]]) -> int:
        m, n = len(grid), len(grid[0])
        pre = [[0] * (n + 1) for _ in range(m + 1)]
        for i, row in enumerate(grid):
            for j, x in enumerate(row):
                pre[i + 1][j + 1] += pre[i][j + 1] + pre[i + 1][j] - pre[i][j] + x 
        
        if pre[-1][-1] == 0:
            return 0

        def cal(a, b, c, d):
            a += 1; b += 1; c += 1; d += 1
            return pre[c][d] - pre[a - 1][d] - pre[c][b - 1] + pre[a - 1][b - 1]

        ans = 1
        for a in range(m):
            for b in range(n):
                for k in range(ans, min(m, n) + 1):
                    c = a + k; d = b + k
                    if c >= m or d >= n:
                        break
                    if cal(a, b, c, d) - cal(a + 1, b + 1, c - 1, d - 1) == 4 * k:
                        ans = k + 1
        return ans * ans
```

### 1.接雨水 https://leetcode.cn/problems/trapping-rain-water/description/

**思路：对于每一个位置来说，算他的接水量，最后累加起来即可。那每个位置的接水量其实就需要比较当前位置，左右两端的最大值，其中的最小值-本身柱子高度就是当前位置能够接的水量。那左右两端的最大值怎么预处理出来？用前后缀分解，求最大值，推荐以下写法，用n的空间，和前缀和n + 1区间有所不同，下面这种写法不容易出错**

**代码：**
```python
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        lmax, rmax = [0] * n, [0] * n

        lmax[0] = height[0]
        for i in range(1, n):
            lmax[i] = max(lmax[i - 1], height[i])
        
        rmax[-1] = height[-1]
        for i in range(n - 2, -1, -1):
            rmax[i] = max(rmax[i + 1], height[i])

        ans = 0 
        for i in range(n):
            ans += max(0, min(lmax[i], rmax[i]) - height[i])
        return ans 
```

### 2.Eat The Candy https://ac.nowcoder.com/acm/contest/127703/E

**思路：贪心以及前后缀和分解。题目要算以最优方式进行操作的情况下，最大的max值为多少。由于我们的操作是两个相邻的数同时-1，然后可以任选一个数进行+1，可以看到，整体的和其实是在不断减小的。因此，当我们枚举某个值为最大值时，我们不会去动这个最大值，而是去算他左右两边的操作次数，然后加在他身上，就成了一个待选的最大值。**

**那我们怎么快速处理左右两边的操作次数？并且进行最大化？那就需要用贪心和前后缀和分解了。具体来说就是，先贪心地算最大操作次数，相邻的如果能操作，就操作了，然后累加操作次数算他的前缀和；后缀和也是同理。最终，枚举a中每个数为最大值时，算出一个待选最大值，然后不断取fmax，就能得到最终答案。**

**本题难点有两个：1.怎么想到最大化最大值；2.前后缀边界处理**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    a = [0] + LII() + [0]

    tmp_a = a[::]
    pre = [0] * (n + 1)
    for i in range(2, n + 1):
        mn = fmin(tmp_a[i - 1], tmp_a[i])
        pre[i] = pre[i - 1] + mn
        tmp_a[i] -= mn

    tmp_b = a[::]
    suf = [0] * (n + 2)
    for i in range(n - 1, 0, -1):
        mn = fmin(tmp_b[i], tmp_b[i + 1])
        suf[i] = suf[i + 1] + mn
        tmp_b[i] -= mn

    ans = 0
    for i in range(1, n + 1):
        ans = fmax(ans, pre[i - 1] + a[i] + suf[i + 1])
    print(ans)

# t = 1
t = II()
for _ in range(t):
    solve()
```

### 3.三个无重叠子数组的最大和 https://leetcode.cn/problems/maximum-sum-of-3-non-overlapping-subarrays/description/

**思路：前后缀分解，然后在算pre和suf的时候，加了点dp在里面。对于这种三个东西求和的问题，我们要优先想到枚举中间，然后左边两遍进行前后缀分解**

**代码：**
```python
class Solution:
    def maxSumOfThreeSubarrays(self, nums: List[int], k: int) -> List[int]:
        n = len(nums)

        sums = [0] * n
        s = l = 0
        for r, x in enumerate(nums):
            s += x
            if r < k - 1:
                continue
            sums[l] = s 
            s -= nums[l]
            l += 1
        
        pre = [0] * (n - k + 1)
        for i in range(1, n - k + 1):
            if sums[i] > sums[pre[i - 1]]:
                pre[i] = i
            else:
                pre[i] = pre[i - 1]

        suf = [0] * (n - k + 1)
        suf[n - k] = n - k
        for i in range(n - k - 1, -1, -1):
            if sums[i] >= sums[suf[i + 1]]: 
                suf[i] = i
            else:
                suf[i] = suf[i + 1]

        mx = 0
        res = []
        for j in range(k, n - 2 * k + 1):
            left = pre[j - k] 
            right = suf[j + k] 
            total = sums[left] + sums[j] + sums[right]
            if total > mx:
                mx = total
                res = [left, j, right]
        
        return res
```

### 4.构造乘积矩阵 https://leetcode.cn/problems/construct-product-matrix/description/?envType=daily-question&envId=2026-03-24

**思路：首先这题不能用整体乘积，然后除当前数，为什么？因为值域很大，会爆空间和时间。那先取mod再算呢？也不行，为什么？因为取mod之后有0的情况出现，此时除法就失效了。因此，我们要用前后缀分解来完成这道题。它相当于是个二维矩阵的前后缀分解，我们只需要把他拉成一维的数组即可，此时每个位置的答案就是pre * suf了**

**代码：**
```python
class Solution:
    def constructProductMatrix(self, grid: List[List[int]]) -> List[List[int]]:
        MOD = 12345
        n, m = len(grid), len(grid[0])
        p = [[0] * m for _ in range(n)]

        suf = 1
        for i in range(n - 1, -1, -1):
            for j in range(m - 1, -1, -1):
                p[i][j] = suf
                suf = suf * grid[i][j] % MOD

        pre = 1
        for i, row in enumerate(grid):
            for j, x in enumerate(row):
                p[i][j] = p[i][j] * pre % MOD
                pre = pre * x % MOD

        return p
```

### 5.超级洗衣机 https://leetcode.cn/problems/super-washing-machines/

**思路：前后缀分解 + 思维。我们对每个洗衣机进行单点分析，有四种情况：1.左需 > 0 and 右需 > 0；2.左需 < 0 and 右需 > 0；3.左需 > 0 and 右需 < 0；4.左需 < 0 and 右需 < 0。讨论出来，其实就是两种结果：除了1的情况，结果是$l + r$以外，其他都是$max(abs(l),\ abs(r))$。因此，最少的转移次数就是每个单点洗衣机操作次数，算出来之后取最大值即可，因为最大值是瓶颈**

**代码：**
```python
class Solution:
    def findMinMoves(self, machines: List[int]) -> int:
        n = len(machines)
        suf = [0] * (n + 1)
        for i in range(n - 1, -1, -1):
            suf[i] = suf[i + 1] + machines[i]
        
        if suf[0] % n != 0:
            return -1 

        need = suf[0] // n
        ans = 0
        pre = 0
        for i, x in enumerate(machines):
            l, r = need * i - pre, need * (n - i - 1) - suf[i + 1]
            if l > 0 and r > 0:
                ans = max(ans, l + r)
            else:
                ans = max(ans, max(abs(l), abs(r)))
            pre += x
        return ans 
```

### 6.等值距离和 https://leetcode.cn/problems/sum-of-distances/description/?envType=daily-question&envId=2026-04-23

**思路：hash表 + 前缀和，有点前后缀分解的意思。思路还是容易想到，但是left和right的计算需要观察出来，这点比较难想**

**代码：**
```python
class Solution:
    def distance(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [0] * n 

        d = defaultdict(list)
        for i, x in enumerate(nums):
            d[x].append(i)
        

        for k, row in d.items():
            m = len(row)
            if m == 1:
                continue 

            pre = [0] * (m + 1)
            for i in range(m):
                pre[i + 1] = pre[i] + row[i]
            
            for j, x in enumerate(row):
                left = j * x - pre[j]
                right = pre[-1] - pre[j] - (m - j) * x  
                ans[x] = left + right 
            
        return ans 
```

### 7.在下标间移动的最小代价 https://leetcode.cn/problems/minimum-cost-to-move-between-indices/description/

**思路：前后缀分解。这题的关键在于：如何想到前后缀分解的？主要是题目问的是在下标间移动的最小代价，比如说是从i -> j -> k的代价，其实就是i -> k的代价，因为数组是严格递增的。因此，我们就可以预处理出前后缀的最小代价数组，之后就可以快速回答查询了，挺巧妙的**

**代码：**
```python
class Solution:
    def minCost(self, nums: list[int], queries: list[list[int]]) -> list[int]:
        n = len(nums)
        sum_l = [0] * n  # sum_l[i] 等于从 i 移动到 0 的代价和
        sum_r = [0] * n  # sum_r[i] 等于从 0 移动到 i 的代价和
        for i in range(1, n):
            # 往左走 i -> i-1
            if i < n - 1 and nums[i] - nums[i - 1] > nums[i + 1] - nums[i]:  # closest(i) = i+1
                cost = nums[i] - nums[i - 1]  # 只能用方式一往左走
            else:
                cost = 1
            sum_l[i] = sum_l[i - 1] + cost

            # 往右走 i-1 -> i
            if i > 1 and nums[i - 1] - nums[i - 2] <= nums[i] - nums[i - 1]:  # closest(i-1) = i-2
                cost = nums[i] - nums[i - 1]  # 只能用方式一往右走
            else:
                cost = 1
            sum_r[i] = sum_r[i - 1] + cost

        ans = [0] * len(queries)
        for i, q in enumerate(queries):
            l, r = q
            if l < r:
                # cost(0 -> r) - cost(0 -> l) = cost(l -> r)
                ans[i] = sum_r[r] - sum_r[l]
            else:
                # cost(l -> 0) - cost(r -> 0) = cost(l -> r)
                ans[i] = sum_l[l] - sum_l[r]
        return ans
```
