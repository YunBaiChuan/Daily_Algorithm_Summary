## 二叉搜索树

### 1.逐光者 https://hydro.ac/d/CUIT_DSA/p/835?tid=6935792d94e63f6489a5d4c4

**思路：写个node类，然后dfs建树，找最后一个节点的父亲，为什么不能用n // 2？因为二叉搜索树如果构成了一条链，那他就不满足这个性质了，只有完全二叉树/堆才满足这个性质。完事之后dfs建图，再用两次bfs算一下树的直径即可**

**代码：**
```python
import sys
from collections import deque, defaultdict

input = lambda: sys.stdin.readline().strip()

def II():
    return int(input())

def LII():
    return list(map(int, input().split()))

class Node:
    __slots__ = ('val', 'left', 'right') # 告诉系统Node只有这几个属性，不再创建其他属性，可以省内存及提速
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def solve():
    n = II()
    a = LII()

    # 1. 递归建树
    last_parent_val = None

    def insert(root, val):
        nonlocal last_parent_val
        if root is None:  # 找到空位
            new_node = Node(val)
            return new_node

        if val < root.val:
            root.left = insert(root.left, val)
            # 只要挂上去且是最后一个插入值，就记录父节点
            if root.left.val == val and val == a[-1]:
                last_parent_val = root.val
        else:
            root.right = insert(root.right, val)
            if root.right.val == val and val == a[-1]:
                last_parent_val = root.val
        return root

    root = Node(a[0])
    for val in a[1:]:
        insert(root, val)

    # 2. 建无向图
    graph = defaultdict(list)

    def dfs_build(node):
        if node is None:
            return
        if node.left:
            graph[node.val].append(node.left.val)
            graph[node.left.val].append(node.val)
            dfs_build(node.left)
        if node.right:
            graph[node.val].append(node.right.val)
            graph[node.right.val].append(node.val)
            dfs_build(node.right)

    dfs_build(root)

    # 3. 两次bfs求树的直径
    def bfs(start):
        dist = {start: 0}
        q = deque([start])
        while q:
            u = q.popleft()
            for v in graph[u]:
                if v not in dist:
                    dist[v] = dist[u] + 1
                    q.append(v)
        farthest = max(dist, key=lambda x: dist[x])
        return farthest, dist[farthest]

    u, _ = bfs(a[0]) # 第1次：找直径端点 u
    v, diameter = bfs(u) # 第2次：找另一端 v 及距离

    print(last_parent_val)
    print(diameter)

t = 1
for _ in range(t):
    solve()
```

### 2.将二叉搜索树变平衡 https://leetcode.cn/problems/balance-a-binary-search-tree/description/?envType=daily-question&envId=2026-02-09

**思路：首先要用中序遍历，把二叉搜索树变成一个有序数组，这是二叉搜索数的性质。然后再用把有序数组变平衡二叉搜索树这个算法，他主要是用中间值作为根，然后左右两遍递归下去**

**代码：**
```python
class Solution:
    # 94. 二叉树的中序遍历
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        def dfs(node: Optional[TreeNode]) -> None:
            if node is None:
                return
            dfs(node.left)        
            ans.append(node.val) 
            dfs(node.right)       

        ans = []
        dfs(root)
        return ans

    # 108. 将有序数组转换为二叉搜索树
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return None
        m = len(nums) // 2
        left = self.sortedArrayToBST(nums[:m])
        right = self.sortedArrayToBST(nums[m + 1:])
        return TreeNode(nums[m], left, right)

    def balanceBST(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        nums = self.inorderTraversal(root)
        return self.sortedArrayToBST(nums)
```

### 1.小红的树上删边 https://www.nowcoder.com/practice/675ca4c5402945148f99c46418c1b82a?channelPut=tracker2

**思路：首先要想到一个特判，那就是当节点n的个数为奇数时，必然不满足条件中的全偶数划分。然后就是贪心进行划分，什么意思？打个比方来说，我在4的时候进行划分，不如在2的时候就进行划分，这样就可以多一次划分，更优**

**所以说，我们可以统计子树大小，一旦遇到偶数就进行划分，这可以用递归dfs来做。具体来说，当我们执行自底向上的归部分时，一旦遇到偶数就进行划分，但不用真的去把子树划分，也不需要改变size的大小，为什么？**

**因为偶数 + 偶数，依旧是偶数，我们自底向上的过程中，是size的不断累加，我们只需要在size为偶数的时候划分就可以了**

**注意：首先需要用python3 + 栈扩展提交，这是题目说的。其次res要初始化为-1，原因是自底向上的归部分，最终是在根节点，此时统计的n为偶数，就是整个树的大小，我们无法进行划分，最终结果要-1，那就可以在初始化这里设置成-1即可**

### 2.分裂二叉树的最大乘积 https://leetcode.cn/problems/maximum-product-of-splitted-binary-tree/?envType=daily-question&envId=2026-01-07

**思路：有点像板子题，主要是怎么求每棵子树的值val，我们可以用dfs，然后在递归的归过程中，实现自底向上的计算，并且放到数组中进行保存。最终算一下s * (total - s)中的max，并且取mod输出即可**

**代码：**
```python
class Solution:
    def maxProduct(self, root: Optional[TreeNode]) -> int:
        mod = 10 ** 9 + 7

        sub_sum = []
        def dfs(root):
            if not root:
                return 0
            s = root.val + dfs(root.left) + dfs(root.right)
            sub_sum.append(s)
            return s
        total = dfs(root)

        ans = max(s * (total - s) for s in sub_sum)
        return ans % mod
```

## 静态区间最值问题(ST表)

### 1.静态区间最值 https://www.nowcoder.com/practice/831a314449d44ea0b1db90ca626bcd1a?channelPut=tracker2

**思路：我们需要回答多次查询的区间最值，每次$O(n)$切片查询的话，时间复杂度就来到$O(q * n)$必然超时，因此我们需要一种新的数据结构来预处理区间最值，方便我们进行$O(1)$的查询。那就需要用到st表了，他的精髓就是：一个区间的最值，就是这个区间左半边的最值，与右半边的最值中的最值，有重叠也没关系。那我们除了这种最重要的思想以外，还有个思想同样重要，那就是怎么分区间？我们用倍增，也就是每次*2翻倍得到区间，然后就可以预处理，并且这个复杂度是$O(nlogn)$的，单次查询的时候是$O(1)$。容易出错的部分应该是两个区间长度，一个i + (1 << (k - 1))，还有一个r - (1 << k) + 1。主要区别有两个，一个是从前往后算与从后往前算的区别；另一个是区间长度的区别，也就是k值。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
感觉是树状数组板子，并非，其实是st表
'''

def solve():
    n, q = MII()
    a = LII()

    LOG = 20
    st_min = [[0] * n for _ in range(LOG)]
    st_max = [[0] * n for _ in range(LOG)]

    # 0 层
    for i in range(n):
        st_min[0][i] = st_max[0][i] = a[i]

    # 倍增，注意递推的右区间开始应该是i + (1 << (k - 1))，不能有-1，那就成了左区间的结束位置，导致重叠了，并且是(1 << (k - 1))，因为是一半长
    for k in range(1, LOG):
        for i in range(n - (1 << k) + 1):
            st_min[k][i] = min(st_min[k - 1][i], st_min[k - 1][i + (1 << (k - 1))])
            st_max[k][i] = max(st_max[k - 1][i], st_max[k - 1][i + (1 << (k - 1))])

    # 预处理 log2 表，查询用，提前算长度对应的k防止后面查询重复调用log2函数
    log_table = [0] * (n + 1)
    for i in range(2, n + 1):
        log_table[i] = log_table[i // 2] + 1

    for _ in range(q):
        op, l, r = MII()
        l -= 1
        r -= 1
        k = log_table[r - l + 1]
        if op == 1:  # 区间最小值
            res = min(st_min[k][l], st_min[k][r - (1 << k) + 1])
        else:  # 区间最大值
            res = max(st_max[k][l], st_max[k][r - (1 << k) + 1])
        print(res)

t = 1
# t = II()
for _ in range(t):
    solve()
```

```python
class UnionFind:
    def __init__(self, n: int):
        # 一开始有 n 个集合 {0}, {1}, ..., {n-1}
        # 集合 i 的代表元是自己，大小为 1
        self._fa = list(range(n))  # 代表元
        self._size = [1] * n  # 集合大小
        self.cc = n  # 连通块个数

    # 返回 x 所在集合的代表元
    # 同时做路径压缩，也就是把 x 所在集合中的所有元素的 fa 都改成代表元
    def find(self, x: int) -> int:
        fa = self._fa
        # 如果 fa[x] == x，则表示 x 是代表元
        if fa[x] != x:
            fa[x] = self.find(fa[x])  # fa 改成代表元
```

### 6.找出知晓秘密的所有专家 https://leetcode.cn/problems/find-all-people-with-secret/

```python
    def __init__(self, n):

        self.secret = [True] + [False] * (n - 1)
        self.sets = n

            self.fa[x] = self.find(self.fa[x])

        if fx == fy:

        self.fa[fx] = fy
        self.size[fy] += self.size[fx]
        self.secret[fy] |= self.secret[fx]
        self.sets -= 1
        return True
```

### 7.好路径的数目 https://leetcode.cn/problems/number-of-good-paths/

**思路：代码好写，但是思路难想。首先以两个端点中val值最大的一个，进行从小到大排序。然后再利用并查集，按照最大val值小的合并到大的，并且统计每个集合中有多少个最大val值，然后也进行相应的合并。如果此时左右两端的最大val值相同，那就会产生好路径，并且条数为$mx\_cnt[fx] * mx\_cnt[fy]$。每合并一次就进行计算，然后返回好path数，最终累加到ans中即可**

**代码：**
```python
class UnionFind:
    def __init__(self, n, vals):
        self.fa = list(range(n))
        self.vals = vals
        self.mx_cnt = [1] * n 

    def find(self, x):
        if x != self.fa[x]:
            self.fa[x] = self.find(self.fa[x])
        return self.fa[x]

    def is_same(self, x, y):
        fx, fy = self.find(x), self.find(y)
        return fx == fy

    def union(self, x, y):
        fx, fy = self.find(x), self.find(y)
        path = 0
        if fx == fy:
            return 0
        if self.vals[fx] > self.vals[fy]:
            self.fa[fy] = fx 
        elif self.vals[fx] < self.vals[fy]:
            self.fa[fx] = fy 
        else:
            path += self.mx_cnt[fx] * self.mx_cnt[fy]
            self.fa[fx] = fy 
            self.mx_cnt[fy] += self.mx_cnt[fx]
        return path

class Solution:
    def numberOfGoodPaths(self, vals: List[int], edges: List[List[int]]) -> int:
        n = len(vals)
        uf = UnionFind(n, vals)
        edges.sort(key=lambda x: max(vals[x[0]], vals[x[1]]))

        ans = n 
        for x, y in edges:
            ans += uf.union(x, y)
        return ans 
```

### 11.小红的图上加边 https://www.nowcoder.com/practice/28c35b0e3f3b4a20aee8f0497ca6745e?channelPut=tracker2

**思路：并查集 + 贪心。也是同样的思路，遇到连通块问题，首先要想到并查集。合并完连通块之后，收集每个连通块中的最大值，然后排个序，再从小到大合并。此时的代价就是从小到大合并中，更大的一方，并且合并的次数为sets - 1，也就是连通块的个数-1，可以发现就是除去第一个位置的其他代价和，直接输出即可**

### 12.执行操作后的最小汉明距离 https://leetcode.cn/problems/minimize-hamming-distance-after-swap-operations/description/?envType=daily-question&envId=2026-04-21

**思路：并查集 + hash表。对于这种交换问题来说，其实就是能够交换的元素，就是一个连通块，那么连通块就可以用dfs或者并查集来做，我比较喜欢用并查集。最终统计的时候需要注意一下，直接统计汉明距离其实是不好算的，因此我们逆向思考，先统计匹配的个数，这个其实就是hash表的cnt次数取min，最终用总长度减去匹配个数，就是当前分组的答案，把他累加到最终ans中即可**

**代码：**
```python
class UnionFind:
    def __init__(self, n):
        self.fa = list(range(n))
        self.size = [1] * n
        self.sets = n

    def find(self, x):
        if x != self.fa[x]:
            self.fa[x] = self.find(self.fa[x])
        return self.fa[x]

    def is_same(self, x, y):
        fx, fy = self.find(x), self.find(y)
        return fx == fy

    def union(self, x, y):
        fx, fy = self.find(x), self.find(y)
        if fx == fy:
            return False
        self.fa[fx] = fy
        self.size[fy] += self.size[fx]
        self.sets -= 1
        return True

class Solution:
    def minimumHammingDistance(self, source: List[int], target: List[int], allowedSwaps: List[List[int]]) -> int:
        n = len(source)
        if not allowedSwaps:
            cnt = 0 
            for i in range(n):
                cnt += (source[i] != target[i])
            return cnt 
        
        uf = UnionFind(n)
        for u, v in allowedSwaps:
            if not uf.is_same(u, v):
                uf.union(u, v)
        
        groups = [[] for _ in range(n)] # 连通块分组
        for i, x in enumerate(uf.fa):
            fx = uf.find(x)
            groups[fx].append(i)
        
        ans = 0 
        for i, row in enumerate(groups):
            if not row:
                continue 
                
            res = 0
            cnt1, cnt2 = Counter(), Counter()
            for j, x in enumerate(row):
                cnt1[source[x]] += 1 
                cnt2[target[x]] += 1
            
            for k, v in cnt1.items(): # 匹配个数
                res += min(v, cnt2[k])
            
            ans += len(row) - res # 不匹配个数（汉明距离）
        return ans 
```

### 13.按公因数计算最大组件大小 https://leetcode.cn/problems/largest-component-size-by-common-factor/description/

**思路：质因数分解 + 并查集 + hash表。我们每次质因数分解，只需要用hash表记录第一次出现的质因数位置，后面再次遇到时，说明前面的数也有这个因子，就可以进行并查集合并，而且不需要更新这个因子位置，根据并查集性质来说**

**代码：**
```python
class UnionFind:
    def __init__(self, n):
        self.fa = list(range(n))
        self.size = [1] * n
        self.sets = n

    def find(self, x):
        if x != self.fa[x]:
            self.fa[x] = self.find(self.fa[x])
        return self.fa[x]

    def is_same(self, x, y):
        fx, fy = self.find(x), self.find(y)
        return fx == fy

    def union(self, x, y):
        fx, fy = self.find(x), self.find(y)
        if fx == fy:
            return False
        self.fa[fx] = fy
        self.size[fy] += self.size[fx]
        self.sets -= 1
        return True

class Solution:
    def largestComponentSize(self, nums: List[int]) -> int:
        n = len(nums)
        d = defaultdict(int)
        uf = UnionFind(n)
        for i, x in enumerate(nums):
            for j in range(2, isqrt(x) + 1):
                if x % j == 0:
                    if j not in d:
                        d[j] = i 
                    else:
                        uf.union(d[j], i)
                    while x % j == 0:
                        x //= j 
            if x > 1:
                if x not in d:
                    d[x] = i 
                else:
                    uf.union(d[x], i)
        return max(uf.size)
```

```python
class LazySegmentTree:

    def __init__(self, arr):
        if isinstance(arr, int):
            arr = [0] * arr
        self.n = len(arr)
        self.size = 1 << (self.n - 1).bit_length()

        self.val = [0] * (2 * self.size)
        self.todo = [0] * (2 * self.size)

        for i in range(self.n):
            self.val[self.size + i] = arr[i]

        for i in range(self.size - 1, 0, -1):
            self.val[i] = self.val[2 * i] + self.val[2 * i + 1]

    def _apply(self, node, l, r, tag):
        """应用标记到节点"""
        if tag:
            self.val[node] = (r - l + 1) - self.val[node]
            self.todo[node] ^= 1

    def _push(self, node, l, r):
        """下传标记"""
        if self.todo[node]:
            mid = (l + r) // 2
            self._apply(2 * node, l, mid, self.todo[node])
            self._apply(2 * node + 1, mid + 1, r, self.todo[node])
            self.todo[node] = 0

    def update(self, ql, qr):
        """区间翻转"""
        def _update(node, l, r):
            if ql > r or qr < l:
                return
            if ql <= l and r <= qr:
                self._apply(node, l, r, 1)
                return
            self._push(node, l, r)
            mid = (l + r) // 2
            _update(2 * node, l, mid)
            _update(2 * node + 1, mid + 1, r)
            self.val[node] = self.val[2 * node] + self.val[2 * node + 1]

        _update(1, 0, self.size - 1)

    def query(self, ql, qr):
        """查询区间1的个数"""
        def _query(node, l, r):
            if ql > r or qr < l:
                return 0
            if ql <= l and r <= qr:
                return self.val[node]
            self._push(node, l, r)
            mid = (l + r) // 2
            left = _query(2 * node, l, mid)
            right = _query(2 * node + 1, mid + 1, r)
            return left + right

        return _query(1, 0, self.size - 1)
```

```python
# 线段树有两个下标，一个是线段树节点的下标，另一个是线段树维护的区间的下标
# 节点的下标：从 1 开始，如果你想改成从 0 开始，需要把左右儿子下标分别改成 node*2+1 和 node*2+2
# 区间的下标：从 0 开始
class SegmentTree:
    def __init__(self, arr, default=0):
        # 线段树维护一个长为 n 的数组（下标从 0 到 n-1）
        # arr 可以是 list 或者 int
        # 如果 arr 是 int，视作数组大小，默认值为 default
        if isinstance(arr, int):
            arr = [default] * arr
        n = len(arr)
        self._n = n
        self._tree = [0] * (2 << (n - 1).bit_length())
        self._build(arr, 1, 0, n - 1)

    # 合并两个 val
    def _merge_val(self, a: int, b: int) -> int:
        return max(a, b)  # **根据题目修改**

    # 合并左右儿子的 val 到当前节点的 val
    def _maintain(self, node: int) -> None:
        self._tree[node] = self._merge_val(self._tree[node * 2], self._tree[node * 2 + 1])

    # 用 a 初始化线段树
    # 时间复杂度 O(n)
    def _build(self, a: List[int], node: int, l: int, r: int) -> None:
        if l == r:  # 叶子
            self._tree[node] = a[l]  # 初始化叶节点的值
            return
        m = (l + r) // 2
        self._build(a, node * 2, l, m)  # 初始化左子树
        self._build(a, node * 2 + 1, m + 1, r)  # 初始化右子树
        self._maintain(node)

    def _update(self, node: int, l: int, r: int, i: int, val: int) -> None:
        if l == r:  # 叶子（到达目标）
            # 如果想直接替换的话，可以写 self._tree[node] = val
            self._tree[node] = self._merge_val(self._tree[node], val)
            return
        m = (l + r) // 2
        if i <= m:  # i 在左子树
            self._update(node * 2, l, m, i, val)
        else:  # i 在右子树
            self._update(node * 2 + 1, m + 1, r, i, val)
        self._maintain(node)

    def _query(self, node: int, l: int, r: int, ql: int, qr: int) -> int:
        if ql <= l and r <= qr:  # 当前子树完全在 [ql, qr] 内
            return self._tree[node]
        m = (l + r) // 2
        if qr <= m:  # [ql, qr] 在左子树
            return self._query(node * 2, l, m, ql, qr)
        if ql > m:  # [ql, qr] 在右子树
            return self._query(node * 2 + 1, m + 1, r, ql, qr)
        l_res = self._query(node * 2, l, m, ql, qr)
        r_res = self._query(node * 2 + 1, m + 1, r, ql, qr)
        return self._merge_val(l_res, r_res)

    # 更新 a[i] 为 _merge_val(a[i], val)
    # 时间复杂度 O(log n)
    def update(self, i: int, val: int) -> None:
        self._update(1, 0, self._n - 1, i, val)

    # 返回用 _merge_val 合并所有 a[i] 的计算结果，其中 i 在闭区间 [ql, qr] 中
    # 时间复杂度 O(log n)
    def query(self, ql: int, qr: int) -> int:
        return self._query(1, 0, self._n - 1, ql, qr)

    # 获取 a[i] 的值
    # 时间复杂度 O(log n)
    def get(self, i: int) -> int:
        return self._query(1, 0, self._n - 1, i, i)
```

```python
# 模板来源 https://leetcode.cn/circle/discuss/mOr1u6/
class Node:
    __slots__ = 'val', 'todo'

class LazySegmentTree:
    # 懒标记初始值
    _TODO_INIT = 0  # **根据题目修改**

    def __init__(self, arr, default=0):
        # 线段树维护一个长为 n 的数组（下标从 0 到 n-1）
        # arr 可以是 list 或者 int
        # 如果 arr 是 int，视作数组大小，默认值为 default
        if isinstance(arr, int):
            arr = [default] * arr
        n = len(arr)
        self._n = n
        self._tree = [Node() for _ in range(2 << (n - 1).bit_length())]
        self._build(arr, 1, 0, n - 1)

    # 合并两个 val
    def _merge_val(self, a: int, b: int) -> int:
        return a + b  # **根据题目修改**

    # 合并两个懒标记
    def _merge_todo(self, a: int, b: int) -> int:
        return a + b  # **根据题目修改**

    # 把懒标记作用到 node 子树（本例为区间加）
    def _apply(self, node: int, l: int, r: int, todo: int) -> None:
        cur = self._tree[node]
        # 计算 tree[node] 区间的整体变化
        cur.val += todo * (r - l + 1)  # **根据题目修改**
        cur.todo = self._merge_todo(todo, cur.todo)

    # 把当前节点的懒标记下传给左右儿子
    def _spread(self, node: int, l: int, r: int) -> None:
        todo = self._tree[node].todo
        if todo == self._TODO_INIT:  # 没有需要下传的信息
            return
        m = (l + r) // 2
        self._apply(node * 2, l, m, todo)
        self._apply(node * 2 + 1, m + 1, r, todo)
        self._tree[node].todo = self._TODO_INIT  # 下传完毕

    # 合并左右儿子的 val 到当前节点的 val
    def _maintain(self, node: int) -> None:
        self._tree[node].val = self._merge_val(self._tree[node * 2].val, self._tree[node * 2 + 1].val)

    # 用 a 初始化线段树
    # 时间复杂度 O(n)
    def _build(self, a: List[int], node: int, l: int, r: int) -> None:
        self._tree[node].todo = self._TODO_INIT
        if l == r:  # 叶子
            self._tree[node].val = a[l]  # 初始化叶节点的值
            return
        m = (l + r) // 2
        self._build(a, node * 2, l, m)  # 初始化左子树
        self._build(a, node * 2 + 1, m + 1, r)  # 初始化右子树
        self._maintain(node)

    def _update(self, node: int, l: int, r: int, ql: int, qr: int, f: int) -> None:
        if ql <= l and r <= qr:  # 当前子树完全在 [ql, qr] 内
            self._apply(node, l, r, f)
            return
        self._spread(node, l, r)
        m = (l + r) // 2
        if ql <= m:  # 更新左子树
            self._update(node * 2, l, m, ql, qr, f)
        if qr > m:  # 更新右子树
            self._update(node * 2 + 1, m + 1, r, ql, qr, f)
        self._maintain(node)

    def _query(self, node: int, l: int, r: int, ql: int, qr: int) -> int:
        if ql <= l and r <= qr:  # 当前子树完全在 [ql, qr] 内
            return self._tree[node].val
        self._spread(node, l, r)
        m = (l + r) // 2
        if qr <= m:  # [ql, qr] 在左子树
            return self._query(node * 2, l, m, ql, qr)
        if ql > m:  # [ql, qr] 在右子树
            return self._query(node * 2 + 1, m + 1, r, ql, qr)
        l_res = self._query(node * 2, l, m, ql, qr)
        r_res = self._query(node * 2 + 1, m + 1, r, ql, qr)
        return self._merge_val(l_res, r_res)

    # 用 f 更新 [ql, qr] 中的每个 a[i]
    # 0 <= ql <= qr <= n-1
    # 时间复杂度 O(log n)
    def update(self, ql: int, qr: int, f: int) -> None:
        self._update(1, 0, self._n - 1, ql, qr, f)

    # 返回用 _merge_val 合并所有 a[i] 的计算结果，其中 i 在闭区间 [ql, qr] 中
    # 0 <= ql <= qr <= n-1
    # 时间复杂度 O(log n)
    def query(self, ql: int, qr: int) -> int:
        return self._query(1, 0, self._n - 1, ql, qr)
```

**思路：字典树板子题，要理解字典树的原理，然后掌握静态数组的字典树写法，实现insert, search, prefixNumber, delete这四个最重要的函数写法，重复度还是比较高的，比较好写**

**代码：**
```python
from typing import List

class Trie:
    def __init__(self, n):
        self.tree = [[0] * 26 for _ in range(n)]
        self.p = [0] * n
        self.e = [0] * n
        self.cnt = 1

    def insert(self, word):
        cur = 1
        self.p[cur] += 1
        for i in range(len(word)):
            path = ord(word[i]) - ord('a')
            if self.tree[cur][path] == 0:
                self.cnt += 1
                self.tree[cur][path] = self.cnt
            cur = self.tree[cur][path]
            self.p[cur] += 1
        self.e[cur] += 1

    def search(self, word):
        cur = 1
        for i in range(len(word)):
            path = ord(word[i]) - ord('a')
            if self.tree[cur][path] == 0:
                return 0
            cur = self.tree[cur][path]
        return self.e[cur]

    def prefixNumber(self, pre):
        cur = 1
        for i in range(len(pre)):
            path = ord(pre[i]) - ord('a')
            if self.tree[cur][path] == 0:
                return 0
            cur = self.tree[cur][path]
        return self.p[cur]

    def delete(self, word):
        if self.search(word) <= 0:
            return
        cur = 1
        for i in range(len(word)):
            path = ord(word[i]) - ord('a')
            self.p[self.tree[cur][path]] -= 1
            if self.p[self.tree[cur][path]] == 0:
                self.tree[cur][path] = 0
                return
            cur = self.tree[cur][path]
        self.e[cur] -= 1

    def clear(self):
        for i in range(1, self.cnt + 1):
            self.p[i] = 0
            self.e[i] = 0
            for j in range(26):
                self.tree[i][j] = 0
        self.cnt = 1

class Solution:
    def trieU(self , operators: List[List[str]]) -> List[str]:
        n = len(operators)
        t = Trie(1000001)
        out = []
        for i in range(n):
            op, word = operators[i]
            if op == '1':
                t.insert(word)
            elif op == '2':
                t.delete(word)
            elif op == '3':
                res = t.search(word)
                out.append("YES" if res else "NO")
            else:
                res = t.prefixNumber(word)
                out.append(str(res))
```

**思路：比较简单的想法就是用前缀树。先把数字存为二进制的形式，然后再逐位进行查找，算最大异或值。不过要注意，python太慢了，非常容易超时。所以说，只能建一次前缀树，后面用clear去清理掉用过的部分，才可以勉强通过。所以，本题更推荐用hash表来写，其原理类似于两数之和**

```python
class Trie:

    def insert(self, x):
```

**思路：本质是回溯，然后用前缀树进行大量剪枝。主要是剪了三点：1.搜索范围(用p)；2.搜索次数(用p)；3.结果收集(用e)**

```python
        t.build(words)

        m, n = len(board), len(board[0])
        ans = []
        def dfs(i, j, k):
            if i < 0 or i >= m or j < 0 or j >= n or board[i][j] == 0:
                return 0 
            
            tmp = board[i][j]
            road = ord(tmp) - ord('a')
            k = t.tree[k][road]
            if t.p[k] == 0:
                return 0

                fix += 1
                ans.append(t.e[k])
                t.e[k] = ""

            fix += dfs(i + 1, j, k)
            fix += dfs(i, j - 1, k)
            fix += dfs(i, j + 1, k)
            t.p[k] -= fix 
            board[i][j] = tmp

        for i in range(m):
```

### 最近公共祖先（LCA）问题

### 1.二叉树的最近公共祖先（LCA）https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/description/

**思路：非常经典的板子题，具体流程如下：dfs二叉树，如果遇到None/p/q中的一个，就可以直接返回了。为什么？None自然不用说，遇到p/q中的一个能返回是因为p和q的两种位置关系，一是包含，而是非包含。先说第一种情况，如果是包含关系，那我们遇到一个之后进行返回，另外一边必然返回空，最终返回非空的那一方，不断传递向上，最终返回p/q；第二种非包含关系，我们分别遇到p和q，到某一点的时候root的left与right汇聚，他就是lca，最终与空相比，取不空的lca，不断向上传递。因此这个代码能够适应这两种情况，是非常标准且优雅的lca扳子**

**代码：**
```python
class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
        if root in (None, p, q):
            return root 
        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)
        if left and right:
            return root
        return left or right
```

### 1.将数组分成最小代价的子数组II https://leetcode.cn/problems/divide-an-array-into-subarrays-with-minimum-cost-ii/description/?envType=daily-question&envId=2026-02-02

**思路：本题翻译一下就是在一个dist + 1的滑动窗口中，找前k - 1个最小值，然后求他们的和再加上初始的nums[0]，就是答案。这个其实可以用对顶堆来做，类似的题目有找数组的中位数。不过对于本题的滑动窗口+对顶堆来说，他不只是简单的像是找中位数，只需要处理堆顶元素，他也有可能处理堆中的元素，所以说如果用对顶堆的话，不能用一般的堆，要用懒删除堆来做**

**不过为了图方便，我们可以用有序集合来做这题，也就是功能十分强大的SortedList，这个结构可以任意删除其中的元素，并且始终保持从小到大的有序状态，还能用bisect方法，非常方便，就可以用他来平替懒删除堆，还更好写**

**代码：**
```python
class Solution:
    def minimumCost(self, nums: List[int], k: int, dist: int) -> int:
        k -= 1 # 提前把nums[0]给减去
        sum_left = sum(nums[:dist + 2])
        L = SortedList(nums[1:dist + 2])
        R = SortedList()

        def L2R():
            x = L.pop()
            nonlocal sum_left 
            sum_left -= x 
            R.add(x)
        
        def R2L():
            x = R.pop(0)
            nonlocal sum_left 
            sum_left += x 
            L.add(x)
        
        while len(L) > k:
            L2R()
        
        ans = sum_left 
        for i in range(dist + 2, len(nums)):
            out = nums[i - dist - 1]
            if out in L:
                sum_left -= out 
                L.remove(out)
            else:
                R.remove(out)

            in_val = nums[i]
            if in_val < L[-1]:
                sum_left += in_val 
                L.add(in_val)
            else:
                R.add(in_val)
            
            if len(L) == k - 1:
                R2L()
            elif len(L) == k + 1:
                L2R()
            
            ans = min(ans, sum_left)
        
        return ans 
```

```python
                st.append(i)
```

```python
        for i in range(n - 1, -1, -1):
            x = nums[i]
            while st and nums[st[-1]] <= x:
```

### 4.去掉重复字符 https://leetcode.cn/problems/remove-duplicate-letters/description/

**思路：单调栈，大的字典序字符压在小的字典序上面，一旦发现栈顶元素比当前字符更小，就需要进行出栈，能否出栈的标准，取决于后面还有没有要出栈的这个字符。所以说，事先要用Counter来统计下词频，然后还需要用vis记录一下哪些字符已经出现过了，因为栈里面不能出现重复字符。最终，自下而上栈里面的结果就是答案**

**代码：**
```python
class Solution:
    def removeDuplicateLetters(self, s: str) -> str:
        cnt = Counter(s)
        vis = set()

        st = []
        for x in s:
            cnt[x] -= 1
            if x in vis:
                continue 
            while st and st[-1] >= x and cnt[st[-1]] != 0:
                vis.remove(st.pop())
            st.append(x)
            vis.add(x)

        return "".join(st)
```

### 1.子数组的最小值之和 https://leetcode.cn/problems/sum-of-subarray-minimums/description/

**思路：首先要想到贡献法，对于每个位置的数来说，假设他是最小值，那么以他为最小值的子数组范围就应该在他左边最近比它小的数的索引 ~ 他右边最近比它小的数索引之间。那提到这几个关键词，自然就可以想到用单调栈去算了。**

**不过这里要主要细节：如果两个左右都写严格小于，也就是>=弹出的话，会出问题。如果遇到重复值，他会有一个重复统计，比如例子：$arr=[1,2,4,2,3,1]$里面的这两个2，那该如何解决？我们可以单独修改其中的一个边界，变成一个是严格小于，一个不是严格小于，这样就解决了，很巧妙的一个思路**

**代码：**
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        mod = 10 ** 9 + 7 

        n = len(arr)
        st = []
        left = [-1] * n 
        for i, x in enumerate(arr):
            while st and arr[st[-1]] >= x:
                st.pop()
            if st:
                left[i] = st[-1]
            st.append(i)
        
        st.clear()
        right = [n] * n 
        for i in range(n - 1, -1, -1):
            x = arr[i]
            while st and arr[st[-1]] > x:
                st.pop()
            if st:
                right[i] = st[-1]
            st.append(i)
        
        ans = 0 
        for i, (x, l, r) in enumerate(zip(arr, left, right)):
            ans = (ans + x * (i - l) * (r - i)) % mod 
        return ans 
```

### 4.统计全1子矩形（单调栈优化dp） https://leetcode.cn/problems/count-submatrices-with-all-ones/description/

**思路：单调栈，动态规划，数组压缩。这里动态规划的转移方程就是：1.当左边没有比枚举的右侧边界更小的高度，那么当前位置子矩形数量就是$dp[i] = x * (i + 1)$；2.当左边有比枚举的右侧边界更小的高度，那么当前位置子矩形数量就是$dp[i] = dp[l] + x * (i - l)$，每次算好了之后，不断累加ans，最后输出就是一个一维数组的全1子矩形数量**

**最终，用数组压缩方法求解以0~m - 1行为底的全1子矩形数量，累加起来就是最终答案。这里面的的递推可以好好记一下，挺巧妙的一个递推关系**

**代码：**
```python
class Solution:
    def numSubmat(self, mat: List[List[int]]) -> int:
        m, n = len(mat), len(mat[0])
        
        def solve(a):
            st = []
            left = [-1] * n 
            for i, x in enumerate(a):
                while st and a[st[-1]] >= x:
                    st.pop()
                if st:
                    left[i] = st[-1]
                st.append(i)

            ans = 0 
            dp = [0] * n 
            for i, x in enumerate(a):
                l = left[i]
                if l == -1:
                    dp[i] = x * (i + 1)
                else:
                    dp[i] = dp[l] + x * (i - l)
                ans += dp[i]
            return ans 
        
        tmp = [0] * n
        ans = 0 
        for i in range(m):
            for j in range(n):
                tmp[j] = 0 if mat[i][j] == 0 else tmp[j] + 1
            ans += solve(tmp)
        
        return ans 
```

### 4.和至少为k的最短子数组 https://leetcode.cn/problems/shortest-subarray-with-sum-at-least-k/description/

**思路：这题和单调队列的一般用法不同，而和单调栈的应用题很相似，他也是用其本身的含义去做的，重点是挖掘出单调性。对于本题来说，首先看到子数组和，要想到用前缀和处理。然后就是构建一个从左往右，从小到大的单调队列**

**当有前缀和进入队列时，先与队列头元素进行相减并更新答案，越往右走，判断越短的数组和能否满足条件。完成之后，要让当前元素入队列，也就是维持从左往右，从小到大的单调队列，一旦发现队列尾元素>=当前元素，就进行尾部出队列操作**

**那为什么要维护这样一个从左往右，从小到大的单调队列？假设我后续有个前缀和是x，如果减去当前更小的不达标，那么减去他前面更大的就更不达标了，所以前面大的应该弹出**

**代码：**
```python
class Solution:
    def shortestSubarray(self, nums: List[int], k: int) -> int:
        inf = 10 ** 18
        n = len(nums)
        pre = [0] * (n + 1)
        for i, x in enumerate(nums):
            pre[i + 1] = pre[i] + x 
        
        q = deque()
        ans = inf
        for r, x in enumerate(pre):
            while q and x - pre[q[0]] >= k:
                ans = min(ans, r - q.popleft())
            while q and pre[q[-1]] >= x:
                q.pop()
            q.append(r)
        return ans if ans < inf else -1
```

### 5.满足不等式的最大值 https://leetcode.cn/problems/max-value-of-equation/description/

**思路：这题思路和4差不多，首先要变换一下式子为：(后)x + y + (前)y - x，然后就根据(前)y - x的单调性。从左往右，构建一个从大到小的单调队列，然后求解方式和4类似**

**代码：**
```python
class Solution:
    def findMaxValueOfEquation(self, points: List[List[int]], k: int) -> int:
        ans = -inf 
        q = deque()
        for x, y in points:
            while q and q[0][0] < x - k:
                q.popleft()
            if q:
                ans = max(ans, x + y + q[0][1])
            while q and q[-1][1] <= y - x:
                q.pop()
            q.append((x, y - x))
        return ans 
```

### 6.你可以安排的最多任务数目 https://leetcode.cn/problems/maximum-number-of-tasks-you-can-assign/description/

**思路：有单调性，可以用二分答案。然后check函数，需要用贪心来做，也就是需要单调队列来做。因为我们已经提前把两个数组都进行了从小到大的排序，所以说这里的单调队列天然就具有单调性**

**那具体有哪些贪心？第一个就是挑最简单的mid个任务和最强的mid个工人来进行完成，因为这是最优可能达成目标的组合；第二个就是用单调队列进行贪心匹配完成，而不是用简单的双指针，因为这题我们额外引入了药丸这个概念**

**具体的贪心策略如下：首先把当前功能如果吃药之后，能够入队列的都给入队列。然后判断此时的队列是否为空，如果为空说明吃了药都无法完成任务，直接返回False；否则就可以完成任务，我们优先在不吃药的情况下，贪心地完成队首的简单任务，这样能够简化逻辑，并且他后面的工人也能完成更难的任务**

**要是无法在不吃药的情况下完成任务，那就判断还有没有药可以吃，没有的话直接返回False，否则吃药，然后完成队尾的困难任务，因为我们不能浪费这个药的作用，要尽量完成自己能够完成的最难任务。**

**最终，如果能完成指定的mid个任务，就返回True。然后不断二分找答案，最终输出结果即可**

**代码：**
```python
class Solution:
    def maxTaskAssign(self, tasks: List[int], workers: List[int], pills: int, strength: int) -> int:
        m, n = len(tasks), len(workers)
        tasks.sort()
        workers.sort()

        def check(mid):
            i, p = 0, pills 
            q = deque()
            for w in workers[n - mid:]: # 取末尾mid个工人，注意不要写-mid，遇到workers[-0:]为所有workers，而不是我们要的末尾0个工人
                while i < mid and tasks[i] <= w + strength:
                    q.append(i)
                    i += 1
                if not q:
                    return False 
                if w >= tasks[q[0]]:
                    q.popleft()
                    continue 
                if p == 0:
                    return False 
                p -= 1
                q.pop()
            return True

        l, r = 0, min(m, n)
        while l <= r:
            mid = (l + r) >> 1
            if check(mid):
                l = mid + 1
            else:
                r = mid - 1
        return r 
```

```python
        m, n = len(heightMap), len(heightMap[0])
        vis = [[False] * n for _ in range(m)]

        ans = 0 
        h = []
        for i in range(m):
            for j in range(n):
                if i == 0 or i == m - 1 or j == 0 or j == n - 1:
                    vis[i][j] = True
                    heappush(h, (heightMap[i][j], i, j))

        while h:
            d, i, j = heappop(h)
            ans += max(0, d - heightMap[i][j])
```
