### 1.数组中的第K个最大元素 https://leetcode.cn/problems/kth-largest-element-in-an-array/description/

**思路1：直接排序然后算，简单直白，暴力至上。时间复杂度是$O(nlogn)$**

**思路2：用堆来算，对于第K个最小元素来说，需要用大根堆；对于第K个最大元素来说，需要用小根堆。为什么？拿第K个最大元素来说，我们每次要弹出最小元素，留下的就是K个最大元素，此时堆顶就是第K个最大元素；同理，第K个最小元素，我们每次要弹出最大元素，剩下的就是K个最小元素，此时堆顶就是第K个最小元素。时间复杂度是$O(nlogk)$**

**代码：**
```python
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        h = []
        for x in nums:
            heappush(h, x)
            if len(h) > k:
                heappop(h)
        return h[0]
```

**思路3：用快排的思想来做，可以把它理解成快速选择算法。这里没用荷兰国旗优化，它空间复杂度是$O(n)$，如果用了荷兰国旗优化，那么原地三指针操作的空间复杂度就是$O(1)$。不过时间复杂度不变，都是$O(n)$，推荐下面的写法，看着很清楚，写起来也比较简单**

**代码：**
```python
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        def quick_select(nums, k):
            pivot = random.choice(nums)
            big, equal, small = [], [], []
            for num in nums:
                if num > pivot:
                    big.append(num)
                elif num < pivot:
                    small.append(num)
                else:
                    equal.append(num)
            if k <= len(big):
                return quick_select(big, k)
            if k > len(big) + len(equal):
                return quick_select(small, k - len(big) - len(equal))
            return pivot
        return quick_select(nums, k)
```

### 2.小红的矩阵 https://www.nowcoder.com/practice/eec2ed5b48b04a1492209ba08593a830?channelPut=tracker2

**思路：考虑上面的三种思路的话，是不可行的，为什么？因为我们没有数据啊，此题给出的是如何生成数据，而不是给出直接的数据。如果说我们模拟去生成数据的话，时间复杂度是$O(m * n)$的，量级达到了惊人的$10^{10}$，所以我们不能去生成数据。应该考虑二分，这又是为什么？这个首先要想到计数排序，对于数字比较少的情况，我们可以统计每个数字，小于它的个数，统计完成之后，那么这个数字就该放在小于它的个数那里，索引从0开始的话。所以说，这题同理，我们先二分答案出来个值，他表示i * j，然后check函数里面就逐行计算小于等于i * j的格子数，最终求和。如果cnt >= k的话，说明我们可以缩小答案；反之，需要扩大答案。可以看到，他也是满足单调性的。最终，我们就可以在不需要生成数据的情况下，以$O(nlog(mn))$的时间复杂度解决此题了。**

**补充：二分答案并不要求值域范围是连续的，他只要求数据具有单调性，即使值域有很多空洞，但他们并不会影响结果的计算，只有那些存在的值，才会造成跳变，影响答案**

**代码：**

```python
import math

def I():

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

    return list(map(float, input().split()))


fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
这题挺好啊，有点计数排序的味道了，相比于用堆来做，更快。为什么？
因为堆来做的话，这个矩阵的元素值你就不好处理，他对于已经确定的值来说好做
'''

def solve():
    n, m, k = LII()

    def check(mid):
        cnt = 0
        for i in range(1, n + 1): # 统计每一行中“乘积 <= mid”的格子数，并且累加得到整个矩阵的“乘积 <= mid”的格子数
            cnt += fmin(m, mid // i)
        return cnt >= k

    l, r = 1, n * m
    while l <= r:
        mid = (l + r) >> 1
        if check(mid):
            r = mid - 1
        else:
            l = mid + 1
    print(l)

# t = II()
for _ in range(t):
    solve()
```

**那本题其实更像是一种不均匀的分组问题，然后查当前音符是属于哪个组的，如果是均匀分组的话，我们其实就可以用类似于位图的处理方法，用divmod计算除数和余数，确定分组，那对于这种不均匀的分组，我们其实可以用前缀和 + 二分查找来做**

### 1.小苯最短路 https://www.nowcoder.com/practice/2130991f48ae48b588398ba5842c7ec0?channelPut=tracker2

**思路：思维。首先我们要知道1到其他点的最短路径是什么？由于题目说了是无向完全图，即每个点都能直达另一个点，所以说直达可能是最短路径。假设直达不是最短路径，说明真正的最短路径之间会经过一些点，那么经过这些点的权重就是(a ^ b) + (b ^ c) + (c ^ d)....根据异或的性质，直达最短路径的权重，即起点值^终点值能够写成(a ^ b) ^ (b ^ c) ^ (c ^ d)...由于同形式的异或和不会大于算术和，因此直达的权重更小，它就是最短路径**

**并且题目要求把1到其他所有点的最短路径异或值给异或起来，其实就是(1 ^ 1) ^ (1 ^ 2) ^ (1 ^ 3) ^ (1 ^ 4).....写成式子就是(n & 1) ^ reduce(xor, range(1, n + 1))，即n的奇偶性^1到n的异或和。但这样写会超时，本题的n很大，所以说可以把上面的式子打表出来找规律，发现只有4种情况，为n, 0, n + 1, 1，那就直接根据%4的余数来输出答案即可**

### 2.小美的数组操作 https://www.nowcoder.com/practice/b65be999590b4dd5881461a8e892d76e?channelPut=tracker2

**思路：思维题。base case比较容易想到：1.操作必须同时+1和-1，所以说整个数组和不变；2.要求最大化众数的个数，由于可以进行任意次操作，所以说令$s = sum(a)$，那么当$s\ \%\ n == 0$时，我们可以把数组a中的所有数都变成平均数，他也是众数。那么plus(+1)和sub(-1)操作操作相同，只需要算一个就行。**

**除去base case以外就不太容易想了，由于$s\ \%\ n\ != 0$，所以说会产生余数，导致数组a中的元素无法都变成平均数，那此时的最大众数有多少个？答案是n - 1个，我们可以选出一个数字来通过和他操作，以存放多余的值。那么这个数字应该怎么选？贪心的想，为了操作最小化，那就应该用最小值或者最大值来存放这些多余的值**

**接下来就需要分类讨论下了：1.去掉最大值，把剩下的n - 1个数字都变成一样的，也就是剩下数的平均数。但这里的平均数我们需要区分一下是上限还是下限，为什么？因为我们额外拿出了最大值，他其实充当了一个中介，多了数可以给他，少了数可以从他那里拿，所以说上限和下限都能算。具体计算过程就是cal函数写的，排除最大值索引，然后算其他n - 1个数变成上限/下限的plus(+1)和sub(-1)操作数，最终返回其中的最大值，就是我们实际的操作数。那这两个操作数的差值是什么，其实就是额外和最大值(中介)操作的次数**

**然后就是去掉最小值的情况，他和去掉最大值的情况同理，这里不再过多描述。最后，不断取fmin然后输出结果即可**

**代码：**
```python
import sys
from collections import Counter

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    a = LII()

    s = sum(a)
    if s % n == 0:
        op = 0
        num = s // n
        for x in a:
            op += x - num if x > num else 0
        print(op)
        return

    cur_mx = max(v for v in Counter(a).values())
    if cur_mx == n - 1:
        print(0)
        return

    def cal(p, idx):
        plus = sub = 0
        for i, x in enumerate(a):
            if i == idx:
                continue
            if x < p:
                plus += p - x
            elif x > p:
                sub += x - p
        return fmax(plus, sub)

    m = M = 0
    for i, x in enumerate(a):
        if x > a[M]:
            M = i
        if x < a[m]:
            m = i

    ans = inf
    # 情况1：排除最大值，让其他n-1个数相等
    sum_without_M = s - a[M]
    base = sum_without_M // (n - 1)

    ans = fmin(ans, cal(base, M)) # 下限
    ans = fmin(ans, cal(base + 1, M)) # 上限

    # 情况2：排除最小值，让其他n-1个数相等
    sum_without_m = s - a[m]
    base = sum_without_m // (n - 1)

    ans = fmin(ans, cal(base, m)) # 下限
    ans = fmin(ans, cal(base + 1, m)) # 上限

    print(ans)

t = 1
# t = II()
for _ in range(t):
    solve()
```

### 3.小红的数组清空 https://www.nowcoder.com/practice/c0784de498ca4779b3dc2a75fddcf12b?channelPut=tracker2

**思路：思维题。由于每个数可能出现多次，所以说先用hash表统计一下，然后排序贪心地算可以免费删除的最大次数$sub$，具体流程如下：先定义一个$pre\_val$变量表示上一个元素是多少，初始化为-1，因为题目说了输入数据$a_i >= 1$，这样初始化能保证他不会和第一个数连续**

**再定义一个$pre\_cnt$变量表示上一个元素出现的次数，然后遍历hash表判断是否连续。如果连续的话，免费的删除次数就是$fmin(pre\_cnt,\ c)$，累加到$sub$中。然后更新$pre\_val$和$pre\_cnt$，这里需要注意一下，由于我们可以形成的连续链长度>= 2，所以说$pre\_cnt$应该更新为当前元素的出现次数c，他不仅可以被上一个元素作用，也可以作用于接下来的元素。如果碰到不连续的了，那么直接令$pre\_cnt = 0$表示不会产生免费删除次数**

**最后清空整个数组的最小操作代价就是n - sub，直接输出结果即可**

**代码：**
```python
import sys
from collections import Counter

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    a = LII()

    cnt = Counter(a)
    sub = 0
    pre_val = -1
    pre_cnt = 0

    for val, c in sorted(cnt.items()):
        if val != pre_val + 1:
            pre_cnt = 0

        sub += fmin(pre_cnt, c)
        pre_val = val
        pre_cnt = c

    print(n - sub)

t = 1
# t = II()
for _ in range(t):
    solve()
```

### 5.加油站 https://leetcode.cn/problems/gas-station/description/

**思路：两种解法，一种是优化过的滑动窗口，一种是思维**

**代码：优化过的滑动窗口**
```python
class Solution:
    def canCompleteCircuit(self, gas: List[int], cost: List[int]) -> int:
        n = len(gas)
        tmp = [0] * n 
        for i in range(n):
            tmp[i] = gas[i] - cost[i]

        l = r = 0
        while r < 2 * n:
            s = 0
            while s + tmp[r % n] >= 0:
                if r - l + 1 == n:
                    return l 
                s += tmp[r % n]
                r += 1
            l = r + 1
            r = l

        return -1
```

### 6.游游的最小公倍数 https://www.nowcoder.com/practice/385c7aa397e54bb58f36286ab0d65156?channelPut=tracker2

**思路：思维题。首先直接算最大的lcm比较困难，我们用另一个转换式子：$lcm(a,\ b) = a\ /\ gcd(a,\ b) * b$，这个写法是防溢出的。因此我们要让$lcm$最大，就需要让$a * b$最大，$gcd(a,\ b)$最小**

**对于$a * b$最大这个要求来说，我们根据题目说的$a + b = n$条件，以及高中学过的基本不等式，即$a + b = n$的前提下，有$a + b >= 2 * \sqrt{a * b}$，转化式子得到$a * b <= n^2/4$，当且仅当$a = b = n / 2$时取得$a * b$的最大值**

**对于$gcd(a,\ b)$最小来说，他只可能是1，因此我们可以写个gcd函数来方便后面的验证，当然python也可以直接调用math.gcd，这里我习惯自己手写**

**所以说有了上面两个分析之后，我们就可以先把$n$一分为二，然后再判断$n\ //\ 2$与$n\ - n\ //\ 2$的$gcd$是否为1。如果两者的$gcd$为1，不用操作；如果不为1，那就不断a--和b++（这样能保证两者的和始终为n），直到两者的gcd为1时停止。最终，输出结果即可**

**代码：**

### 7.二进制手表 https://leetcode.cn/problems/binary-watch/description/?envType=daily-question&envId=2026-02-17

**思路：正向思考通过turnedOn去找数非常困难，考虑逆向思考。直接暴力枚举可能的h和m，之后再统计枚举的h和m是否满足其二进制1个数和为turnedOn即可**

**然后可以注意下f""格式化输出里面的:02d这个写法，他和:.2f比较相似，也就是把小数点改成了0表示位数不够用0填充，2的含义相同都表示整体位数固定为2，然后前面的d表示为整数，后者的f表示为浮点数**

**代码：**
```python
class Solution:
    def readBinaryWatch(self, turnedOn: int) -> List[str]:
        ans = []
        for h in range(12):
            for m in range(60):
                if h.bit_count() + m.bit_count() == turnedOn:
                    ans.append(f"{h}:{m:02d}")
        return ans 
```

### 8.计数二进制子串 https://leetcode.cn/problems/count-binary-substrings/description/?envType=daily-question&envId=2026-02-19

**思路：分组循环，然后发现答案就是相邻两个连续的不同字符，其长度中最小值的累加和**

**代码：**
```python
class Solution:
    def countBinarySubstrings(self, s: str) -> int:
        n = len(s)
        pre = cur = ans = 0 
        for i in range(n):
            cur += 1
            if i == n - 1 or s[i] != s[i + 1]:
                ans += min(pre, cur)
```

### 2.种树 https://www.nowcoder.com/practice/9602d50d189b4fa69755ba7b2b6df4a8?channelPut=tracker2

**思路：贪心。为了保证根节点具有最大的生命值，我们可以用一个贪心策略：大剪刀能用$m = (cnt + 1)\ //\ 2$次，那么就在前m层使用大剪刀，后更深的层使用小剪刀即可。并且题目说了，每次修剪为左右孩子到根，这个顺序天然和后序遍历一致。因此，我们就可以递归算一遍后序遍历，以及对应节点的深度；之后，再用贪心策略求出最终的答案并输出即可**

**注意：递归这里python很有可能超时，推荐用模拟栈来写。模拟栈的后序遍历有两种写法：1.先序遍历改一下孩子的访问顺序，最后整体翻转；2.额外用state状态变量进行标记**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    ls = [0] * (n + 1)
    rs = [0] * (n + 1)

    cnt = 0
    for u in range(1, n + 1):
        l, r = MII()
        ls[u], rs[u] = l, r
        if l != 0:
            cnt += 1

    a = [0] + LII()
    m = (cnt + 1) // 2

    depth = [0] * (n + 1) # 存放节点深度
    order = [] # 存放后序遍历
    stack = [(1, 0, 1)]  # (节点, 状态, 深度)

    while stack:
        u, state, d = stack.pop()
        if state == 0:
            depth[u] = d
            stack.append((u, 1, d))
            if ls[u] != 0:
                stack.append((rs[u], 0, d + 1)) # 由于栈是后进先出，所以先入右
                stack.append((ls[u], 0, d + 1)) # 后入左，后续处理时左先弹出
        else:
            order.append(u)

    # 题目的操作天然就是左右根，符合后序遍历
    for u in order:
        if ls[u] == 0:
            continue

        l, r = ls[u], rs[u]
        if depth[u] <= m: # 贪心，上m层用大剪刀
            a[u] = fmax(a[l], a[r])
        else: # 下面更深的层用小剪刀
            a[u] = fmin(a[l], a[r])

    print(a[1])

t = 1
# t = II()
for _ in range(t):
    solve()
```

```python
                return -1
            else:
                return 1
```

```python
        min_h, max_h = [], []
        for x, y in a:
            heappush(min_h, (y, x))
        
        cnt = 0
        while cnt < k:
            while min_h and w >= min_h[0][0]:
                pop_min_h = heappop(min_h)
                heappush(max_h, (-pop_min_h[1], pop_min_h[0]))
            
            if not max_h:
                break

            pop_max_h = heappop(max_h)
            w += -pop_max_h[0]
            cnt += 1
        return w
```

### 11.最短无序连续子数组 https://leetcode.cn/problems/shortest-unsorted-continuous-subarray/description/

**思路：很巧妙的$O(n)$做法，首先从前往后遍历一遍，找最右边的不满足位置，那么他更右边的位置必然满足排序；同理再从后往前遍历一遍，找最左边的不满足位置，那么他更左边的位置必然满足排序。最终，将两个位置中间的长度就是需要改变的长度**

```python
class Solution:
    def findUnsortedSubarray(self, nums: List[int]) -> int:
        n = len(nums)
        mx = -inf
        r = 0
        for i, x in enumerate(nums):
            if x >= mx:
                mx = x
            else:
                r = i 
        
        mn = inf 
        l = n
        for i in range(n - 1, -1, -1):
            if nums[i] <= mn:
                mn = nums[i]
            else:
                l = i 
        
        return max(0, r - l + 1)
```

```python
                    cnt += 1

            cnt += 1
```

### 16.最大平均通过率 https://leetcode.cn/problems/maximum-average-pass-ratio/description/

**思路：大根堆 + 贪心。每次进来一个人的时候，算$diff = (x + 1)\ /\ (y + 1) - x\ /\ y$，最终把那个人给diff增量最大的，这就可以用到大根堆结构。然后，为了追求更高精度，可以把上面的diff化简，写成单独的分式：$diff = (y - x)\ /\ (y * (y + 1))$**

**代码：**
```python
class Solution:
    def maxAverageRatio(self, classes: List[List[int]], extraStudents: int) -> float:
        n = len(classes)
        h = []
        for x, y in classes:
            diff = (x + 1) / (y + 1) - x / y # (y - x) / (y * (y + 1)) 精度丢失更少
            h.append((-diff, x, y))
        heapify(h)

        for _ in range(extraStudents):
            _, x, y = heappop(h)
            x += 1; y += 1 # 先加再算diff
            diff = (x + 1) / (y + 1) - x / y # (y - x) / (y * (y + 1)) 精度丢失更少
            heappush(h, (-diff, x, y))
        
        return sum(x / y for _, x, y in h) / len(h)
```

```python
    def rotateGrid(self, grid: List[List[int]], k: int) -> List[List[int]]:
        m0, n0 = len(grid), len(grid[0])

        # 从外到内枚举圈
        for i in range(min(m0, n0) // 2):

            for dx, dy in DIRS:
                for _ in range(n - 1):
                    a.append(grid[x][y])
                    x += dx
                    y += dy
                m, n = n, m  # 每发生一次旋转，行列进行交换

                    x += dx
                    y += dy
```

```python
                    even.append(x)
```

**思路：贪心 + 区间合并。首先看数据量，用dp肯定过不了，那就考虑贪心；接下来把x和h进行排序，也就是用(x, h)进行自定义排序；然后进行区间合并操作，具体来说就是用栈进行模拟操作，模拟完后得到合并的多段区间。最终，按照区间长度进行降序排列，选择前m个长度最大的区间，加起来就是答案了**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n, m = MII()
    h = LII()
    x = LII()

    a = sorted(zip(x, h))

    st = []
    for pos, h in a:
        cur_r = pos + h

        if not st or st[-1][0] < pos:
            st.append([cur_r, 1])
        else:
            st[-1][0] = fmax(st[-1][0], cur_r)
            st[-1][1] += 1

    res = [size for r, size in st]
    res.sort(reverse=True)

    ans = sum(res[:fmin(m, len(res))])
    print(ans)

# t = 1
t = II()
for _ in range(t):
    solve()

```

### 2.线段重合 https://www.nowcoder.com/practice/1ae8d0b6bb4e4bcdbf64ec491f63fc37

**思路：有两种思路，一个是排序 + 小根堆，还有一个就是差分。推荐前者，代码比较好写，不过时间复杂度差点，空间复杂度挺好**

### 3.课程表III https://leetcode.cn/problems/course-schedule-iii/description/

**思路：首先根据结束时间进行排序，接着用堆进行贪心选课，最终的堆大小就是结果**

**代码：**
```python
class Solution:
    def scheduleCourse(self, courses: List[List[int]]) -> int:
        courses.sort(key=lambda x:x[1])
        h = []
        time = 0 
        for x, y in courses:
            if time + x <= y:
                time += x 
                heappush(h, -x)
            else:
                if h and -h[0] > x:
                    time += heappop(h) + x
                    heappush(h, -x)
        return len(h)
```

### 4.最多可以参加的会议数目 https://leetcode.cn/problems/maximum-number-of-events-that-can-be-attended/

**思路：贪心，先按照开始时间进行排序，然后再枚举天数 + 小根堆计算最多可以参加的会议数目**

**代码：**
```python
class Solution:
    def maxEvents(self, events: List[List[int]]) -> int:
        n = len(events)
        events.sort()

        mn = events[0][0]
        mx = max(y for x, y in events)

        h = []
        i = ans = 0 
        for day in range(mn, mx + 1): # 枚举天数进行判断
            while i < n and events[i][0] == day:
                heappush(h, events[i][1])
                i += 1
            while h and h[0] < day:
                heappop(h)
            if h:
                heappop(h)
                ans += 1
        return ans 
```

### 5.灌溉花园的最少水龙头数目 https://leetcode.cn/problems/minimum-number-of-taps-to-open-to-water-a-garden/description/

**思路：主要是要想到预处理出$right[i] = j$数组表示左端点为$i$，向右最远的灌溉距离。然后也是定义三个变量进行贪心，这个写法和跳跃游戏II很像，最终遍历一遍找答案**

**代码：**
```python
class Solution:
    def minTaps(self, n: int, ranges: List[int]) -> int:
        right = [0] * (n + 1)
        for i in range(n + 1):
            l = max(0, i - ranges[i])
            right[l] = max(right[l], i + ranges[i])

        ans = cur = nxt = 0 
        for i in range(n):
            nxt = max(nxt, right[i])
            if cur == i:
                if nxt > i:
                    cur = nxt 
                    ans += 1
                else:
                    return -1
        return ans 
```

## 中位数

### 1.小红的中位数 https://ac.nowcoder.com/acm/contest/130107/D

**思路：首先逆向思考，删除最少数相当于保留最多数，使得中位数发生改变。并且还有个重要的思考，我们是否必须从中间删除？答案是否定的，我们其实从两端删一样可以改变中位数，并且编码难度要远远低于中间删的操作。因此，我们获取到中位数之后，就可以分别计算两端改变中位数时，保留的最长长度，答案就是n - 其中的更大值**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    a = LII()
    a.sort()

    if a[0] == a[-1]:
        print(-1)
        return

    v = a[(n - 1) // 2] # 注意题目说的中位数是第，也就是索引从1开始，而不是从0开始
    ans = 0

    for i in range(n):
        if a[i] == v:
            ans = max(ans, 2 * i) # 保留最长左段
            break

    for i in range(n - 1, -1, -1):
        if a[i] == v:
            ans = max(ans, 2 * (n - 2 - i) + 1) # 保留最长右段
            break

    print(n - ans) # 最少删除次数

t = 1
# t = II()
for _ in range(t):
    solve()
```

### 2.获取单值网络的最小操作数 https://leetcode.cn/problems/minimum-operations-to-make-a-uni-value-grid/description/?envType=daily-question&envId=2026-04-28

**思路：同余贪心 + 中位数贪心。对于同余贪心来说，证明比较简单：$(grid[i][j] + kx)\ \%\ x$就是我们每个数操作后的结果，k可正可负。进一步推导式子可以得到$(grid[i][j] + kx)\ \%\ x = grid[i][j]\ \%\ x$，也就是说在任意+-x的操作下，要使得所有元素相同，那么所有数% x的结果必须相同**

**对于中位数贪心来说：以一个场景举例，很多人在操场上站成了一排，怎么快速将大家聚齐在同一个位置。应该往中间靠，也就是中位数的位置，这个证明比较麻烦，反正不是从平均值位置靠，中位数是操作数最少的**

**代码：**
```python
class Solution:
    def minOperations(self, grid: List[List[int]], x: int) -> int:
        a = []
        mod = grid[0][0] % x

        for row in grid:
            for v in row:
                if v % x != mod:
                    return -1
                a.append(v)
        
        a.sort()
        median = a[len(a) // 2]
        return sum(abs(v - median) // x for v in a)
```

### 1.元素方碑 https://www.nowcoder.com/practice/5c6e7ed4726e41f4ac99a4dedf1e5bb2?channelPut=tracker2

**思路：思维题。有点类似01串翻转问题，不过01串翻转问题是相邻进行操作，本题是中间隔了一个元素，然后有个+1 -1或者-1 +1。这类问题的通用思考是多次向后滑动操作，规律就显现出来了。本题就是分成两组，这两组元素相同，那就分别根据相等元素的差值进行遍历，最终判断差值是否为0即可**

## 环形数组最短距离

### 1.到目标字符串的最短距离 https://leetcode.cn/problems/shortest-distance-to-target-string-in-a-circular-array/description/?envType=daily-question&envId=2026-04-15

**思路：模拟。主要是知道环形数组，左右移动的距离怎么算，对于本题来说，可以分类讨论一下大小关系，这样写起来简单点**

**代码：**
```python
class Solution:
    def closestTarget(self, words: List[str], target: str, startIndex: int) -> int:
        inf = 10 ** 18 
        n = len(words)
        tmp = []
        for i, x in enumerate(words):
            if x == target:
                tmp.append(i)

        if not tmp:
            return -1 
        
        ans = inf 
        for x in tmp:
            if startIndex > x:
                l = startIndex - x 
                r = n - l 
            else:
                r = x - startIndex 
                l = n - r 
            ans = min(ans, l, r)
        return ans 
```

### 2.距离最小相等元素查询 https://leetcode.cn/problems/closest-equal-element-queries/description/?envType=daily-question&envId=2026-04-16

**思路：和昨天那题最短距离差不多，不过是增强版，需要用到hash表和二分查找。怎么想到二分查找？比较简单，就是每个固定的查询必然在hash表中出现，并且我们只需要找到前一个和后一个数，算这两个就行，其他位置必然会比这两个位置更差**

**代码：**
```python
class Solution:
    def solveQueries(self, nums: List[int], queries: List[int]) -> List[int]:
        m = len(nums)
        d = defaultdict(list)
        for i, x in enumerate(nums):
            d[x].append(i)

        n = len(queries)
        ans = [-1] * n 
        for i, x in enumerate(queries):
            num = nums[x]
            row = d[num]

            rn = len(row)

            if len(row) < 2:
                continue

            p = bisect_left(row, x)
            l, r = (p - 1 + rn) % rn, (p + 1) % rn
            pl, pr = row[l], row[r]
            if pl > x:
                res1 = min(pl - x, m - (pl - x))
            else:
                res1 = min(x - pl, m - (x - pl))
            if pr > x:
                res2 = min(pr - x, m - (pr - x))
            else:
                res2 = min(x - pr, m - (x - pr))

            ans[i] = min(res1, res2)
            
        return ans 
```
