## 双指针

### 1.寻找重复数 https://leetcode.cn/problems/find-the-duplicate-number/

**思路：快慢指针，和链表里面找入环节点是一样的。具体来说就是：初始化慢指针s走一步，快指针f走两步。然后不断这样走，知道两者第一次相遇。相遇之后，快指针f回到开头，然后两者都只走1步，最终相遇的位置就是入环点**

**代码：**
```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        s, f = nums[0], nums[nums[0]]
        while s != f:
            s = nums[s]
            f = nums[nums[f]]
            
        f = 0 
        while s != f:
            s = nums[s]
            f = nums[f]
        return s
```

### 2.接雨水 https://leetcode.cn/problems/trapping-rain-water/description/

**思路：通过单调性，优化掉原先的2*n辅助空间。具体说来是这样的：lmax和rmax分别代表左边/右边的最大值，当某一边更小的时候我们要进行结算，为什么？原因有两个：1.我们接水都是取lmax和rmax中更小的一方；2.当这方更小时，右边更大的区域有可能会更大，也可能更小，但此时都会大于更小的这一方。因此，结算了不会影响最终答案，最后不断累加接水量就是答案了**

**代码：**
```python
class Solution:
    def trap(self, height: List[int]) -> int:
        l, r, lmax, rmax = 1, len(height) - 2, height[0], height[-1]
        ans = 0
        while l <= r:
            if lmax < rmax:
                ans += max(0, lmax - height[l])
                lmax = max(lmax, height[l])
                l += 1
            else:
                ans += max(0, rmax - height[r])
                rmax = max(rmax, height[r])
                r -= 1
        return ans 
```

### 3.供暖器 https://leetcode.cn/problems/heaters/description/

**思路：排序双指针，找匹配房屋i的最优供暖器j，并且算出其距离值，然后不断滑动取max，就是答案**

**代码：**
```python
class Solution:
    def findRadius(self, houses: List[int], heaters: List[int]) -> int:
        m, n = len(houses), len(heaters)
        houses.sort()
        heaters.sort()

        def best(i, j):
            return j == n - 1 or abs(heaters[j] - houses[i]) < abs(heaters[j + 1] - houses[i])

        ans = 0
        i = j = 0 
        while i < m:
            while not best(i, j):
                j += 1
            ans = max(ans, abs(heaters[j] - houses[i]))
            i += 1
        return ans 
```

### 4.缺失的第一个正数 https://leetcode.cn/problems/first-missing-positive/description/

**思路：把双指针赋予文字上的意义：对于左指针i来说，他表示0~i的部分都满足nums[i] == i + 1；对于右指针j来说，他第一个表示r及右边的区域是垃圾，第二个表示从0到r - 1索引对应的值是可能对的最优解，其具体值是从1到r。这样遍历一遍，最终输出i + 1即可**

**代码：**
```python
class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        n = len(nums)

        def swap(i, j):
            nums[i], nums[j] = nums[j], nums[i]

        i, j = 0, n 
        while i < j:
            if nums[i] == i + 1:
                i += 1
            elif nums[i] <= i or nums[i] > j or nums[nums[i] - 1] == nums[i]:
                j -= 1
                swap(i, j)
            else:
                swap(i, nums[i] - 1)
        return i + 1
```

### 5.下标对中的最大距离 https://leetcode.cn/problems/maximum-distance-between-a-pair-of-values/description/?envType=daily-question&envId=2026-04-19

**思路：双指针。由题意可知，两个数组天然有序，并且满足单调性，因此可以用双指针滑动一遍，最终得到答案**

**代码：**
```python
class Solution:
    def maxDistance(self, nums1: List[int], nums2: List[int]) -> int:
        m, n = len(nums1), len(nums2)
        i = j = 0 
        ans = 0 
        while i < m and j < n:
            while j < n and nums2[j] >= nums1[i]:
                j += 1
            ans = max(ans, j - i - 1)
            i += 1
        return ans 
```

## 分组循环

### 1.游游的最长稳定子数组 https://www.nowcoder.com/practice/ea7098b7960348f6915e252f0c4debcc?channelPut=tracker2

**思路：预处理 + 分组循环。首先把原数组相邻元素进行相减并取abs，处理出tmp数组。接着根据题意，我们就找这其中连续的最长为1的子数组即可，最终答案为其长度 + 1。因为我们算的是处理过的相邻之差子数组长度，所以要补个1才是原子数组长度**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    a = LII()
    tmp = [0] * (n - 1)
    for i in range(1, n):
        tmp[i - 1] = abs(a[i] - a[i - 1])

    mx = i = 0
    while i < n - 1:
        if tmp[i] > 1:
            i += 1
            continue
        st = i
        i += 1
        while i < n - 1 and tmp[i] <= 1:
            i += 1
        mx = fmax(mx, i - st)
    print(mx + 1)

t = 1
# t = II()
for _ in range(t):
    solve()
```

### 2.穷哈哈 https://www.nowcoder.com/practice/5b3184b233f34fb39a7f259ae82eb42c?channelPut=tracker2

**思路：预处理 + 分组循环。对于交替这个要求，从字符的角度来说不好处理，因此我们可以把目标字符转化为数字0和1，然后再用分组循环的异或，因为异或的^1性质非常适合两个字符的交替，最终不断取长度的max就是答案**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    s = I()

    a = [-1] * n
    for i, c in enumerate(s):
        if c == 'a':
            a[i] = 0
        elif c == 'h':
            a[i] = 1

    ans = 0
    i = 0
    while i < n:
        if a[i] == -1:
            i += 1
            continue
        st = i
        i += 1
        while i < n and a[i] == a[i - 1] ^ 1:
            i += 1
        ans = max(ans, i - st)
    print(ans)

t = 1
# t = II()
for _ in range(t):
    solve()
```
