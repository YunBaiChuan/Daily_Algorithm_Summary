## 第一题：二维网格中探测环 https://leetcode.cn/problems/detect-cycles-in-2d-grid/?envType=daily-question&envId=2026-04-26

**思路：dfs + vis标记。找环问题不能用bfs，这个无法判断是否成环了，本题算一个很经典的板子题，可以记一下**

**代码：**
```python
class Solution:
    def containsCycle(self, grid: List[List[str]]) -> bool:
        m, n = len(grid), len(grid[0])
        
        vis = [[False] * n for _ in range(m)]
        def dfs(i: int, j: int, fai: int, faj: int) -> bool:
            if vis[i][j]:
                return True
            vis[i][j] = True
            flag = False
            for x, y in (i + 1, j), (i - 1, j), (i, j + 1), (i, j - 1):
                if x == fai and y == faj:
                    continue
                if 0 <= x < m and 0 <= y < n and grid[x][y] == grid[i][j]:
                    flag |= dfs(x, y, i, j)
            return flag

        for i in range(m):
            for j in range(n):
                if not vis[i][j] and dfs(i, j, -1, -1):
                    return True
        return False
```
