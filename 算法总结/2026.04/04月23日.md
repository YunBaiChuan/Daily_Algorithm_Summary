## 第一题：等值距离和 https://leetcode.cn/problems/sum-of-distances/description/?envType=daily-question&envId=2026-04-23

**思路：hash表 + 前缀和，有点前后缀分解的意思。思路还是容易想到，但是left和right的计算需要观察出来，这点比较难想**

**代码：**
```python
class Solution:
    def distance(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [0] * n 

        d = defaultdict(list)
        for i, x in enumerate(nums):
            d[x].append(i)
        

        for k, row in d.items():
            m = len(row)
            if m == 1:
                continue 

            pre = [0] * (m + 1)
            for i in range(m):
                pre[i + 1] = pre[i] + row[i]
            
            for j, x in enumerate(row):
                left = j * x - pre[j]
                right = pre[-1] - pre[j] - (m - j) * x  
                ans[x] = left + right 
            
        return ans 
```
