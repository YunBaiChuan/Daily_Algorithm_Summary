## 第一题：检查网格中是否存在有效路径 https://leetcode.cn/problems/check-if-there-is-a-valid-path-in-a-grid/?envType=daily-question&envId=2026-04-27

**思路：网格图dfs，方向向量进行简化。有了方向向量，然后就是经典的网格图dfs板子了，不过有个判断需要想好，即$(-dx, -dy)$这个方向向量，必须存在于相邻的街道中，否则走不过去**

**代码：**
```python
DIRS = ( # 方向向量进行简化
    (),
    ((0, -1), (0, 1)),  # 站在街道 1，可以往左或者往右
    ((-1, 0), (1, 0)),  # 站在街道 2，可以往上或者往下
    ((0, -1), (1, 0)),  # 站在街道 3，可以往左或者往下
    ((0, 1), (1, 0)),   # 站在街道 4，可以往右或者往下
    ((0, -1), (-1, 0)), # 站在街道 5，可以往左或者往上
    ((0, 1), (-1, 0)),  # 站在街道 6，可以往右或者往上
)

class Solution:
    def hasValidPath(self, grid: list[list[int]]) -> bool:
        m, n = len(grid), len(grid[0])
        vis = [[False] * n for _ in range(m)]

        def dfs(x, y):
            if x == m - 1 and y == n - 1:
                return True
            vis[x][y] = True
            for dx, dy in DIRS[grid[x][y]]:
                i, j = x + dx, y + dy
                if 0 <= i < m and 0 <= j < n and not vis[i][j] and \
                   (-dx, -dy) in DIRS[grid[i][j]] and dfs(i, j): # 这里的(-dx, -dy)不太好想，主要是如果当前街道能走，那么相邻街道必须包含反方向向量，否则是走不通的
                    return True
            return False

        return dfs(0, 0)
```
