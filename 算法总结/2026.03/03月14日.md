## 第一题：清楚姐姐买竹鼠 https://www.nowcoder.com/practice/816dfe69f83042108cf8c74531a94a35?channelPut=tracker2

**思路：思维。我们贪心地想：如果$3 * a <= b$，那么肯定是买单只更好；如果$3 * a > b$，我们优先买$3$只，最终还剩下mod只需要购买。此时就需要进行讨论，由于题目说的是至少，因此我们买的数量是可以超的。那就判断一下，剩下数量买单只的花费$mod * a$与直接买3只的花费$b$，哪个小就选择哪个。最终，输出结果即可**

## 第二题：长度为 n 的开心字符串中字典序第 k 小的字符串 https://leetcode.cn/problems/the-k-th-lexicographical-string-of-all-happy-strings-of-length-n/description/?envType=daily-question&envId=2026-03-14

**思路：暴力回溯，固定长度类型，不用显式写恢复现场。并且从小到大填，最终字典序必然是从小到大排好序的**

**代码：**
```python
class Solution:
    def getHappyString(self, n: int, k: int) -> str:
        a = ['a', 'b', 'c']
        res = []
        path = [''] * n
        def dfs(i):
            if i == n:
                if all(x != y for x, y in pairwise(path)):
                    res.append(''.join(path))
                return 
            for j in range(3):
                path[i] = a[j]
                dfs(i + 1)
        dfs(0)
        return res[k - 1] if len(res) >= k else ""
```
