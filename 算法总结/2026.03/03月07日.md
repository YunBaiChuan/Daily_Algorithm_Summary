## 第一题：使二进制字符串字符交替的最少反转次数 https://leetcode.cn/problems/minimum-number-of-flips-to-make-the-binary-string-alternating/description/?envType=daily-question&envId=2026-03-07

**思路：主要是怎么想到转化成定长滑动窗口的，具体如下：首先操作1没有限制次数，那就可以操作任意次数，其操作出来的结果其实是s + s中长为n的滑动窗口字串；然后就是交替的处理方式，对于"010101..."来说其索引奇偶性和对应的数值有相等关系；反之对于"101010..."来说其索引奇偶性和对应的数值有不相等关系。因此，我们就做一个长度为n的定长滑动窗口，之后统计其中不相等/相等的位置数量，他就是要操作的次数，之后再用长度n去减，就得到相等/不相等的位置数，这两个取min就是本次的最小操作次数，然后再和ans不断取min就可以得到最终答案了**

**代码：**
```python
class Solution:
    def minFlips(self, s: str) -> int:
        inf = 10 ** 18 
        ans = inf
        n = len(s)
        cnt = 0 
        for i in range(2 * n - 1):
            if int(s[i % n]) != i & 1:
                cnt += 1
            l = i - n + 1
            if l < 0:
                continue 
            ans = min(ans, cnt, n - cnt)
            if int(s[l]) != l & 1:
                cnt -= 1
        return ans 
```

## 第二题：Rinne Loves Edges https://www.nowcoder.com/practice/080d6bd6edb043bd855f633f70edde1d?channelPut=tracker2

**思路：想法还是比较简单的，一个dfs从s遍历到叶子节点，每次删除路径上的最小权重边，最终累加到答案中**

**代码：**

def I():

def solve():
    n, m, s = LII()
    g = [[] for _ in range(n + 1)]
