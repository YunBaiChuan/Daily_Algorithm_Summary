## 第二题：找出第n个二进制字符串中的第k位 https://leetcode.cn/problems/find-kth-bit-in-nth-binary-string/description/?envType=daily-question&envId=2026-03-03

**思路：观察题目，打表。注意到每个位置的数都是固定的，并且状态是由上一个回文位置的翻转数转移而来，并且数据量比较小，那就可以通过递推来打表，直接查询答案即可**

**代码：**
```python
ans = [0] * (2 ** 20 + 1)
cnt = 1
for i in range(2, 2 ** 20 + 1):
    if i == (2 ** cnt):
        ans[i] = 1
        cnt += 1
    else:
        ans[i] = ans[(2 ** cnt) - i] ^ 1

class Solution:
    def findKthBit(self, n: int, k: int) -> str:
        return str(ans[k])
```
