## 第二题：十-二进制数的最小数目 https://leetcode.cn/problems/partitioning-into-minimum-number-of-deci-binary-numbers/description/?envType=daily-question&envId=2026-03-01

**思路：思维题。由于只能由0和1组成，那么数位上的最大值，就是用十-二进制数组成他的最小数目**

**代码:**
```python
class Solution:
    def minPartitions(self, n: str) -> int:
        return max(map(int, n))
```
