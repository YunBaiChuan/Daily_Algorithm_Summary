## 第一题：判断矩阵经轮转后是否一致 https://leetcode.cn/problems/determine-whether-matrix-can-be-obtained-by-rotation/description/?envType=daily-question&envId=2026-03-22

**思路：模拟。主要就是知道二维矩阵的旋转变换公式，如果是顺时针，那么有变换$(i,\ j) -> (j,\ n - 1 - i)$，逆时针就是$(i,\ j) -> (n - 1 - j,\ i)$**

**代码：**
```python
class Solution:
    def findRotation(self, mat: List[List[int]], target: List[List[int]]) -> bool:
        m, n = len(mat), len(mat[0])

        def func(a):
            return [[a[j][n - 1 - i] for j in range(n)] for i in range(m)]
        
        def check(a, b):
            for i in range(m):
                for j in range(n):
                    if a[i][j] != b[i][j]:
                        return False 
            return True 

        for _ in range(4):
            res = check(mat, target)
            if res:
                return True 
            mat = func(mat)
        return False
```

## 第二题：游游的最长稳定子数组 https://www.nowcoder.com/practice/ea7098b7960348f6915e252f0c4debcc?channelPut=tracker2

**思路：预处理 + 分组循环。首先把原数组相邻元素进行相减并取abs，处理出tmp数组。接着根据题意，我们就找这其中连续的最长为1的子数组即可，最终答案为其长度 + 1。因为我们算的是处理过的相邻之差子数组长度，所以要补个1才是原子数组长度**
