## 第一题：小红的连续段 https://www.nowcoder.com/practice/252310f0f8cd4101be25517652fc5bdf?channelPut=tracker2

**思路：组合数学。需要使用到原始的隔板法结论，用于分段，然后还需要进行为'a'分几段的讨论。因此，每次分段的方法数ans就是'a'分p段，其隔板法方案数为comb(x - 1, p - 1)，即从x - 1个间隔中放置p - 1个板子，就可以获得p段；'b'分段隔板法方案为comb(y - 1, q - 1)，即从x - 1个间隔中放置q - 1个板子，就可以获得q段，两者的乘积就是当前'a'分p段的答案；同理'a'分q段可以算出另一个答案。最终，不断累加方案数并取mod，然后整体输出即可**

**代码：**

mod = 10 ** 9 + 7

MAXN = 2000
fac = [1] * (MAXN + 1)
for i in range(1, MAXN + 1):
    fac[i] = fac[i - 1] * i % mod

def inv(x):
    return pow(x, mod - 2, mod)

def comb(a, b):
    if b < 0 or b > a:
        return 0
    return fac[a] * inv(fac[b]) * inv(fac[a - b]) % mod

def solve():
    x, y = MII()
    out = []
    for i in range(1, x + y + 1):
        p, q = (i + 1) >> 1, i >> 1  

        ans = 0
        if 1 <= p <= x and q <= y:
            ans = (ans + comb(x - 1, p - 1) * comb(y - 1, q - 1)) % mod

        if 1 <= p <= y and q <= x:
            ans = (ans + comb(x - 1, q - 1) * comb(y - 1, p - 1)) % mod
        out.append(str(ans))
        
    print('\n'.join(out))

t = 1
# t = II()
for _ in range(t):
    solve()
