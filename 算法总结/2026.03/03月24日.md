## 第一题：太阳之华 https://www.nowcoder.com/practice/be67461f82ee46f69c5ffec7b549bb7e?channelPut=tracker2

**思路：思维 + bfs模拟。首先红方要赢的话，必须在一开始就能把所有蓝色染为红色，否则蓝色总能进行周旋，最终达成平局；同理蓝色要赢的话，必须一开始所有块都是蓝色。因此，我们就遍历到红色快的时候，进行bfs模拟染色，看能否把剩下的蓝色都变成红色，如果有一个结果可以，那么红色方就能赢**

**代码：**
```python
import sys
from collections import deque

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

def solve():
    n, m = MII()
    a = [input().strip() for _ in range(n)]

    cntr, cntb = 0, 0
    for i in range(n):
        for j in range(m):
            if a[i][j] == '.':
                cntb += 1
            else:
                cntr += 1

    if cntr == 0:
        print("Blue")
        return
    if cntb == 0:
        print("Red")
        return

    blue_vis = [[0] * m for _ in range(n)]
    red_id = 0

    for i in range(n):
        for j in range(m):
            if a[i][j] == '.' or blue_vis[i][j] < 0:
                continue

            red_id += 1
            q = deque()
            q.append((i, j))
            blue_vis[i][j] = -1
            cnt = 0

            while q:
                i, j = q.popleft()
                for x, y in (i + 1, j), (i - 1, j), (i, j + 1), (i, j - 1):
                    if 0 <= x < n and 0 <= y < m:
                        if a[x][y] == '#':
                            if blue_vis[x][y] >= 0:
                                blue_vis[x][y] = -1
                                q.append((x, y))
                        else:
                            if blue_vis[x][y] != red_id:
                                blue_vis[x][y] = red_id
                                cnt += 1

            if cnt == cntb:
                print("Red")
                return

    print("Draw")

t = II()
for _ in range(t):
    solve()
```

## 第二题：构造乘积矩阵 https://leetcode.cn/problems/construct-product-matrix/description/?envType=daily-question&envId=2026-03-24

**思路：首先这题不能用整体乘积，然后除当前数，为什么？因为值域很大，会爆空间和时间。那先取mod再算呢？也不行，为什么？因为取mod之后有0的情况出现，此时除法就失效了。因此，我们要用前后缀分解来完成这道题。它相当于是个二维矩阵的前后缀分解，我们只需要把他拉成一维的数组即可，此时每个位置的答案就是pre * suf了**

**代码：**
```python
class Solution:
    def constructProductMatrix(self, grid: List[List[int]]) -> List[List[int]]:
        MOD = 12345
        n, m = len(grid), len(grid[0])
        p = [[0] * m for _ in range(n)]

        suf = 1
        for i in range(n - 1, -1, -1):
            for j in range(m - 1, -1, -1):
                p[i][j] = suf
                suf = suf * grid[i][j] % MOD

        pre = 1
        for i, row in enumerate(grid):
            for j, x in enumerate(row):
                p[i][j] = p[i][j] * pre % MOD
                pre = pre * x % MOD

        return p
```
