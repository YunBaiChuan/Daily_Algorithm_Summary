## 第一题：小美打怪 https://www.nowcoder.com/practice/5fb4f77c5f0e41118d16a23112b64ea6?channelPut=tracker2

**思路：排序 + LIS。他的本质是算一个二维最长严格递增子序列，然后再用小红本身的H和A，去判断是否能够战胜最大的二维严格递增子序列结尾。如果可以的话，就更新ans = max(ans, dp[i])，最终输出ans即可**

**注意：这里排序是为了贪心的得到更长的二维最长严格递增子序列，而且题目只限制了h, a的值关系，位置可以改变**

**代码：**
```python
import sys

import math
inf = 10 ** 18

def I():
    return input()

    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n, H, A = LII()
    h = LII()
    a = LII()
    nums = list(zip(h, a))
    nums.sort()

    dp = [0] * n
    for i in range(n):
        for j in range(i):
            if nums[i][0] > nums[j][0] and nums[i][1] > nums[j][1]:
                dp[i] = fmax(dp[i], dp[j])
        dp[i] += 1

    ans = 0
    for i in range(n):
        if H > nums[i][0] and A > nums[i][1]:
            ans = fmax(ans, dp[i])
    print(ans)

t = 1
# t = II()
for _ in range(t):
    solve()
```

## 第二题：最大方阵和 https://leetcode.cn/problems/maximum-matrix-sum/description/?envType=daily-question&envId=2026-01-05

**思路：思维题，模拟一下可以发现：当负数有偶数个的时候，总能全部变成正数；当负数有奇数个的时候，我们只需要令绝对值最小的那个数为负数，最终减去他即可，注意要减两遍，因为s是整个矩阵和，多算了一次**

**代码：**
```python
class Solution:
    def maxMatrixSum(self, matrix: List[List[int]]) -> int:
        s = cnt = 0 
        mn = 10 ** 8
        for i, row in enumerate(matrix):
            for j, x in enumerate(row):
                if x < 0:
                    cnt += 1
                s += abs(x)
                mn = min(mn, abs(x))
        return s - 2 * mn if cnt & 1 else s
```

## 第三题：清楚姐姐带带我 https://ac.nowcoder.com/acm/problem/240498

**思路：主要是怎么解决取mod还不丢失x, y的大小关系，我们可以引入一个flag位，在值超过10 ** 9时取mod**

**代码：**
```python
mod = 19980829 
n = int(input())
x = 0 
f = 0 
for _ in range(n): 
    a, b = map(int, input().split())
    if f: 
        if b == 1: 
            x = (x + a) % mod 
        else: 
            x = x * b % mod 
    else: 
        if b * x > a + x: 
            x *= b 
        else: 
            x += a 
        if x > 10 ** 9: # 阈值 > mod
            f = 1 
            x %= mod 
print(x % mod) 
```

## 第四题：石子游戏 https://ac.nowcoder.com/acm/problem/19156

**思路：猜测最优策略是大于1的奇数堆分成1和剩余偶数，就是ops这里没和max取0，导致没过完**

**代码：**
```python

n = int(input())
a = list(map(int, input().split()))
evens = olds = 0 
for x in a:
    if x & 1:
        olds += 1 if x != 1 else 0
    else:
        evens += 1
ops = max(olds * 2 + evens - 1, 0)
print("Alice" if ops & 1 else "Bob")
```

## 第五题：小杜捕鱼 https://ac.nowcoder.com/acm/problem/247478

**思路：找四个边角取max就行，没限制渔网中心不能在有鱼的位置，有点迟钝了思维，下机！**

**代码：**
```python

n, m = map(int, input().split())
s = [list(input()) for i in range(n)]

ans = 0
for i in range(n):
    for j in range(m):
        if s[i][j] == '#':
            ans = max(ans, i + j)
            ans = max(ans, n - 1 - i + j)
            ans = max(ans, i + m - 1 - j)
            ans = max(ans, n + m - 2 - i - j)
print(ans)
```
