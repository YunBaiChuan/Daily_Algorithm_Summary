## 第一题：两个字符串的最小删除ASCII码和 https://leetcode.cn/problems/minimum-ascii-delete-sum-for-two-strings/description/?envType=daily-question&envId=2026-01-10

**思路：可以逆向进行考虑，用总的ascii值来减去最大相同ascii码和，然后这个最大相同ascii码和其实就是用LIS来做**

**代码：**
```python
class Solution:
    def minimumDeleteSum(self, s1: str, s2: str) -> int:
        n, m = len(s1), len(s2)
        total = sum(map(ord, s1)) + sum(map(ord, s2))

        f = [[0] * (m + 1) for _ in range(n + 1)]
        for i, x in enumerate(s1):
            for j, y in enumerate(s2):
                if x == y:
                    f[i + 1][j + 1] = f[i][j] + ord(x)
                else:
                    f[i + 1][j + 1] = max(f[i][j + 1], f[i + 1][j])
        return total - f[n][m] * 2
```

## 第二题：小红的平滑值插值 https://www.nowcoder.com/practice/97f6b71b1df745cd9926eb861e2d89df?channelPut=tracker2

**思路：整体思路比较容易想到：当两数间的差值$> k$时，用公差为$k$的等差数列去插入，结果就是$(abs(a[i] - a[i - 1]) - 1) \ //\  k$，然后不断累加即可。但是有个特判不好注意到，整体思路可以处理差值中有k的情况，此时会正确输出0；但当所有差值都< k时，此时应该输出1，因为我们要额外构造一个数，使得差值为k。所以说再额外引入一个flag数组，作为特判的标记**
