## 第一题：小红的gcd https://www.nowcoder.com/practice/5500a375098947c482b2c4787057cb13?channelPut=tracker2

**思路：思维题。题目说了我们可以进行任意次操作，那么直接算整个数组a的gcd即可，为什么？因为越多数去算gcd，其结果不会变大，而会不变或者变小，这恰好满足元素和越小越好的要求；并且我们能够进行任意次操作，那就可以把所有的元素都变成整个数组a的gcd。所以我们先算出整个数组a的gcd再 * n就得到最终结果，然后输出即可**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
如果python的版本足够高，也可以用下面两种调库和解包的写法:
print(n * reduce(math.gcd, a))
print(n * math.gcd(*a))
'''

# 以下是能够贴合其他语言的gcd和lcm写法，考虑了溢出问题
def gcd(a, b):
    return a if b == 0 else gcd(b, a % b)

def lcm(a, b):
    return a // gcd(a, b) * b

def solve():
    n = II()
    a = LII()
    
    s = 0
    for x in a:
        s = gcd(s, x)
    print(s * n)

t = 1
# t = II()
for _ in range(t):
    solve()
```

## 第二题：转换字符串的最小成本II https://leetcode.cn/problems/minimum-cost-to-convert-string-ii/description/?envType=daily-question&envId=2026-01-30

**思路：本题和他上一题差不多，就是从单个字符改成了字符串，所以说还是可以用两种方法来解决，一种是dijkstra，一种是floyd。不过最后算代价的变化就大了，因为不是定长的，我们不知道什么时候要进行替换，也就是说贪心是不行的，那就只能用dp来做。首先要预处理出可能的长度，然后写dp转移，最终查找代价，累加答案即可**

**代码：**
```python
class Solution:
    def minimumCost(self, source: str, target: str, original: List[str], changed: List[str], cost: List[int]) -> int:
        inf = 10 ** 18
        
        g = defaultdict(list)
        for u, v, w in zip(original, changed, cost):
            g[u].append((v, w))
        
        def dijkstra(u):
            dis = defaultdict(lambda: inf)
            dis[u] = 0 
            h = [(0, u)]
            while h:
                d, x = heappop(h)
                if d > dis[x]:
                    continue 
                for y, w in g[x]:
                    new_dis = d + w 
                    if new_dis < dis[y]:
                        dis[y] = new_dis 
                        heappush(h, (new_dis, y))
            return dis 

        dist = {}
        for x in original:
            dist[x] = dijkstra(x)
        
        # 预处理所有可能的长度，方便后续dp进行枚举
        words = set(original)
        lengths = set(len(w) for w in words)
        
        n = len(source)
        dp = [inf] * (n + 1) # dp[i]表示把source[0:i]变成target[0:i]的最小代价
        dp[0] = 0 # 初始化空串为0
        
        for i in range(n):
            if dp[i] == inf:
                continue
            
            # 如果相同则直接从上一个进行状态转移
            if source[i] == target[i]:
                dp[i + 1] = min(dp[i + 1], dp[i])
            
            # 如果不同就枚举可能的长度l，然后进行状态转移
            for l in lengths:
                if i + l > n:
                    continue
                u = source[i:i + l] 
                v = target[i:i + l] 

                if u in dist and v in dist[u]: 
                    dp[i + l] = min(dp[i + l], dp[i] + dist[u][v])
        
        return dp[n] if dp[n] < inf else -1
```
