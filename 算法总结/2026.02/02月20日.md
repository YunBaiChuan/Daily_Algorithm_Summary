## 第一题：种树 https://www.nowcoder.com/practice/9602d50d189b4fa69755ba7b2b6df4a8?channelPut=tracker2

**思路：贪心。为了保证根节点具有最大的生命值，我们可以用一个贪心策略：大剪刀能用$m = (cnt + 1)\ //\ 2$次，那么就在前m层使用大剪刀，后更深的层使用小剪刀即可。并且题目说了，每次修剪为左右孩子到根，这个顺序天然和后序遍历一致。因此，我们就可以递归算一遍后序遍历，以及对应节点的深度；之后，再用贪心策略求出最终的答案并输出即可**

**注意：递归这里python很有可能超时，推荐用模拟栈来写。模拟栈的后序遍历有两种写法：1.先序遍历改一下孩子的访问顺序，最后整体翻转；2.额外用state状态变量进行标记**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    ls = [0] * (n + 1)
    rs = [0] * (n + 1)

    cnt = 0
    for u in range(1, n + 1):
        l, r = MII()
        ls[u], rs[u] = l, r
        if l != 0:
            cnt += 1

    a = [0] + LII()
    m = (cnt + 1) // 2

    depth = [0] * (n + 1) # 存放节点深度
    order = [] # 存放后序遍历
    stack = [(1, 0, 1)]  # (节点, 状态, 深度)

    while stack:
        u, state, d = stack.pop()
        if state == 0:
            depth[u] = d
            stack.append((u, 1, d))
            if ls[u] != 0:
                stack.append((rs[u], 0, d + 1)) # 由于栈是后进先出，所以先入右
                stack.append((ls[u], 0, d + 1)) # 后入左，后续处理时左先弹出
        else:
            order.append(u)

    # 题目的操作天然就是左右根，符合后序遍历
    for u in order:
        if ls[u] == 0:
            continue

        l, r = ls[u], rs[u]
        if depth[u] <= m: # 贪心，上m层用大剪刀
            a[u] = fmax(a[l], a[r])
        else: # 下面更深的层用小剪刀
            a[u] = fmin(a[l], a[r])

    print(a[1])

t = 1
# t = II()
for _ in range(t):
    solve()
```

## 第二题：特殊的二进制字符串 https://leetcode.cn/problems/special-binary-string/description/?envType=daily-question&envId=2026-02-20

**思路：题目说了给定的特殊二进制字符串具有如下性质：1.二进制字符串中所有的'0'和'1'数量相同；2.前缀'1'的数量总是大于前缀'0'的数量。综合这两个性质，他其实和合法括号是一样的，合法括号也是这两个性质。因此，我们可以把'0'当成右括号，把'1'当成左括号，这样就实现了两者的转化**

**那具体该如何得到最大字典序字符呢？我们有两种情况可以进行讨论：1.拼接，我们把所有合法的小括号序列进行从大到小排序，之后在拼接起来，就可以得到最大的字典序；2.嵌套，对于一般的合法小括号序列来说，嵌套的字典序会比不嵌套的更大，所以说我们在遇到合法小括号的时候，就可以进行嵌套处理，即外面再套一层。由于我们可以进行任意次操作，所以说这个是可以实现的；并且套了一层之后，其内部剩余的字符串，我们也是采用相同的处理方法，那说明就改用递归去解决**

**最终，我们收集完所有合法并且最大化的小括号序列后，再用前面提到的排序拼接，就可以得到最终答案了**

**代码：**
```python
class Solution:
    def makeLargestSpecial(self, s: str) -> str:
        if len(s) <= 2:
            return s 
        
        ans = []
        diff = 0 
        start = 0 
        for i, c in enumerate(s):
            if c == '1':
                diff += 1
            else:
                diff -= 1
                if diff == 0:
                    ans.append('1' + self.makeLargestSpecial(s[start + 1:i]) + '0')
                    start = i + 1
                    
        ans.sort(reverse=True)
        return ''.join(ans)
```
