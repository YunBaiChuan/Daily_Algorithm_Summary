## 第二题：2054.两个最好的不重叠活动 https://leetcode.cn/problems/two-best-non-overlapping-events/description/?envType=daily-question&envId=2025-12-23

**思路：排序好想，但是栈+二分不好想，不过最通用的做法应该是dp。这题严格来说，是不相交区间问题。感觉比较典型，可以总结一下**

**代码：**
```python
class Solution:
    def maxTwoEvents(self, events: List[List[int]]) -> int:
        events.sort(key=lambda e:e[1])

        st = [(0, 0)]
        ans = 0 
        for start_time, end_time, value in events:
            i = bisect_left(st, (start_time,)) - 1
            ans = max(ans, st[i][1] + value) # 相当于是覆盖结果
            if value > st[-1][1]:
                st.append((end_time, value))
        return ans 
```
