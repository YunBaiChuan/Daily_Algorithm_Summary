**思路：对于这种完全没思路的题目怎么办？用小数据来找找规律，也就是俗称的打表。打表之后，需要仔细观察才能发现如下规律：每7个智数是一个周期，每个周期之间有30的递增值，然后就能做出来了。不过这里还有一个数学的知识点需要记忆：各位数字之和是 3 的倍数，等价于自身是 3 的倍数，因为一个数是 3 的倍数 ↔ 各位和是 3 的倍数。这个知识点需要记住，它和11的倍数那个各位之和是0差不多**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
感觉是预处理奇数，示例2给了点提示，最大智数不超过5 * 10^9
但这个大小用O(n)构造必然超时，有可能是数位dp，用位运算来做

没想到是枚举观察，找规律题。他的周期是7，然后每个周期增值是30
'''

a = [3, 5, 9, 15, 21, 25, 27]

def solve():
    k = II()
    k -= 1
    t, r = divmod(k, 7)
    print(30 * t + a[r])

# tt = 1
tt = II()
for _ in range(tt):
    solve()
```
