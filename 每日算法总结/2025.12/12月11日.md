## 第一题：3531.统计被覆盖的建筑 https://leetcode.cn/problems/count-covered-buildings/?envType=daily-question&envId=2025-12-11

**思路：写得有点冗余，但整体思路是这样的：对于每一行、每一列我们都要计算其最大值和最小值，最终遍历一遍，并且判断当前点是否夹在中间即可。需要注意一点，那就是最终判断是否夹在中间的时候，对于x来说应该是用col，也就是列固定判断；对于y来说应该用row，也就是行固定判断**

**代码：**
```python
class Solution:
    def countCoveredBuildings(self, n: int, buildings: List[List[int]]) -> int:
        d_row = defaultdict(list)
        d_col = defaultdict(list)
        for x, y in buildings:
            d_row[x].append([x, y])
            d_col[y].append([x, y])
        
        dd_row = [[-1, -1] for _ in range(n + 1)]
        dd_col = [[-1, -1] for _ in range(n + 1)]

        for k, row in d_row.items():
            mx = max([y for x, y in row])
            mn = min([y for x, y in row])
            dd_row[k][0] = mn
            dd_row[k][1] = mx
        
        for k, row in d_col.items():
            mx = max([x for x, y in row])
            mn = min([x for x, y in row])
            dd_col[k][0] = mn 
            dd_col[k][1] = mx
        
        ans = 0 
        for x, y in buildings:
            if dd_row[x][0] < y < dd_row[x][1] and dd_col[y][0] < x < dd_col[y][1]:
                ans += 1
        return ans
```

## 第二题：小红的矩阵 https://www.nowcoder.com/practice/eec2ed5b48b04a1492209ba08593a830?channelPut=tracker2

**思路：新的topK解决方法，利用计数排序的思想，以及二分答案为什么得到的结果必然存在于矩阵中，这个需要用反证法来证明。后面具体总结一下**

def solve():
    n, m, k = LII()

    def check(mid):
        cnt = 0
        for i in range(1, n + 1): # 统计每一行中“乘积 <= mid”的格子数，并且累加得到整个矩阵的“乘积 <= mid”的格子数
            cnt += fmin(m, mid // i)
        return cnt >= k

    l, r = 1, n * m
    while l <= r:
