## 第一题：2211.统计道路上的碰撞次数 https://leetcode.cn/problems/count-collisions-on-a-road/?envType=daily-question&envId=2025-12-04

**思路：脑经急转弯，没想到怎么去串联已有的信息。首先我们知道，前缀'L'以及后缀'R'，必然无法碰撞，直接去掉。那么剩下的车辆必然会发生碰撞，而且只有能动的车可以发生碰撞，这个性质是观察到了的。那么最终的ans就是剩余能碰撞的车辆数 - 静止的车辆数**

**代码：**
```python
class Solution:
    def countCollisions(self, s: str) -> int:
        s = s.lstrip('L')
        s = s.rstrip('R')
        return len(s) - s.count('S')
```

## 第二题：质数统计 https://www.nowcoder.com/practice/d832b0c1a0bd4394a3229f06c6f0b50b?channelPut=tracker2

**思路：经典的快速确定质数，以及区间查询问题。那就用埃式筛 + 前缀和处理即可，需要注意的就是埃式筛前两个数不是质数，也就是0和1，需要额外初始化为0**

**代码：**
```python
import sys
from itertools import accumulate

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
埃式筛 + 前缀和优化
'''

mxn = 10 ** 6 + 5
res = [0] * 2 + [1] * mxn
for i in range(2, 10 ** 3 + 1):
    for j in range(i * 2, mxn, i):
        res[j] = 0
ans = list(accumulate(res))

def solve():
    n = II()
    for _ in range(n):
        l, r = MII()
        print(ans[r] - ans[l - 1])

t = 1
for _ in range(t):
    solve()
```
