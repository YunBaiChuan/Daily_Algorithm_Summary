## 第一题：3577.统计计算机解锁顺序排列数 https://leetcode.cn/problems/count-the-number-of-computer-unlocking-permutations/description/?envType=daily-question&envId=2025-12-10

**思路：脑经急转弯，我们只需要用根计算机去解锁其他计算机就行，首先特判能否全部解锁，之后剩下的n - 1个位置做一个n - 1的全排列即可，并且取个mod**

**代码：**
```python
class Solution:
    def countPermutations(self, complexity: List[int]) -> int:
        mod = 10 ** 9 + 7 
        n = len(complexity)
        if min(complexity[1:]) <= complexity[0]:
            return 0 
        return math.factorial(n - 1) % mod
```

## 第二题：大撒币 https://www.nowcoder.com/practice/ff0f60d80ee94d94aba959b2b4c1941a?channelPut=tracker2

**思路：先手有必胜策略，利用几何对称性，只要Alice能放第一枚硬币，然后后面仿照Bob放对称，就可以必胜**

**代码：**
```python
import sys

def LII():
    return list(map(int, input().split()))


def LFI():
    return list(map(float, input().split()))


fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
最优策略应该是放中间，挤占左右两边的位置

放奇数个盘子就a赢，放偶数个盘子就b赢
'''

def solve():
    a, b, r = LII()
    def cnt(x): # 模拟
        ans = 0
        while x >= 2 * r:
            ans += 1
            x //= 2
            x -= r
        return fmax(0, 1 + 2 * (ans - 1)) # 数数要对，因为我们ans其实算的是某一方向上的中间，以及一边的一半
    res = fmin(cnt(a), cnt(b))
    print("Alice" if res & 1 else "Bob")

    '''
    有必胜策略，利用几何对称性，只要alice能放第一枚硬币，然后后面仿照bob放对称，就可以必胜
    print("Alice" if a >= 2 * r and b >= 2 * r else "Bob")
    '''

t = 1
# t = II()
for _ in range(t):
    solve()
```
