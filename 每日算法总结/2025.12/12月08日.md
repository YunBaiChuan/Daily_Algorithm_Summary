## 第一题：1925.统计平方和三元组的数目 https://leetcode.cn/problems/count-square-sum-triples/description/?envType=daily-question&envId=2025-12-08

**思路：直接枚举三个变量必然会爆炸，由于三个变量之间存在关系，因此枚举a或者b以及c，这两个变量，然后算出第三个变量，并且进行验证即可，还有就是注意下数据范围**

**代码：**
```python
class Solution:
    def countTriples(self, n: int) -> int:
        cnt = 0 
        for c in range(1, n + 1):
            for a in range(1, n + 1):
                if a > c:
                    continue
                b = isqrt(c * c - a * a)
                if b == 0 or b > n:
                    continue
                if a * a + b * b == c * c:
                    cnt += 1
        return cnt 
```
