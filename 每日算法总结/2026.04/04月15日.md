## 第一题：到目标字符串的最短距离 https://leetcode.cn/problems/shortest-distance-to-target-string-in-a-circular-array/description/?envType=daily-question&envId=2026-04-15

**思路：模拟。主要是知道环形数组，左右移动的距离怎么算，对于本题来说，可以分类讨论一下大小关系，这样写起来简单点**

**代码：**
```python
class Solution:
    def closestTarget(self, words: List[str], target: str, startIndex: int) -> int:
        inf = 10 ** 18 
        n = len(words)
        tmp = []
        for i, x in enumerate(words):
            if x == target:
                tmp.append(i)

        if not tmp:
            return -1 
        
        ans = inf 
        for x in tmp:
            if startIndex > x:
                l = startIndex - x 
                r = n - l 
            else:
                r = x - startIndex 
                l = n - r 
            ans = min(ans, l, r)
        return ans 
```
