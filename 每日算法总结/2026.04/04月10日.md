## 第一题：三个相等元素之间的最小距离 https://leetcode.cn/problems/minimum-distance-between-three-equal-elements-ii/description/

**思路：hash表 + 贪心。最短的距离和必然是相邻的三个数，不妨令i < j < k，就可以去掉绝对值，并且不会对最终答案产生影响，然后不断取2 * (k - i)的最小值，就是答案了**

**代码：**
```python
class Solution:
    def minimumDistance(self, nums: List[int]) -> int:
        inf = 10 ** 18
        n = len(nums)
        if n < 3:
            return -1
        
        d = defaultdict(list)
        for i, x in enumerate(nums):
            d[x].append(i)
        
        ans = inf
        for row in d.values():
            if len(row) < 3:
                continue 
            for j in range(2, len(row)):
                ans = min(ans, 2 * (row[j] - row[j - 2]))
        return ans if ans < inf else -1
```

## 第二题：小红的图上加边 https://www.nowcoder.com/practice/28c35b0e3f3b4a20aee8f0497ca6745e?channelPut=tracker2

**思路：并查集 + 贪心。也是同样的思路，遇到连通块问题，首先要想到并查集。合并完连通块之后，收集每个连通块中的最大值，然后排个序，再从小到大合并。此时的代价就是从小到大合并中，更大的一方，并且合并的次数为sets - 1，也就是连通块的个数-1，可以发现就是除去第一个位置的其他代价和，直接输出即可**

**代码：**
```python

class UnionFind:
    def __init__(self, n, a):
        self.fa = list(range(n))
        self.size = [1] * n
        self.mx = [0] + a[:]
        self.sets = n

    def find(self, x):
        if x != self.fa[x]:
            self.fa[x] = self.find(self.fa[x])
        return self.fa[x]

    def is_same(self, x, y):
        return self.find(x) == self.find(y)

    def union(self, x, y):
        fx, fy = self.find(x), self.find(y)
        if fx == fy:
            return False
        self.fa[fx] = fy
        self.size[fy] += self.size[fx]
        self.mx[fy] = fmax(self.mx[fy], self.mx[fx])
        self.sets -= 1
        return True

def solve():
    n, m = MII()
    a = LII()

    uf = UnionFind(n + 1, a)
    for _ in range(m):
        u, v = MII()
        if not uf.is_same(u, v):
            uf.union(u, v)

    st = set()
    for i in range(1, n + 1):
        st.add(uf.find(i))

    tmp = []
    for x in st:
        tmp.append(uf.mx[x])
    tmp.sort()

    print(sum(tmp[1:]))
```
