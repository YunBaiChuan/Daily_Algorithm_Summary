**思路：推式子，pow函数优化。根据题意，我们可以得出数学公式：a对应的位置数 * 该位置的权重 * b的长度，然后不断累加取mod就是最终答案。不过直接用pow函数的话，他是$O(log)$的复杂度，外面还要嵌套一个for，那么整体时间复杂度就来到了$O(mlogm + nlogn)$，会超时**

**那么除了用if判断a，b长度大小关系来优化，还有什么优化可以解决pow的问题？那就是预处理出10的幂次方数组，这个是$O(n)$的，最终优化时间复杂度为$O(m + n)$，能够通过**

**代码：**

mod = 998244353

def solve():
    a, b = LI()

    m, n = len(a), len(b)
    if m > n:
        m, n = n, m
        a, b = b, a

    pow10_n = [1] * n
    for i in range(1, n):
        pow10_n[i] = pow10_n[i - 1] * 10 % mod

    pow10_m = [1] * m
    for i in range(1, m):
        pow10_m[i] = pow10_m[i - 1] * 10 % mod

    nb = 0
    for i in range(n):
        nb = (nb + int(b[i]) * pow10_n[n - i - 1]) % mod

    ans = 0
    for i in range(m):
        ans = (ans + n * int(a[i]) * pow10_m[m - i - 1] + nb) % mod

    print(ans)

# t = 1
t = II()
for _ in range(t):
    solve()
