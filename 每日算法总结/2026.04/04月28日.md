## 第一题：获取单值网络的最小操作数 https://leetcode.cn/problems/minimum-operations-to-make-a-uni-value-grid/description/?envType=daily-question&envId=2026-04-28

**思路：同余贪心 + 中位数贪心。对于同余贪心来说，证明比较简单：$(grid[i][j] + kx)\ \%\ x$就是我们每个数操作后的结果，k可正可负。进一步推导式子可以得到$(grid[i][j] + kx)\ \%\ x = grid[i][j]\ \%\ x$，也就是说在任意+-x的操作下，要使得所有元素相同，那么所有数% x的结果必须相同**

**对于中位数贪心来说：以一个场景举例，很多人在操场上站成了一排，怎么快速将大家聚齐在同一个位置。应该往中间靠，也就是中位数的位置，这个证明比较麻烦，反正不是从平均值位置靠，中位数是操作数最少的**

**代码：**
```python
class Solution:
    def minOperations(self, grid: List[List[int]], x: int) -> int:
        a = []
        mod = grid[0][0] % x

        for row in grid:
            for v in row:
                if v % x != mod:
                    return -1
                a.append(v)
        
        a.sort()
        median = a[len(a) // 2]
        return sum(abs(v - median) // x for v in a)
```
