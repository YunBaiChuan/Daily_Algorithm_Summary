## 第一题：距离字典两次编辑以内的单词 https://leetcode.cn/problems/words-within-two-edits-of-dictionary/description/?envType=daily-question&envId=2026-04-22

**思路：数据范围很小，直接暴力即可**

**代码：**
```python
class Solution:
    def twoEditWords(self, queries: List[str], dictionary: List[str]) -> List[str]:
        def cal(s, t):
            n = len(s)
            ans = 0 
            for i in range(n):
                ans += (s[i] != t[i])
            return ans 
        
        ans = []
        for i, x in enumerate(queries):
            for j, y in enumerate(dictionary):
                if cal(x, y) <= 2:
                    ans.append(x)
                    break 
        return ans
```
