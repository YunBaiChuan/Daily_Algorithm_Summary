## 第一题：分元宵 https://www.nowcoder.com/practice/0be3d626297a4c0aa99bf99d7cb888d5?channelPut=tracker2

**思路：组合数学。这题其实不难，不过变量比较多。注意到碗和元宵其实都是独立的，因此每个碗可以放的元宵种类数为a * b，然后有c * d个碗，最终答案就是(a * b) ^ (c * d)了**

## 第二题：镜像对之间最小绝对距离 https://leetcode.cn/problems/minimum-absolute-distance-between-mirror-pairs/description/?envType=daily-question&envId=2026-04-17

**思路：枚举右，维护左。比较简单，但是更新答案那里要想清楚，容易绕进去**

**代码：**
```python
class Solution:
    def minMirrorPairDistance(self, nums: List[int]) -> int:
        inf = 10 ** 18
        def rev(x):
            return int(str(x)[::-1])
        
        d = defaultdict(int)
        ans = inf
        for i, x in enumerate(nums):
            if x in d:
                ans = min(ans, i - d[x])
            d[rev(x)] = i 
            
        return ans if ans < inf else -1
```
