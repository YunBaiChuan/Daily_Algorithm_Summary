## 第二题：重新排列后的最大子矩阵 https://leetcode.cn/problems/largest-submatrix-with-rearrangements/description/?envType=daily-question&envId=2026-03-17

**思路：大思路想到了，就是压缩数组。接着贪心排序，算每个高度下的最大子矩阵面积，然后不断取max，就可以得到最终答案了**

**代码：**
```python
class Solution:
    def largestSubmatrix(self, matrix: List[List[int]]) -> int:
        n = len(matrix[0])
        heights = [0] * n 
        ans = 0 

        for row in matrix:
            for j, x in enumerate(row):
                if x == 0:
                    heights[j] = 0 
                else:
                    heights[j] += 1
        
            hs = sorted(heights)
            for i, h in enumerate(hs):
                ans = max(ans, (n - i) * h)
        return ans 
```
