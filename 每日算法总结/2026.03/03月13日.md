## 第一题：移山所需要的最小秒数 https://leetcode.cn/problems/minimum-number-of-seconds-to-make-mountain-height-zero/description/?envType=daily-question&envId=2026-03-13

**思路：二分答案 + 数学推导。答案的秒数具有单调性，越大越能满足条件，反之越不能满足条件，然后就是等差数列求和公式以及二次函数推导出秒数可以移除的山高度，之后就可以进行二分答案了**

**代码：**
```python
class Solution:
    def minNumberOfSeconds(self, mountainHeight: int, workerTimes: List[int]) -> int:
        def check(mid):
            cnt = 0 
            for x in workerTimes:
                tmp = 2 * mid // x 
                cnt += (-1 + isqrt(1 + 4 * tmp)) // 2
                if cnt >= mountainHeight:
                    return True 
            return False

        l, r = 1, 10 ** 18
        while l <= r:
            mid = (l + r) >> 1
            if check(mid):
                r = mid - 1 
            else:
                l = mid + 1
        return l
```

## 第二题：显生之宙 https://www.nowcoder.com/practice/c2335f2530974af3a4ec15db2851f532?channelPut=tracker2

**思路：排序贪心。首先每个数的位置无关，因此可以进行排序操作，然后就可以执行如下贪心策略：当前数加上前缀和，如果此时为负数，继续加入到前缀和中；如果是正数就直接加到答案中，或者逐个往下加这两个操作等价，最终剩下的数再加上负数的前缀和就是答案**

**代码：**
