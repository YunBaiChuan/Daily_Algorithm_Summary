**思路：我最开始想的是用数学方法，然后按照奇偶性进行讨论，不过有点问题。我们观察到n具有单调性，n越大越能满足条件，反之越不能满足条件。所以说就可以进行二分答案，找最小的那个n**

**具体的check函数就是分别算出方块的两个部分，一个多的部分，一个少的部分，然后分别对mn, mx进行比较，同时满足条件就可以往小的方向进行二分，反之往大的方向进行二分。最终得到答案，加入到out数组中，然后整体输出即可**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.isqrt(x))

'''

'''

out = []

def solve():
    a, b = MII()
    mx = fmax(a, b)
    mn = fmin(a, b)

    def check(mid):
        x = mid * mid // 2
        y = mid * mid - x
        return mn <= x and mx <= y

    l, r = 1, 2 * 10 ** 5
    while l <= r:
        mid = (l + r) // 2
        if check(mid):
            r = mid - 1
        else:
            l = mid + 1

    out.append(str(l))

# t = 1
t = II()
for _ in range(t):
    solve()

print('\n'.join(out))
```
