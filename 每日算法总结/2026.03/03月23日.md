## 第二题：矩阵的最大非负积 https://leetcode.cn/problems/maximum-non-negative-product-in-a-matrix/?envType=daily-question&envId=2026-03-23

**思路：这题算是数组最大乘积dp + 网格图dp了。整体写下来有点像树型dp的信息传递那块，也就是每次递归需要两个参数才能确定状态的转移，对于此题来说就是最大乘积和最小乘积。这里就只给出记忆化搜索的版本**

**代码：**
```python
class Solution:
    def maxProductPath(self, grid: List[List[int]]) -> int:
        mod = 10 ** 9 + 7
        @lru_cache(maxsize=None)
        def dfs(i: int, j: int) -> Tuple[int, int]:
            x = grid[i][j]
            if i == j == 0:
                return x, x

            res_min, res_max = inf, -inf
            if i > 0:
                mn, mx = dfs(i - 1, j)
                res_min = min(mn * x, mx * x)
                res_max = max(mn * x, mx * x)
            if j > 0:
                mn, mx = dfs(i, j - 1)
                res_min = min(res_min, mn * x, mx * x)
                res_max = max(res_max, mn * x, mx * x)

            return res_min, res_max

        ans = dfs(len(grid) - 1, len(grid[0]) - 1)[1]
        return -1 if ans < 0 else ans % mod
```
