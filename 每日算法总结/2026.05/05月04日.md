## 第一题：旋转图像 https://leetcode.cn/problems/rotate-image/description/?envType=daily-question&envId=2026-05-04

**思路：原地旋转图像板子题，就是在转置写法的基础上多一个行翻转**

**代码：**
```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        for i, row in enumerate(zip(*matrix[::-1])):
            matrix[i] = row
```
