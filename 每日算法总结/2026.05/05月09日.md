## 第一题：循环轮转矩阵 https://leetcode.cn/problems/cyclically-rotating-a-grid/description/?envType=daily-question&envId=2026-05-09

**思路：模拟。利用方向向量进行问题的简化，先枚举每一圈，进行该圈元素的收集；接着旋转收集到的一维数组元素；最后填回二维矩阵中**

**代码：**
```python
DIRS = (0, 1), (1, 0), (0, -1), (-1, 0)  # 右下左上

class Solution:
    def rotateGrid(self, grid: List[List[int]], k: int) -> List[List[int]]:
        m0, n0 = len(grid), len(grid[0])

        # 从外到内枚举圈
        for i in range(min(m0, n0) // 2):
            m, n = m0 - i * 2, n0 - i * 2  # 这一圈的行数和列数
            x, y = i, i  # 这一圈的左上角
            a = []
            for dx, dy in DIRS:
                for _ in range(n - 1):
                    a.append(grid[x][y])
                    x += dx
                    y += dy
                m, n = n, m  # 每发生一次旋转，行列进行交换

            shift = k % len(a)
            a = a[shift:] + a[:shift] # 收集并处理该圈元素

            j = 0 
            for dx, dy in DIRS:
                for _ in range(n - 1):
                    grid[x][y] = a[j] # 再转一遍，填写元素
                    j += 1
                    x += dx
                    y += dy
                m, n = n, m

        return grid
```
