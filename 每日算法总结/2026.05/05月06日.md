## 第一题：旋转盒子 https://leetcode.cn/problems/rotating-the-box/description/?envType=daily-question&envId=2026-05-06

**思路：模拟。模拟重力下落以及矩阵旋转**

**代码：**
```python
class Solution:
    def rotateTheBox(self, grid: List[List[str]]) -> List[List[str]]:

        def rotate(a):
            res = []
            for i, row in enumerate(zip(*a[::-1])):
                res.append(row)
            return res
        
        m, n = len(grid), len(grid[0])
        for i, row in enumerate(grid):
            cnt1 = cnt2 = 0 
            res = ""
            for j, x in enumerate(row):
                if x == '#':
                    cnt1 += 1
                elif x == '.':
                    cnt2 += 1
                else:
                    res += '.' * cnt2 + '#' * cnt1 + '*'
                    cnt1 = cnt2 = 0

            res += '.' * cnt2 + '#' * cnt1
            grid[i] = list(res)
                        
        return rotate(grid)
```
