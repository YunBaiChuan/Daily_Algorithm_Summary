## 第一题：到达末尾下标所需的最大跳跃次数 https://leetcode.cn/problems/maximum-number-of-jumps-to-reach-the-last-index/?envType=daily-question&envId=2026-05-10

**思路：一维dp。不过要注意初始化dp数组怎么写，创建的时候应该写-inf，表示不可达，然后初始化dp[0] = 0表示到达索引0位置最多走0步，然后就可以两个for算dp数组了**

**代码：**
```python
class Solution:
    def maximumJumps(self, nums: List[int], target: int) -> int:
        inf = 10 ** 18
        n = len(nums)
        dp = [-inf] * n 
        dp[0] = 0
        for i in range(n):
            for j in range(i + 1, n):
                if abs(nums[i] - nums[j]) <= target:
                    dp[j] = max(dp[j], dp[i] + 1)
        return dp[-1] if dp[-1] > 0 else -1
```
