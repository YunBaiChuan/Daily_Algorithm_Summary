## 第一题：寻找旋转排序数组中的最小值 II https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array-ii/description/?envType=daily-question&envId=2026-05-16

**思路：二分搜索。修改一下条件判断，如果mid位置的元素和r位置的元素相同，那么我们只需要缩减一次r即可，为什么？两种情况：1.mid位置是最小值，那么缩减一次r合理，最小值mid还在查找范围内；2.mid位置不是最小值，那么缩减一次r依旧合理，因为最小值肯定还在查找范围内；如果mid位置元素更小，那就缩减r范围到mid，而不是常规的mid - 1，因为此时mid更小，有可能是最小值；如果mid元素位置更大，必然不是最小值，增加l范围到mid + 1，最终找到答案**

**代码：**
```python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        n = len(nums)
        l, r = 0, n - 1
        while l <= r:
            mid = (l + r) >> 1
            if nums[mid] == nums[r]:
                r -= 1
            elif nums[mid] < nums[r]:
                r = mid
            else:
                l = mid + 1
        return nums[l]
```

## 第二题：异或 https://www.nowcoder.com/practice/4c788d7749874d4387843fe2ff99a2f3?channelPut=tracker2

**思路：数学，区间交集。首先两个数异或和为0，说明这两个数相等，因此先算两个区间的交集。对于区间$[a, b]$，$[c, d]$来说，令$L = fmax(a, c)$，$R = fmin(b, d)$他们的交集为$[L, R]$，如果$L < R$，则说明没有交集。算出交集之后，两个区间随机到同一个数的概率就是$1\ /\ l\ * 1\ /\ r * both$，表示首先两个区间要选到同一个数，然后同一个数的可能性有both种。因此得到分子和分母分别是both和l * r，然后用gcd进行化简，输出化简后的分式形式即可**
