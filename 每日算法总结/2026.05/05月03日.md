## 第一题：前缀平方和序列 https://www.nowcoder.com/practice/5a4a7feaa70e45f1ad69ac86f0e4b246?channelPut=tracker2

**思路：二维dp + 状态数优化 + 枚举优化。首先要根据数据量进行dp的状态定义：$dp[i][j]$表示当前第i个前缀和为j * j，然后就可以得到状态转移方程为$dp[i][j] = sum(dp[i - 1][k])\ (k < j)$，对于前i - 1个前缀和来说，只要其小于k，我们都可以通过$j * j - k * k$找到一个数来满足状态转移，然后这个转移方程我们可以用前缀和来进行枚举优化，最终累加$dp[n][j]$到ans中得到结果**

**代码：**

def solve():
    n, x = MII()
    mod = 10 ** 9 + 7

    m = isqrt(x)
    dp = [[0] * (m + 2) for _ in range(n + 2)] # dp[i][j]表示前i个前缀和为j^2的方案数

    for j in range(1, m + 1):
        dp[1][j] = 1

    for i in range(2, n + 1):
        pref = [0] * (m + 2) # 前缀和优化
        for j in range(1, m + 1):
            pref[j] = (pref[j - 1] + dp[i - 1][j]) % mod

        for j in range(1, m + 1): # 状态转移，应该是前面dp[i-1][k]（k < j）的累加和，为什么？因为不管我前面dp[i - 1][k]这个k是多少，我都能找到数字填在当前位置，使其满足dp[i][j]的状态转移
            dp[i][j] = pref[j - 1] % mod

    ans = 0 # 枚举累加答案
    for j in range(1, m + 1):
        ans = (ans + dp[n][j]) % mod

    print(ans)

t = 1

    solve()
