## 第一题：跳跃游戏Ⅳ https://leetcode.cn/problems/jump-game-iv/description/?envType=daily-question&envId=2026-05-18

**思路：bfs扩展。最少的操作次数，其实就是从0到n - 1位置的最短路径。因此我们可以额外收集相同的元素索引，存入hash表，之后再跑一个bfs，算出dist数组，得到最终答案**

**代码：**
```python
class Solution:
    def minJumps(self, arr: List[int]) -> int:
        inf = 10 ** 18
        n = len(arr)
        
        d = defaultdict(list)
        for i, x in enumerate(arr):
            d[x].append(i)
        
        dist = [inf] * n
        dist[0] = 0
        q = deque([0])
        
        while q:
            for _ in range(len(q)):
                i = q.popleft()
                if i == n - 1:
                    return dist[i]
                
                for j in [i - 1, i + 1]:
                    if 0 <= j < n and dist[j] == inf:
                        dist[j] = dist[i] + 1
                        q.append(j)
                
                for j in d[arr[i]]:
                    if dist[j] == inf:
                        dist[j] = dist[i] + 1
                        q.append(j)
                
                d[arr[i]] = []
        
        return dist[n - 1]
```

## 第二题：小红勇闯地下城 https://www.nowcoder.com/practice/745da48b6faf45788f014d84704cc258?channelPut=tracker2

**思路：分层图bfs的优化。比较容易看出来是分层图bfs，但是一个状态就是(i, j, k)，那么整个算法的时间复杂度就是$O(n * m * h)$，大概是$10^{14}$，那肯定是不行的。因此，我们可以发现对于血量h来说，它具有单调性，也就是要么不变，要么减少，并且同一个位置来说，血量少的肯定劣于血量高的。那就可以去掉k这一血量维度，只用(i, j)状态，并且该位置状态存放的是到达该位置时，最多的血量。这样就可以通过了，本题用dijkstra算法也是可以的**

**代码：**

    return input()

    return int(input())

def solve():
    n, m, h = LII()
    s = [list(I()) for _ in range(n)]

    si, sj = [[i, j] for i in range(n) for j in range(m) if s[i][j] == 'S'][0]
    ti, tj = [[i, j] for i in range(n) for j in range(m) if s[i][j] == 'T'][0]

    best = [[-1] * m for _ in range(n)]
    best[si][sj] = h
    q = deque([(si, sj)])

    while q:
        for _ in range(len(q)):
            i, j = q.popleft()
            cur = best[i][j]

            if i == ti and j == tj:
                print("Yes")
                return

            for x, y in (i + 1, j), (i - 1, j), (i, j + 1), (i, j - 1):
                if 0 <= x < n and 0 <= y < m:
                    if x == ti and y == tj:
                        if cur > 0:
                            print("Yes")
                            return
                        continue

                    if s[x][y] == 'S':
                        cost = 0
                    else:
                        cost = int(s[x][y])

                    new_hp = cur - cost
                    if new_hp > 0 and new_hp > best[x][y]:
                        best[x][y] = new_hp
                        q.append((x, y))

    print("No")

# t = 1
t = II()
for _ in range(t):
    solve()
