## 第一题：使数组互补的最少操作次数 https://leetcode.cn/problems/minimum-moves-to-make-array-complementary/description/?envType=daily-question&envId=2026-05-13

**思路：思维 + 差分。这题难的原因不在于通过区间+-想到差分，而是思维上怎么处理题目给出的条件。本题引入了一个$a[k]$数组，表示$nums[i] + nums[n - 1 - i]$变成k需要的操作次数，接着我们就想到对于每一对$nums[i] + nums[n - 1 - i]$都算出$a[k]$，那么答案就是他们的总和了。那操作数怎么算？就根据区间的交并来得到操作数为0，1，2的具体区间，接着想到对于区间的+-来说，差分是高效的处理方式。最终，前缀和算整体变成k的最小值就是答案**

**代码：**
```python
class Solution:
    def minMoves(self, nums: List[int], limit: int) -> int:
        n = len(nums)
        diff = [0] * (limit * 2 + 2)

        for i in range(n // 2):
            x = nums[i]
            y = nums[-1 - i]
            l = min(x, y) + 1
            r = max(x, y) + limit

            diff[2] += 2
            diff[l] -= 2

            diff[l] += 1
            diff[r + 1] -= 1

            diff[x + y] -= 1
            diff[x + y + 1] += 1
            
            diff[r + 1] += 2

        ans = inf
        sum_d = 0
        for d in diff[2 : limit * 2 + 1]:
            sum_d += d
            ans = min(ans, sum_d)
        return ans
```

## 第二题：最近公共祖先（LCA）https://www.nowcoder.com/practice/8004903f8eff4473b5c590b85afd7217?channelPut=tracker2

**思路：LCA模板题。对于这种多次查询来说，每次跑一个lca肯定会超时，因此要用倍增进行优化**

**代码：**

LOG = 20

def solve():
    n, m, r = LII()

    g = [[] for _ in range(n + 1)]
    for _ in range(n - 1):
        u, v = MII()
        g[u].append(v)
        g[v].append(u)

    fa = [[0] * (n + 1) for _ in range(LOG)]
    dep = [0] * (n + 1)

    q = deque()
    q.append(r)
    dep[r] = 1
    fa[0][r] = 0

    while q:
        u = q.popleft()
        for v in g[u]:
            if v != fa[0][u]:
                dep[v] = dep[u] + 1
                fa[0][v] = u
                q.append(v)

    for k in range(1, LOG):
        for u in range(1, n + 1):
            fa[k][u] = fa[k - 1][fa[k - 1][u]]

    def lca(u, v):
        if dep[u] < dep[v]:
            u, v = v, u

        diff = dep[u] - dep[v]
        for k in range(LOG - 1, -1, -1):
            if diff >> k & 1:
                u = fa[k][u]

        if u == v:
            return u

        for k in range(LOG - 1, -1, -1):
            if fa[k][u] != fa[k][v]:
                u = fa[k][u]
                v = fa[k][v]

        return fa[0][u]

    out = []
    for _ in range(m):
        u, v = MII()
        out.append(str(lca(u, v)))
    print('\n'.join(out))

t = 1

for _ in range(t):
    solve()
