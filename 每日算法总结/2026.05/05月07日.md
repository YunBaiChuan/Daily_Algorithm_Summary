## 第一题：小红的矩阵染色 https://www.nowcoder.com/practice/f8b771318bb04490b7389cc35e148166?channelPut=tracker2

**思路：预处理 + 排序贪心。先按列把每个可染色段的长度算出，并且存放在tmp数组；接着从大到小排序，然后贪心地染色，选$fmin(length,\ k)$，而不是$k >= length$，为什么？因为我们可以染其中的一部分，而不是每段必须染完，最终累加答案，并且输出即可**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''


def solve():
    n, m, k = LII()
    a = [list(I()) for _ in range(n)]

    tmp = []
    for j in range(m):
        cnt = 0
        for i in range(n):
            if a[i][j] == '*':
                if cnt: tmp.append(cnt)
                cnt = 0
            else:
                cnt += 1
        if cnt: tmp.append(cnt)

    tmp.sort(reverse=True)

    ans = 0
    for length in tmp:
        if k <= 1:
            break
        use = fmin(length, k) # 应该选更小的一方进行染色
        ans += use - 1
        k -= use

    print(ans)

t = 1
# t = II()
for _ in range(t):
    solve()
```
