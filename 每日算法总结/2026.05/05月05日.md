## 第一题：旋转链表 https://leetcode.cn/problems/rotate-list/description/?envType=daily-question&envId=2026-05-05

**思路：链表操作，根据题意模拟即可**

**代码：**
```python
class Solution:
    def rotateRight(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        if not head:
            return head

        n = 1
        p = head 
        while p.next:
            n += 1
            p = p.next 
        
        k = k % n 
        if k == 0:
            return head

        cnt = 1
        pre = head 
        nxt = pre.next
        while cnt < n - k:
            pre = pre.next 
            nxt = nxt.next
            cnt += 1
        
        pre.next = None 
        p.next = head 
        return nxt 
```
