## 第一题：跳跃游戏 III https://leetcode.cn/problems/jump-game-iii/description/?envType=daily-question&envId=2026-05-17

**思路：建图 + bfs/dfs。首先题目给定的start单个起始位置，首先就要想到bfs，dfs，dijkstra这类算法了，然后再根据题目要求建图，跑一下bfs/dfs即可，由于这题没有> 1的权值所以说不需要用dijkstra，并且没让算最短到达路径也可以不用bfs，总之都可以做**

**代码：**
```python
class Solution:
    def canReach(self, arr: List[int], start: int) -> bool:
        if 0 not in arr:
            return False

        n = len(arr)
        g = [[] for _ in range(n)]
        for i, x in enumerate(arr):
            if i - x >= 0:
                g[i].append(i - x)
            if i + x < n:
                g[i].append(i + x)
        
        q = deque([start])
        vis = [False] * n 
        vis[start] = True 
        while q:
            for _ in range(len(q)):
                x = q.popleft()
                if arr[x] == 0:
                    return True 
                for y in g[x]:
                    if not vis[y]:
                        q.append(y)
                        vis[y] = True 
        return False
```
