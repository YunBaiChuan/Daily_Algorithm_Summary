## 第一题：旋转函数 https://leetcode.cn/problems/rotate-function/description/?envType=daily-question&envId=2026-05-01

**思路：数学，推式子。最难想到的一点就是$O(n^2)$怎么优化到$O(n)$，这个要推式子。具体来说就是根据邻项$F(i)$与$F(i - 1)$的差值进行找规律，推式子。可以得到结论为$F(i) = F(i - 1) + S - n * a_{n - i}$，有了这个结论就可以$O(n)$做了**

**代码：**
```python
class Solution:
    def maxRotateFunction(self, nums: list[int]) -> int:
        n = len(nums)
        f = sum(i * x for i, x in enumerate(nums))
        s = sum(nums)

        ans = f
        for i in range(n - 1, 0, -1):
            f += s - n * nums[i]
            ans = max(ans, f)
        return ans
```

## 第二题：小红关鸡 https://www.nowcoder.com/practice/5982b0bfe2fe4ed2a872fd5383271f5f?channelPut=tracker2

**思路：不定长滑动窗口找最大覆盖个数。严格来说，其实是一个双指针。我们就根据不超过区间长度k这个条件，先排序，再用不定长滑动窗口找到合法区间范围中，能够覆盖的最大点数，最终再除总个数，就是答案**
