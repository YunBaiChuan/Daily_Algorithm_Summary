## 第一题：音符 https://www.nowcoder.com/practice/fbef4a433583436cbf1a7861a5c110d7?channelPut=tracker2

**思路：首先最简单的想法就是构建出查询数组，但他的时间复杂的是$O(n * b)$，也就是$5 * 10^8$，不管是时间还是空间上都不行，因此我们不能直接去构建查询数组**

**那本题其实更像是一种不均匀的分组问题，然后查当前音符是属于哪个组的，如果是均匀分组的话，我们其实就可以用类似于位图的处理方法，用divmod计算除数和余数，确定分组，那对于这种不均匀的分组，我们其实可以用前缀和 + 二分查找来做**

**怎么想到的？注意到题目给出的查询t数组，其实就是一种前缀和，因此我们首先要处理出一个前缀和数组，并且由于前缀和数组天生具有的单调递增特性，我们就可以用二分查找。**

**这里我直接用bisect_right库函数，就是二分查找>当前数的第一个数索引，然后-1就是<=当前数的最后一个数索引，由于音符类型是从1开始的，所以说还要+1，那么两个抵消之后，其实就只是bisect_right的结果了，最终回答多次查询，再整体输出即可**

**代码：**
```python

    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y

    n, q = MII()
    b = LII()
    t = LII()

    pre = [0] * (n + 1)
    for i, x in enumerate(b):
        pre[i + 1] = pre[i] + x

    out = []
    for x in t:
        p = bisect_right(pre, x)
        out.append(str(p))

    print("\n".join(out))
```

## 第二题：Balanced 01-String https://www.nowcoder.com/practice/cbd14d9770ca4efb98ad690272d76757?tpId=391&contestId=126798&channelPut=tracker1

**思路：很好的01字符串思维题，正难则反。题目说的是相邻相同的字符个数要为偶数，那么我们可以先算相邻不同字符个数的奇偶性，然后用(n - 1 - 相邻不同奇偶性) % 2来进行计算，那么相邻不同奇偶性怎么计算？**

**首先我们可以算相邻数异或的结果，然后把他们累加起来，再%2算其奇偶性，但由于异或与奇偶性的性质，上面的式子其实就是把所有式子都给异或起来，两者是恒等关系，比如：（s1 ^ s2 + s2 ^ s3 + s3 ^ s4）% 2 = s1 ^ s2 ^ s2 ^ s3 ^ s3 ^ s4**

**那么我们就可以得到结论：相邻不同的奇偶性由头尾异或和决定，相邻相同的奇偶性由n - 1和头尾异或和决定，那么方案数就好算了，直接判断下中间的'?'数量，然后判断下头尾的'?'情况即可**

**代码：**

```python
    return list(map(float, input().split()))

mod = 998_244_353

def solve():
    s = I()

    n = len(s)
    cnt = s[1:-1].count('?')

    if s[0] == '?' and s[-1] == '?':
        print(pow(2, cnt + 1, mod))
    elif s[0] == '?' or s[-1] == '?':
        print(pow(2, cnt, mod))
    else:
```
