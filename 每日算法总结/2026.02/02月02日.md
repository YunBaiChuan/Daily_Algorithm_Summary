## 第一题：躲藏 https://www.nowcoder.com/practice/b19eda9490d54505a1065a70c3005b47?channelPut=tracker2

**思路：很经典的s, t子序列匹配板子题，如果说题目要求判断s的子序列中有没有t，就可以直接用双指针，不过本题是要求算具体的数量，就该用子序列dp来做了。然后加了一个多输入，对于python来说要用try-except语句来实现。最终，输出答案即可**

## 第二题：将数组分成最小代价的子数组II https://leetcode.cn/problems/divide-an-array-into-subarrays-with-minimum-cost-ii/description/?envType=daily-question&envId=2026-02-02

**思路：本题翻译一下就是在一个dist + 1的滑动窗口中，找前k - 1个最小值，然后求他们的和再加上初始的nums[0]，就是答案。这个其实可以用对顶堆来做，类似的题目有找数组的中位数。不过对于本题的滑动窗口+对顶堆来说，他不只是简单的像是找中位数，只需要处理堆顶元素，他也有可能处理堆中的元素，所以说如果用对顶堆的话，不能用一般的堆，要用懒删除堆来做**

**不过为了图方便，我们可以用有序集合来做这题，也就是功能十分强大的SortedList，这个结构可以任意删除其中的元素，并且始终保持从小到大的有序状态，还能用bisect方法，非常方便，就可以用他来平替懒删除堆，还更好写**

**代码：**
```python
class Solution:
    def minimumCost(self, nums: List[int], k: int, dist: int) -> int:
        k -= 1 # 提前把nums[0]给减去
        sum_left = sum(nums[:dist + 2])
        L = SortedList(nums[1:dist + 2])
        R = SortedList()

        def L2R():
            x = L.pop()
            nonlocal sum_left 
            sum_left -= x 
            R.add(x)
        
        def R2L():
            x = R.pop(0)
            nonlocal sum_left 
            sum_left += x 
            L.add(x)
        
        while len(L) > k:
            L2R()
        
        ans = sum_left 
        for i in range(dist + 2, len(nums)):
            out = nums[i - dist - 1]
            if out in L:
                sum_left -= out 
                L.remove(out)
            else:
                R.remove(out)

            in_val = nums[i]
            if in_val < L[-1]:
                sum_left += in_val 
                L.add(in_val)
            else:
                R.add(in_val)
            
            if len(L) == k - 1:
                R2L()
            elif len(L) == k + 1:
                L2R()
            
            ans = min(ans, sum_left)
        
        return ans 
```
