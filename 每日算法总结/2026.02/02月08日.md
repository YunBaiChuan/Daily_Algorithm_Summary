## 第二题：平衡二叉树 https://leetcode.cn/problems/balanced-binary-tree/?envType=daily-question&envId=2026-02-08

**思路：在算树高度的基础上，加上那条if限制语句，就可以把无法满足条件的情况用-1作为返回，最终判断一下是否不等于-1就可以了**

**代码：**
```python
class Solution:
    def isBalanced(self, root: Optional[TreeNode]) -> bool:
        def get_height(root):
            if not root:
                return 0
            left_high = get_height(root.left)
            right_high = get_height(root.right)
            if left_high == -1 or right_high == -1 or abs(left_high - right_high) > 1:
                return -1
            return max(left_high, right_high) + 1
        return get_height(root) != -1
```
