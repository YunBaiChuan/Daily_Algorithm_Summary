## 第一题：小红的好排列 https://www.nowcoder.com/practice/67bf8c0b7fc64f3bb9cc21bf6dbbd14f?channelPut=tracker2

**思路：组合数学，推式子。这种组合数学比较常用的就是fac预处理以及乘法逆元inv和comb的函数了，要写熟。然后就是数学过程的推式子了，令$cnt = n \ //\  3$表示$n$中有多少个位置是3的倍数，同时也表示排列中有多少个数是3的倍数；再令$need = n\ //\ 2$表示我们最终需要的3的倍数位置个数。**

**n其实可以分成两组数据，一组为3的倍数，一组为不是3的倍数。那么答案就是先$*\ comb(cnt,\ need - cnt)$表示需要从cnt个3的倍数中选出除去3的倍数位置外，额外需要的3的倍数，他们的方案数，要把他们放到外面（不是3的倍数位置）；然后再$*\ comb(n - cnt,\ need - cnt)$表示从不是3的倍数中选出和刚才放到外面的3的倍数相同的不是3的倍数的数量，他们的方案数，要把他们放到里面（是3的倍数位置）；最后再各自$*\ fac[cnt] * fac[n - cnt]$分别表示里面和外面元素的全排列，因此答案为$ans = comb(cnt,\ need - cnt) * comb(n - cnt,\ need - cnt) * fac[cnt] * fac[n - cnt]$**

## 第二题：字典树的实现 https://www.nowcoder.com/practice/a55a584bc0ca4a83a272680174be113b?tpId=196&tqId=37151&ru=/exam/oj

**思路：字典树板子题，要理解字典树的原理，然后掌握静态数组的字典树写法，实现insert, search, prefixNumber, delete这四个最重要的函数写法，重复度还是比较高的，比较好写**

**代码：**
```python
from typing import List

class Trie:
    def __init__(self, n):
        self.tree = [[0] * 26 for _ in range(n)]
        self.p = [0] * n
        self.e = [0] * n
        self.cnt = 1

    def insert(self, word):
        cur = 1
        self.p[cur] += 1
        for i in range(len(word)):
            path = ord(word[i]) - ord('a')
            if self.tree[cur][path] == 0:
                self.cnt += 1
                self.tree[cur][path] = self.cnt
            cur = self.tree[cur][path]
            self.p[cur] += 1
        self.e[cur] += 1

    def search(self, word):
        cur = 1
        for i in range(len(word)):
            path = ord(word[i]) - ord('a')
            if self.tree[cur][path] == 0:
                return 0
            cur = self.tree[cur][path]
        return self.e[cur]

    def prefixNumber(self, pre):
        cur = 1
        for i in range(len(pre)):
            path = ord(pre[i]) - ord('a')
            if self.tree[cur][path] == 0:
                return 0
            cur = self.tree[cur][path]
        return self.p[cur]

    def delete(self, word):
        if self.search(word) <= 0:
            return
        cur = 1
        for i in range(len(word)):
            path = ord(word[i]) - ord('a')
            self.p[self.tree[cur][path]] -= 1
            if self.p[self.tree[cur][path]] == 0:
                self.tree[cur][path] = 0
                return
            cur = self.tree[cur][path]
        self.e[cur] -= 1

    def clear(self):
        for i in range(1, self.cnt + 1):
            self.p[i] = 0
            self.e[i] = 0
            for j in range(26):
                self.tree[i][j] = 0
        self.cnt = 1

class Solution:
    def trieU(self , operators: List[List[str]]) -> List[str]:
        n = len(operators)
        t = Trie(1000001)
        out = []
        for i in range(n):
            op, word = operators[i]
            if op == '1':
                t.insert(word)
            elif op == '2':
                t.delete(word)
            elif op == '3':
                res = t.search(word)
                out.append("YES" if res else "NO")
            else:
                res = t.prefixNumber(word)
                out.append(str(res))
        return out
```
