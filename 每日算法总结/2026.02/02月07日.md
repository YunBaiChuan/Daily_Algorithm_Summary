## 第一题：相邻的糖果 https://www.nowcoder.com/practice/5799a1a6fb104a63bda2614cdf64b09b?channelPut=tracker2

**思路：定长滑动窗口+贪心。我们要让每个滑动窗口的值都$< x$并且操作数最少，那就主要操作窗口中靠近右侧的元素。为什么？因为越靠近右侧的元素，在我们向右滑动窗口时，更容易被保留下来，也就会产生更大的贡献。**

**因此，我们就$O(n)$滑动窗口，当发现窗口中的和$s > x$时，就累加操作数$op$，然后从窗口右端开始往左端，对元素进行减操作，修改元素值。这里我理解错了题意，题目说的是输入数据$a_i >= 1$，我就下意识以为操作后的每个盒子，其糖果数也必须$a_i >= 1$，这里调了挺久，其实是可以变成0的。**

**最后，输出累加后的操作数$op$即可，他的时间复杂度应该是$O(n * m)$，但能过就很神秘**

**代码：**

def solve():
    n, m, x = LII()
    a = LII()

    op = s = 0
    for r, v in enumerate(a):
        s += v
        if r < m - 1:
            continue

        if s > x:
            diff = s - x
            op += diff

            i = r
            while i >= r - m + 1 and diff > 0:
                if diff > a[i]:
                    diff -= a[i]
                    s -= a[i]
                    a[i] = 0
                else:
                    a[i] -= diff
                    s -= diff
                    diff = 0
                i -= 1

        s -= a[r - m + 1]
