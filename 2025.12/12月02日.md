## 第一题：3623.统计梯形的数目I https://leetcode.cn/problems/count-number-of-trapezoids-i/description/?envType=daily-question&envId=2025-12-02

**思路：就是统计y坐标相同的数量，然后C(n, 2)求每个y的组合数，并且相乘求和。如果用两个for来相乘求和必然超时，那就可以记录一个前缀组合数和s来实现一次遍历求和，最终得到ans**

**代码：**
```python
class Solution:
    def countTrapezoids(self, points: List[List[int]]) -> int:
        mod = 10 ** 9 + 7 
        d = defaultdict(int)
        for x, y in points:
            d[y] += 1 
        
        ans = s = 0
        for k, v in d.items():
            ans = (ans + v * (v - 1) // 2 * s) % mod 
            s += v * (v - 1) // 2 
        return ans
```

## 第二题：小红的gcd https://www.nowcoder.com/practice/9a5ac1e1c2fb4f8d85595a52ff0a00fa?channelPut=tracker2

**思路：一眼欧几里得优化，但是对于这题而言还不够，还会超时。因此，我们要利用mod的性质，把a用递推式计算一遍，最终得到的va就可以直接和b进行gcd，这样就不会超时了**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
a非常大，不能用简单的欧几里得方法，即math.gcd(a, b) = math.gcd(b, a % b)计算，也会超时
所以说要把a用字符串进行存储，然后逐步%b得到处理后的va，最终再和b进行gcd即可

这里va的过程又用到了mod性质，每次用余数操作+递推式
'''

def solve():
    a = I()
    b = II()
    va = 0
    for ch in a:
        va = (va * 10 + int(ch)) % b
    print(math.gcd(va, b))


t = 1
for _ in range(t):
    solve()
```
