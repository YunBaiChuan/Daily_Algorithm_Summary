## 第一题：1523.在区间范围内统计奇数 https://leetcode.cn/problems/count-odd-numbers-in-an-interval-range/description/?envType=daily-question&envId=2025-12-07

**思路：首先区间问题 + 大数据范围，要想到用前缀和，这里选择的是统计1到high的奇数 - 1到low - 1的奇数，就是闭区间[low, high]的奇数了，那么1到x区间的奇数个数其实就是(x - 1) // 2 + 1，也就是区间长度 // 2的上限**

**代码：**
```python
class Solution:
    def countOdds(self, low: int, high: int) -> int:
        def odd(x):
            return (x - 1) // 2 + 1
        return odd(high) - odd(low - 1)
```

## 第二题：降温 https://www.nowcoder.com/practice/194a05e3142b40e8aac2657793f57ee7?channelPut=tracker2

**思路：最开始想的是贪心构造，然后分别统计计算，但其实是不可行的，尤其是对于求mn来说，不只是前面的温度有影响，后面的温度也会影响当前位置，因此要用dp来做，他这里是线性dp + 状态压缩，先放了。不对，这题贪心也能做，并且复杂度更优，不过我前面用的贪心策略有问题**

import sys

def I():

def solve():
    n, x = MII()
    a = LII()

    # 温度范围 [-50, 50]
    MIN_T = -50
    MAX_T = 50
    RANGE = MAX_T - MIN_T + 1

    # 初始化 DP 数组
    # dp_max[i][t] 表示前 i 天，第 i 天气温为 t 时，最多寒潮次数
    # dp_min[i][t] 表示前 i 天，第 i 天气温为 t 时，最少寒潮次数
    dp_max = [[-inf] * RANGE for _ in range(n)]
    dp_min = [[inf] * RANGE for _ in range(n)]

    # 第一天初始化
    if a[0] == -999:
        for t in range(RANGE):
            dp_max[0][t] = 0
            dp_min[0][t] = 0
    else:
        t = a[0] - MIN_T
        dp_max[0][t] = 0
        dp_min[0][t] = 0

    # 从第二天开始转移
    for i in range(1, n):
        if a[i] == -999:
            # 枚举当前天所有可能温度
            for cur in range(RANGE):
                # 枚举前一天的温度
                for prev in range(RANGE):
                    if dp_max[i-1][prev] == -inf:
                        continue
                    cnt = 1 if (prev + MIN_T) - (cur + MIN_T) >= x else 0
                    dp_max[i][cur] = max(dp_max[i][cur], dp_max[i-1][prev] + cnt)
                    dp_min[i][cur] = min(dp_min[i][cur], dp_min[i-1][prev] + cnt)
        else:
            # 当前天温度固定
            cur = a[i] - MIN_T
            for prev in range(RANGE):
                if dp_max[i-1][prev] == -inf:
                    continue
                cnt = 1 if (prev + MIN_T) - (cur + MIN_T) >= x else 0
                dp_max[i][cur] = max(dp_max[i][cur], dp_max[i-1][prev] + cnt)
                dp_min[i][cur] = min(dp_min[i][cur], dp_min[i-1][prev] + cnt)

    # 取最后一天所有可能温度的最大/最小值
    max_ans = max(dp_max[-1])
    min_ans = min(dp_min[-1])
    print(max_ans, min_ans)

t = 1
for _ in range(t):
    solve()
