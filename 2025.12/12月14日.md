## 第一题：2147.分割长廊的方案数 https://leetcode.cn/problems/number-of-ways-to-divide-a-long-corridor/description/?envType=daily-question&envId=2025-12-14

**思路：划分加方案数，很容易想到划分dp，但这题并不然。题目要求了每个划分必须是两个椅子，因此我们$O(n)$遍历一遍，然后每有两个椅子，到第三个椅子的时候，我们就进行划分，方案数其实就是$i - last_s$，由于每个部分独立，因此不断相乘取mod就是最终的ans**

**代码：**
```python
class Solution:
    def numberOfWays(self, corridor: str) -> int:
        mod = 10 ** 9 + 7
        ans = 1 
        cnt_s = last_s = 0 

        for i, ch in enumerate(corridor):
            if ch == 'S':
                cnt_s += 1
                if cnt_s >= 3 and cnt_s % 2:
                    ans = ans * (i - last_s) % mod 
                last_s = i 
        
        if cnt_s == 0 or cnt_s % 2:
            return 0
        
        return ans
```

## 第二题：静态区间最值 https://www.nowcoder.com/practice/831a314449d44ea0b1db90ca626bcd1a?channelPut=tracker2

**思路：非常经典的静态区间用st表回答最值查询的询问，我必须马上总结到板子题里面**

def solve():

    log_table = [0] * (n + 1)

    for _ in range(q):
